# Outline Câu Hỏi Mức Độ Vận Dụng

## Phạm vi
- Khóa học: `INT_HCM_Trí tuệ nhân tạo Trung học 1`
- Trạng thái: `PREVIEW_OUTLINE`
- Mục đích: duyệt trước các tình huống thực tế giảng dạy và mức độ tổ hợp kiến thức

## Phân bổ trong file này
- Số câu: `9`
- Mã câu: `22-30`
- Mức độ: `Vận dụng`

## Coverage dạng câu hỏi trong file này
- Chọn 1 đáp án đúng: `4`
- Chọn nhiều đáp án đúng: `4`
- Đúng / Sai: `0`
- Ghép nối: `0`
- Điền số / từ ngắn: `0`

## Nguyên tắc tăng độ khó
- `Câu 22-24`: tổ hợp 2 mảng kiến thức
- `Câu 25-27`: tổ hợp 3 mảng kiến thức kèm bối cảnh lớp học
- `Câu 28-30`: tổ hợp nhiều bước xử lý, yêu cầu chọn giải pháp hợp lý nhất cho dự án/thực hành
