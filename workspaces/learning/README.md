# Learning Workspace

Workspace này dùng cho workflow mặc định:

```text
Capture -> Learn Note -> Promote if valuable
```

Nội dung ở đây chỉ để học nhanh và tra cứu nhanh. Nó không phải nguồn sự thật (source of truth).

Ghi chú có giá trị lâu dài có thể được chuyển sang:

```text
1-projects/learning_maps/
```

Tri thức canonical chỉ được tạo qua:

```text
.agent/workflows/ingest-lifecycle.md
```

## Folders

```text
inbox/      # tài liệu học nhanh đặt tạm trong learning workspace
notes/      # learning notes, summaries, outlines
questions/  # câu hỏi tự học, câu hỏi tra cứu
flashcards/ # thẻ ghi nhớ
outputs/    # artifact xuất ra cho học tập, non-canonical
```

Không dùng các folder này để tạo Atom chính thức hoặc ghi vào `3-resources/`.
