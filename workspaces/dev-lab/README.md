# Dev Lab

`dev-lab` dùng cho thử nghiệm kỹ thuật và script work trước khi thay đổi vận hành chính.

Nội dung ở đây là non-canonical.

Không ghi vào `3-resources/raw_*` hoặc `3-resources/wiki/synthesis/` nếu không có GO chính xác.

## Folders

```text
repos/       # clone/link repo phụ hoặc notes về repo thử nghiệm
experiments/ # thử nghiệm script/tool/API
reports/     # benchmark report, patch report, review notes
sandbox/     # file tạm có thể bỏ
```

Patch vào `../../scripts/` chỉ thực hiện khi AN GO với file/scope rõ.
