# [NON_MCQ] Câu 25 (Thông hiểu)

## ❓ Câu hỏi
Hãy sắp xếp thứ tự phù hợp để dùng chế độ “Mô hình AI” và lập trình điều khiển Rover qua việc nhận diện hình ảnh?

## 📝 Đáp án lựa chọn
- **A/ (Đáp án)**
  Kết nối Bluetooth và tải chương trình vào thiết bị
  C-F-D-E-G-B-A
- **B/**
  Lập trình ở chế độ “Lập trình AI” và “Lập trình thiết bị”
  F-D-C-E-G-B-A
- **C/**
  Mở chế độ “Mô hình AI”, chọn “Tạo mô hình nhận diện hình ảnh”
  D-B-A-F-E-G-C
- **D/**
  Mở webcam, chụp ảnh hoặc tải ảnh từ máy tính
  D-C-F-A-E-G-B
- **E/**
  Chọn “Huấn luyện mô hình”
- **F/**
  Đặt tên nhóm cần huấn luyện nhận diện hình ảnh hình
  G/ Đặt tên dự án và lưu mô hình
