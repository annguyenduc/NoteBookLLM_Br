# [MCQ] Câu 2 (Nhận biết)

## ❓ Câu hỏi
Để tái (nạp) năng lượng cho robot Codey, chúng ta thực hiện bằng cách nào?

## 📝 Đáp án lựa chọn
- **A/**
  Robot Codey tự nạp năng lượng mặt trời tích hợp trên màn hình LCD (không khởi động robot Codey).
- **B/**
  Robot Codey sử dụng 4 pin AAA 1,5V. Khi hết pin có thể mở nắp pin bên dưới Rocky để thay.
- **C/**
  Robot Codey sử dụng pin Lithium. Khi hết pin có thể mở nắp pin bên dưới Rocky để thay. Pin này có thể sạc được.
- **D/ (Đáp án)**
  Để nạp năng lượng cho robot Codey, ta chỉ cần kết nối robot Codey với máy tính thông qua dây USB (không khởi động robot).
