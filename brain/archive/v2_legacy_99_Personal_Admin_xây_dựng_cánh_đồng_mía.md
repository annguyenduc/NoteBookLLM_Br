# Kiến trúc & Triển khai Cánh đồng Mía Tự động (Minecraft Edu + MakeCode)

## Nguyên lý hoạt động
Hệ thống vận hành dựa trên cơ chế phát hiện chiều cao bằng `Observer`. Khi mía phát triển đạt 3 block, observer gửi tín hiệu redstone kích hoạt `Piston` phá thân mía. Mía rơi xuống dòng nước, được đẩy về hệ thống `Hopper` → `Chest` để lưu trữ tự động, loại bỏ hoàn toàn thao tác thu hoạch thủ công.

## Phân chia tác vụ
- **Code Builder:** Đảm nhận hạ tầng chính và logic lặp lại: xây khung, đào mương nước, đặt mạch redstone (observer, piston, repeater), lắp hệ thống thu gom (hopper, chest). Ưu tiên dùng vòng lặp và biến tọa độ để sinh cấu trúc theo ma trận.
- **Agent:** Xử lý tác vụ chi tiết, độ chính xác cao hoặc yêu cầu tương tác vật lý: đặt block mía, block phụ trợ, điều chỉnh vị trí vi mô mà code builder khó căn chỉnh trực tiếp trong môi trường block.

## Cấu trúc Code MakeCode
- **Kích hoạt:** Dùng `on chat command` hoặc `on start` để chạy script.
- **Cấu hình linh hoạt:** Khai báo biến `soHang`, `soCot`, `khoangCach` cho phép test nhanh với kích thước nhỏ (1x3, 2x2) trước khi mở rộng.
- **Logic xây dựng:** 
  - Vòng lặp `for` kết hợp `agent.teleport` và `blocks.place` để dựng khung & redstone theo tọa độ tương đối.
  - Tách hàm rõ ràng: `xayKhung()`, `lapMachRedstone()`, `trongMia()`, `thietLapThuGom()` giúp dễ debug và bảo trì.
- **Quản lý tài nguyên:** Dùng `agent.setSlot()` và `agent.setItemCount()` để tự động phân phối vật liệu vào đúng slot trước khi thực thi.

## Lưu ý kỹ thuật
- Luôn xác minh logic redstone và hướng dòng nước ở kích thước tối thiểu trước khi nhân bản mô-đun.
- Đảm bảo agent có đủ inventory slot chứa block xây, mía và linh kiện redstone; thiếu slot sẽ gây lỗi gián đoạn.
- Code block MakeCode hỗ trợ xuất sang JavaScript/TypeScript, cho phép tinh chỉnh logic phức tạp hoặc thêm cơ chế tự động nạp lại vật liệu.
- Nguyên tắc mở rộng: Chỉ cần thay đổi biến cấu hình, hệ thống tự động tính toán offset tọa độ và nhân bản mô-đun mà không cần viết lại code nền tảng.
