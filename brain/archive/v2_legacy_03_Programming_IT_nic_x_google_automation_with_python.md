## 🛠️ Hướng dẫn kết nối VSCode với WSL
*(Đã cài VSCode trên Windows → **Không cần cài lại** trong WSL)*

1. **Cài Extension:**
   - Mở VSCode → `Ctrl + Shift + X` (Extensions)
   - Tìm & cài **`Remote - WSL`** (của Microsoft)

2. **Mở Project WSL trong VSCode:**
   - **Cách 1 (Từ Terminal WSL):**
     ```bash
     cd /đường/dẫn/thư/mục
     code .
     ```
     → VSCode sẽ mở folder và tự động kết nối môi trường WSL.
   - **Cách 2 (Từ VSCode):**
     - Nhấn `F1` hoặc `Ctrl + Shift + P`
     - Gõ `Remote-WSL: New Window` hoặc `Connect to WSL using Distro...`
     - Chọn distro Ubuntu → Mở folder trong WSL.
