# Thêm dấu `-` vào dòng cụ thể trong ô Google Sheets bằng Apps Script

Khi cần chèn dấu gạch ngang (`-`) vào một số dòng nhất định trong cùng một ô (phân tách bằng `Alt + Enter`), công thức hay Find/Replace sẽ tác động lên toàn bộ nội dung ô. Giải pháp chính xác là xử lý chuỗi theo từng dòng bằng Google Apps Script.

### 🛠️ Cách thực hiện
1. Mở Google Sheets → `Extensions` > `Apps Script`.
2. Dán đoạn mã sau vào trình soạn thảo:
```javascript
function addHyphenToSelectedLines() {
  const cell = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet().getActiveCell();
  const lines = cell.getValue().toString().split('\n');
  
  // Chỉ định chỉ số dòng cần thêm dấu '-' (đếm từ 0)
  const targetIndices = [0, 2]; // Ví dụ: dòng 1 và 3
  
  const updatedLines = lines.map((line, i) => 
    targetIndices.includes(i) ? `- ${line}` : line
  );
  
  cell.setValue(updatedLines.join('\n'));
}
```
3. Lưu dự án (`Ctrl/Cmd + S`), quay lại Sheet, chọn ô mục tiêu.
4. Chạy hàm `addHyphenToSelectedLines` từ menu Apps Script hoặc gán vào nút/phím tắt. Cấp quyền truy cập khi được yêu cầu.

### 💡 Lưu ý kỹ thuật
- Dòng trong ô được đánh chỉ số từ `0`. Mảng `targetIndices` quyết định dòng nào được chèn dấu `-`.
- Có thể thay logic `targetIndices.includes(i)` bằng điều kiện nội dung, ví dụ: `line.trim().startsWith('TODO')`.
- Script mặc định chạy trên ô đang chọn (`getActiveCell()`). Để áp dụng cho nhiều ô, thay bằng `getRange().getValues()`, lặp qua từng giá trị, xử lý và `setValues()` ngược lại.
- Ký tự xuống dòng trong Sheets luôn là `\n`, không dùng `\r\n`.
