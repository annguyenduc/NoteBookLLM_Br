# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Quản lý sao lưu & dung lượng iPhone, Tối ưu thông số in 3D (Elegoo Neptune 4), Cấu hình OrcaSlicer & Klipper G-code.
- **Vấn đề / Câu hỏi:** 
  1. Cơ chế backup iPhone, cách xử lý dung lượng "Other" và di chuyển file nhanh.
  2. Tính điện năng tiêu thụ, giới hạn tốc độ thực tế với nhựa Ender-PLA và thông số OrcaSlicer cho mô hình chi tiết.
  3. Khắc phục việc Klipper lưu Bed Mesh vào profile số, yêu cầu tối ưu G-code để đo và in ngay không cần restart.
- **Cách giải quyết / Logic lõi:** 
  **1. Quản lý Backup & Dung lượng iPhone:**
  - Bản backup chỉ chứa dữ liệu người dùng & cài đặt, **không** bao gồm hệ điều hành iOS hay file cài app (.IPA).
  - Mục "Other" (Dữ liệu hệ thống) chứa cache, file tạm, offline data. Không được backup. Cách dọn triệt để: Backup toàn bộ → Xóa trắng máy (Erase All) → Restore.
  - Di chuyển file nhanh trên Windows: Dùng `Ctrl + X` → `Ctrl + V` (cập nhật File Index thay vì copy vật lý). Dùng `mklink` tạo Symbolic Link để chuyển hướng thư mục mặc định.
  - Công cụ khuyến nghị: **3uTools** (miễn phí, Windows) hỗ trợ Customized Backup, Garbage Cleaner và kiểm tra phần cứng chi tiết.

  **2. Tối ưu Máy in 3D Neptune 4 & OrcaSlicer:**
  - Điện năng: Công suất trung bình ~150W (in PLA). 300h/tháng ≈ 45 kWh.
  - Tốc độ: Lý thuyết 500 mm/s, khuyến nghị 250 mm/s. Với Ender-PLA nên giữ **100-150 mm/s**, giới hạn `Max Volumetric Speed: 12-14 mm³/s`.
  - Thông số OrcaSlicer (Mô hình chi tiết): Layer `0.12-0.16mm`, Wall `3`, Top/Bottom `5`. Support: `Tree Organic`, Top Z distance `0.2mm`, tích chọn `Support on build plate only`.
  - Hiệu chỉnh rung: Dùng lệnh Fluidd `SHAPER_CALIBRATE` → `SAVE_CONFIG` để lưu Input Shaper.

  **3. Tối ưu G-code Klipper (Bed Mesh):**
  - Klipper lưu mesh tạm vào RAM (profile số). Lệnh `SAVE_CONFIG` gây restart máy, làm hỏng bản in đang chạy.
  - Giải pháp: Chèn trực tiếp vào Start G-code để xóa mesh cũ, đo mới và áp dụng ngay:
  ```gcode
  G90
  G28
  BED_MESH_CLEAR              ; Xóa mesh cũ trong RAM
  BED_MESH_CALIBRATE          ; Đo mesh mới hoàn toàn
  BED_MESH_PROFILE LOAD=default ; Áp dụng ngay mesh vừa đo
  ```
- **Ghi chú quan trọng:** 
  - iPhone 11 Pro Max dùng cổng Lightning (USB 2.0), tốc độ backup thực tế ~30-40MB/s (ước tính 60-90 phút cho 90GB).
  - Khi in chi tiết nhọn, bật `Z-hop when retracting: 0.2mm` và quạt `100%` để tránh va chạm đầu phun và biến dạng nhựa.
  - Dùng `Support Blocker`/`Enforcer` trong OrcaSlicer để kiểm soát chính xác vùng mọc support, kết hợp `Tree Support` giúp bề mặt sau khi tháo sạch hơn.
  - Tuyệt đối không dùng `SAVE_CONFIG` trong lúc đang in.
