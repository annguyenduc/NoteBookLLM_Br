# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Triển khai và cấu hình Giao thức Ngữ cảnh Mô hình (MCP) trên Cursor AI IDE.
- **Vấn đề / Câu hỏi:** Làm thế nào để kết nối Cursor AI với các nguồn dữ liệu/công cụ bên ngoài (tìm kiếm web, quản lý dự án, bộ nhớ cục bộ) thông qua chuẩn MCP nhằm mở rộng ngữ cảnh và khả năng tự động hóa của AI Agent?
- **Cách giải quyết / Logic lõi:** 
  - **Kiến trúc MCP:** Chuẩn mở client-server, giải quyết bài toán tích hợp N*M bằng giao thức chung. Cursor đóng vai trò MCP Client, kết nối với các MCP Server bên ngoài để lấy ngữ cảnh thời gian thực.
  - **Phương thức Transport:** 
    - `stdio`: Dành cho server cục bộ, giao tiếp đồng bộ qua luồng chuẩn. Thường triển khai bằng lệnh `npx`.
    - `sse` (Server-Sent Events): Dành cho server từ xa/cục bộ, giao tiếp bất đồng bộ qua URL.
  - **Cấu hình trong Cursor:** Thêm qua UI (`Settings > Cursor Settings > MCP > + Add New`) hoặc chỉnh sửa trực tiếp file `mcp.json`.
  - **Mẫu cấu hình `mcp.json`:**
    ```json
    // Asana (SSE qua proxy mcp-remote)
    {
      "asana": {
        "command": "npx",
        "args": ["mcp-remote", "https://mcp.asana.com/sse"]
      }
    }
    // Pieces LTM (SSE trực tiếp)
    {
      "mcpServers": {
        "Pieces": {
          "url": "http://localhost:{port}/model_context_protocol/{version}/sse"
        }
      }
    }
    ```
  - **Vận hành:** Chuyển Cursor sang chế độ `Agent`. AI tự động nhận diện ngữ cảnh và gọi công cụ MCP phù hợp, hoặc người dùng gọi thủ công. Cursor hỗ trợ xem log, kiểm tra tham số và cơ chế phê duyệt lệnh gọi công cụ.
- **Ghi chú quan trọng:** 
  - **Bảo mật:** Khóa API và OAuth token cần được quản lý cẩn thận trong lệnh cấu hình hoặc biến môi trường.
  - **Khắc phục lỗi:** 
    - Lỗi `Internal Server Error` (Asana): Xóa thư mục `~/.mcp-auth` và xác thực lại OAuth.
    - Lỗi kết nối: Kiểm tra đúng loại transport (`stdio`/`sse`), đảm bảo server cục bộ (ví dụ: PiecesOS) đang chạy, kiểm tra tường lửa/mạng.
    - Lỗi hiển thị `JSON Blob` trong cài đặt Pieces: Có thể bỏ qua nếu công cụ vẫn hoạt động bình thường.
  - **Xu hướng:** MCP là nền tảng cho hệ sinh thái đa tác tử (multi-agent), cho phép các AI agent khác nhau giao tiếp và sử dụng lẫn nhau như tài nguyên thông qua chuẩn chung, phá vỡ silo dữ liệu.
