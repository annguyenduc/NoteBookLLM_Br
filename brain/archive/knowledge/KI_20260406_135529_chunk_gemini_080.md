# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Khắc phục lỗi bàn phím Windows, Kiến trúc Database & Dashboard trong Notion, Tối ưu hiển thị & Driver cho Laptop Gaming (Acer Nitro 5).
- **Vấn đề / Câu hỏi:** 
  1. Phím Windows & Menu bị liệt đột ngột trên Windows.
  2. Xây dựng hệ thống quản lý công việc với Linked Views, Filter, Sort trong Notion.
  3. Lỗi giật hình (Stuttering) & bóng mờ (Ghosting) khi chuyển cửa sổ (`Alt+Tab`) trên Acer Nitro 5 (AN515-57).
- **Cách giải quyết / Logic lõi:** 
  **1. Khắc phục phím Windows/Menu:**
  - Kiểm tra phím `Win Lock` (biểu tượng ổ khóa/gạch chéo) hoặc tổ hợp `Fn + Win`.
  - Tắt phần mềm can thiệp: AutoHotkey, SharpKeys, phần mềm Gaming (Razer Synapse, Logitech G HUB).
  - Xóa Registry mapping: `Win + R` → `regedit` → `HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Keyboard Layout` → Xóa giá trị `Scancode Map` → Restart.
  - Gỡ cài đặt Driver bàn phím trong `Device Manager` → Restart để Windows tự nhận lại driver chuẩn.
  
  **2. Kiến trúc Notion Dashboard:**
  - Tạo DB gốc: `/table` → Full page. Đặt tên `[DB] Tên_DB`. Cấu hình Properties: `Status` (Inbox, Chưa làm, Đang làm, Hoàn thành), `Date`, `Select` (Ưu tiên), `Relation`, `Created time`.
  - Tạo Linked View: `/linked view` → Chọn DB gốc.
  - Cấu hình View: Đổi Layout sang `List`. Áp dụng `Filter` (VD: `Hạn chót Is Today` AND `Trạng thái Is not ✅ Hoàn thành`). `Sort` theo Ưu tiên. Ẩn cột thừa qua menu `...` → `Properties`.
  
  **3. Tối ưu hiển thị Acer Nitro 5:**
  - Tắt tính năng OEM: `Intel Graphics Command Center` → Tắt `Panel Self Refresh (PSR)`. `NitroSense` → Tắt `LCD Overdrive`.
  - Tắt `Hardware Acceleration` trên Chrome/Discord/Steam.
  - Cấu hình NVIDIA Control Panel: `G-SYNC` = On, `Vertical Sync` = On, `Max Frame Rate` = (Tần số quét màn hình - 3 FPS). Tắt V-Sync trong game.
  - Clean Install Driver: Tải DDU → Vào Safe Mode → Ngắt mạng → Chạy DDU (Clean NVIDIA → Restart → Clean Intel → Restart) → Cài driver OEM/Ổn định → Kết nối mạng.
- **Ghi chú quan trọng:** 
  - Chỉnh sửa Registry (`Scancode Map`) cần thận trọng, sao lưu trước khi xóa.
  - Trong Notion, dùng tiền tố `[DB]` để phân biệt Database gốc và Linked View. Filter/Sort chỉ áp dụng cho View, không làm thay đổi dữ liệu gốc.
  - Với Acer Nitro 5, hiện tượng giật khi `Alt+Tab` thường do xung đột DWM, PSR hoặc Overdrive. Luôn kiểm tra phần cứng bằng cách kết nối màn hình ngoài hoặc vào BIOS để loại trừ lỗi tấm nền/cáp eDP. DDU bắt buộc chạy ở Safe Mode và ngắt kết nối Internet để tránh Windows Update ghi đè driver.
