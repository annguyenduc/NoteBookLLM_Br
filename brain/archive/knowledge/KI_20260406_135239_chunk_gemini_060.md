# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Thiết lập môi trường VS Code + PlatformIO cho mạch OhStem YOLO UNO (ESP32-S3) và triển khai Edge AI/TinyML.
- **Vấn đề / Câu hỏi:** PlatformIO không hỗ trợ mặc định mạch YOLO UNO. Làm sao cấu hình board tùy chỉnh, quản lý dự án chuyên nghiệp và tích hợp mô hình AI lên vi điều khiển?
- **Cách giải quyết / Logic lõi:** 
  1. **Tạo định nghĩa Board tùy chỉnh (`boards/yolo_uno.json`):**
     ```json
     {
       "build": { "mcu": "esp32s3", "f_cpu": "240000000L", "variant": "esp32s3_devkitc_1", "frameworks": ["arduino", "espidf"] },
       "upload": { "flash_size": "4MB", "maximum_ram_size": 327680, "maximum_size": 4194304, "speed": 921600 },
       "name": "OhStem YOLO UNO", "vendor": "OhStem"
     }
     ```
  2. **Cấu hình dự án (`platformio.ini`):**
     ```ini
     [env:yolo_uno]
     platform = espressif32
     board = yolo_uno
     framework = arduino
     boards_dir = boards
     monitor_speed = 115200
     lib_deps = adafruit/Adafruit NeoPixel@^1.12.0
     ```
  3. **Quy trình code chuẩn C++:** File `src/main.cpp` bắt buộc `#include <Arduino.h>`, khai báo hàm rõ ràng trước khi gọi. Sử dụng thanh công cụ PlatformIO để Build/Upload/Monitor.
  4. **Tích hợp Edge AI:** Dùng Edge Impulse thu thập dữ liệu & huấn luyện mô hình, xuất dưới dạng Arduino Library, copy vào thư mục `lib/` của dự án. Gọi hàm `run_classifier()` để suy luận.
- **Ghi chú quan trọng:** 
  - YOLO UNO dùng ESP32-S3 (240MHz, 8MB PSRAM, Wi-Fi/BLE), tương thích form factor Arduino UNO nhưng hiệu năng cao hơn hàng chục lần.
  - PlatformIO quản lý thư viện theo dự án (`lib_deps`), giải quyết triệt để "dependency hell" và đảm bảo khả năng tái tạo môi trường qua file cấu hình khai báo.
  - **Giới hạn AI:** Không thể chạy YOLOv8 trực tiếp trên ESP32-S3. Cần dùng kiến trúc tối ưu cho MCU như **FOMO** (Edge Impulse) hoặc TensorFlow Lite Micro.
  - Khi chuyển từ Arduino IDE, cần tuân thủ cú pháp C++ chuẩn: include tường minh, khai báo prototype hàm, đồng bộ `monitor_speed` trong `platformio.ini` với `Serial.begin()` trong code.
