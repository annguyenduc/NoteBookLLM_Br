# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Tích hợp AI nhận diện vật thể trên điện thoại thông minh để điều khiển Robot Rover (OhStem) cho giáo dục STEM.
- **Vấn đề / Câu hỏi:** Làm thế nào để kết nối điện thoại làm camera/bộ xử lý AI cho robot rover kit, lựa chọn giao thức truyền thông, framework AI di động và xây dựng logic điều khiển robot dựa trên kết quả nhận diện thời gian thực?
- **Cách giải quyết / Logic lõi:** 
  - **Kiến trúc hệ thống:** Điện thoại đóng vai trò "Edge AI Brain" (thu thập ảnh, tiền xử lý, chạy suy luận, gửi lệnh). Robot (MCU như ESP32/Arduino) nhận lệnh và điều khiển động cơ/cảm biến.
  - **Lựa chọn Framework AI di động:**
    - `TensorFlow Lite (TFL)`: Tối ưu độ trễ, hỗ trợ GPU/DSP delegate, phù hợp mô hình tùy chỉnh.
    - `ML Kit`: API đơn giản, mô hình có sẵn, tích hợp nhanh nhưng ít tùy chỉnh danh mục vật thể.
    - `LiteRT`: Hỗ trợ đa framework (TF, PyTorch, JAX), linh hoạt cho nhà phát triển nâng cao.
  - **Giao thức truyền thông (Điện thoại ↔ Robot):**
    - `Bluetooth (BLE)`: Dễ setup, tiết kiệm pin, băng thông thấp → Chỉ dùng cho lệnh điều khiển cơ bản.
    - `Wi-Fi`: Băng thông cao, hỗ trợ video stream → Lý tưởng cho AI nhận diện thời gian thực.
    - `USB OTG`: Độ trễ thấp, băng thông rất cao → Phù hợp truyền video chất lượng cao/gỡ lỗi, hạn chế di động.
  - **Logic điều khiển dựa trên AI (Pseudo-code):**
    ```python
    # 1. Tiền xử lý & Suy luận
    frame = camera.capture()
    detections = ai_model.run(frame) # Trả về list [x, y, w, h, confidence]
    
    # 2. Tính toán lệch tâm so với trung tâm khung hình
    center_x = frame.width / 2
    target = max(detections, key=lambda d: d.confidence)
    obj_center = target.x + target.w / 2
    offset = obj_center - center_x
    
    # 3. Gửi lệnh điều khiển (Threshold tùy chỉnh)
    if abs(offset) < THRESHOLD:
        robot.send("FORWARD") # Vật thể đã căn giữa
    elif offset < 0:
        robot.send("TURN_LEFT")
    else:
        robot.send("TURN_RIGHT")
        
    # 4. Kiểm soát khoảng cách (nếu có cảm biến/LiDAR)
    if distance_to_object < SAFE_DISTANCE:
        robot.send("STOP")
    ```
- **Ghi chú quan trọng:** 
  - **Phần cứng:** Ưu tiên **OhStem xBot** do tích hợp sẵn ESP32-S3, Wi-Fi/BLE, Camera độ sâu & LiDAR, giảm đáng kể độ phức tạp lắp ráp.
  - **Tối ưu hiệu năng:** Sử dụng mô hình "lite" (Tiny-YOLOv3, MobileNet), áp dụng lượng tử hóa (quantization) để giảm RAM và tiêu thụ pin. Tắt màn hình điện thoại khi chạy nền để giảm nhiệt.
  - **An toàn & Giáo dục (K6-9):** Cần giám sát người lớn khi đấu nối điện. Bắt đầu từ điều khiển cơ bản (tiến/lùi/rẽ) trước khi tích hợp AI. Ưu tiên giao diện Web Server hoặc ứng dụng kéo thả để phù hợp lứa tuổi.
