# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Thiết kế & Triển khai Máy chủ Giao thức Ngữ cảnh Mô hình (MCP Server) bằng Python để tích hợp với IDE (Cursor), xử lý Google Sheets và định dạng PPTX.
- **Vấn đề / Câu hỏi:** Làm thế nào để xây dựng một MCP Server an toàn, có khả năng tải/xuất tệp từ Google Sheets, kiểm tra chính tả tiếng Việt, định dạng văn bản PPTX, và cấu hình để IDE nhận diện và gọi công cụ?
- **Cách giải quyết / Logic lõi:** 
  - **Kiến trúc MCP & SDK:** Sử dụng `FastMCP` từ `modelcontextprotocol/python-sdk`. Công cụ được định nghĩa qua decorator `@mcp.tool()` hoặc đăng ký thủ công. Giao tiếp cục bộ qua transport `stdio`.
  - **Xác thực Google API:** Dùng Service Account. Đường dẫn file JSON key được truyền qua biến môi trường `GOOGLE_APPLICATION_CREDENTIALS`. Scopes tối thiểu: `spreadsheets.readonly`, `drive.readonly`.
  - **Xử lý Google Sheets:**
    - Xuất tệp: Gọi Drive API `files.export` với `mimeType` tương ứng (`xlsx`, `csv`, `pdf`).
    - Trích xuất link: Phân tích `formulaValue` (hàm `=HYPERLINK()`) và `textFormatRuns.format.link` (rich text). Tải file qua `requests` hoặc Drive API (`files.get` + `alt=media`).
    - Đối tượng nhúng: API không hỗ trợ trực tiếp. Giải pháp: Xuất sheet sang PDF → dùng `PyMuPDF` trích xuất ảnh, hoặc đọc URL từ `=IMAGE()`.
  - **Xử lý PPTX (`python-pptx`):** Duyệt `slide.shapes`, lọc `shape.has_text_frame`. Áp dụng định dạng: `paragraph.line_spacing = 1.15`, `font.size = Pt(14/12/11)`. Gom đoạn: nối `run.text` của các paragraph trong cùng text box, xóa paragraph thừa.
  - **Cấu hình Cursor IDE (`~/.cursor/mcp.json`):**
    ```json
    {
      "mcpServers": {
        "googleSheetDownloader": {
          "command": "/path/to/venv/bin/python",
          "args": ["-m", "mcp_server.main"],
          "env": {
            "GOOGLE_APPLICATION_CREDENTIALS": "/path/to/service_account_key.json",
            "PYTHONPATH": "/path/to/project"
          }
        }
      }
    }
    ```
  - **Triển khai Tool MCP (Python):**
    ```python
    from mcp.server.fastmcp import FastMCP, Context
    mcp = FastMCP("GoogleSheetDownloader")
    ```

- **Ghi chú quan trọng:** 
  - 🔒 **Bảo mật:** Không bao giờ hardcode credentials. Dùng biến môi trường hoặc Secret Manager. Tuân thủ nguyên tắc đặc quyền tối thiểu (Least Privilege).
  - 🛡️ **Xử lý lỗi & Logging:** Bọc logic API trong `try-except` (bắt `googleapiclient.errors.HttpError`). Dùng `ctx.info()`, `ctx.error()` để log chuẩn MCP. Xử lý rate limit bằng exponential backoff.
  - 🧪 **Kiểm thử:** Sử dụng `MCP Inspector` (công cụ UI web chính thức) để debug tool invocation trực tiếp mà không cần viết client phức tạp.
  - ⚠️ **Giới hạn kỹ thuật:** Google Sheets API không cho phép tải trực tiếp ảnh nổi/biểu đồ dạng nhị phân. Cần chấp nhận giải pháp gián tiếp (xuất PDF) hoặc thông báo rõ giới hạn cho người dùng.
  - 🎓 **Ứng dụng giáo dục (Minecraft Education):** Có thể kết hợp làm dự án liên môn thông qua `Code Builder` (lập trình sự kiện), NPC AI, và chế độ Lớp học để trực quan hóa STEM, tái hiện lịch sử và phát triển kỹ năng mềm.
