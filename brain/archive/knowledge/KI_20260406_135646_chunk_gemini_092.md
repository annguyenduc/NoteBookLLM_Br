# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Quy trình chuyển đổi thiết kế Figma sang mã nguồn bằng Cursor AI
- **Vấn đề / Câu hỏi:** Làm thế nào để tự động hóa/quy trình hóa việc chuyển UI từ Figma sang code (React/HTML/CSS/Tailwind) bằng Cursor AI với chi phí thấp hoặc miễn phí?
- **Cách giải quyết / Logic lõi:** 
  Tồn tại 4 phương pháp chuyển đổi, xếp theo độ phức tạp & chất lượng:
  1. **Chụp màn hình + Prompt:** Nhanh, dùng cho prototype đơn giản. Code sinh ra tĩnh, không responsive.
  2. **Plugin Builder.io (Khuyên dùng):** Cân bằng tốt nhất giữa tốc độ & chất lượng. Plugin phân tích sâu cấu trúc Figma (`Auto Layout`, `Components`) để xuất code có cấu trúc, hỗ trợ đa framework. Có gói miễn phí đủ cho dự án cá nhân.
  3. **Tích hợp MCP Server (Nâng cao):** Kết nối trực tiếp Cursor với Figma API để truy vấn dữ liệu thiết kế thời gian thực. Độ chính xác cao nhất, tự đồng bộ tài sản (ảnh, font).
     Cấu hình MCP trong Cursor (`Settings > MCP > Add new global MCP server`):
     ```json
     {
       "mcpServers": {
         "figma-developer-mcp": {
           "command": "npx",
           "args": ["-y", "figma-developer-mcp", "--stdio"],
           "env": {
             "FIGMA_API_KEY": "<your-figma-api-key>"
           }
         }
       }
     }
     ```
     *Cách dùng:* Tạo Personal Access Token trên Figma → Dán vào config → Copy link selection từ Figma → Dán vào Cursor Agent → Prompt AI.
  4. **Figma Dev Mode:** Thủ công, chính xác từng pixel ở cấp độ snippet. Yêu cầu gói Figma trả phí, phù hợp để kiểm tra thông số hoặc tuân thủ Design System chặt chẽ.
- **Ghi chú quan trọng:** 
  - **Chất lượng đầu vào quyết định đầu ra:** Thiết kế Figma bắt buộc phải dùng `Auto Layout`, `Components`, `Variants` và đặt tên rõ ràng. Thiết kế lộn xộn (dùng absolute positioning) sẽ sinh code khó bảo trì.
  - **Tối ưu chi phí:** Sinh viên đăng ký gói Cursor Pro miễn phí (qua email `.edu`) + Gói miễn phí Builder.io + Figma Free là đủ stack để xây dựng portfolio chuyên nghiệp.
  - **Vai trò AI:** Cursor là "co-pilot", không thay thế tư duy lập trình. Developer cần nắm vững kiến trúc frontend để review, refactor và xử lý logic nghiệp vụ.
  - **Hỗ trợ & Cộng đồng:** Giao thức MCP còn mới, dễ gặp lỗi `sse error` hoặc `failed to create client`. Nên tham khảo giải pháp trên `/r/cursor`, `/r/FigmaDesign` hoặc nhóm `Cursor Meetup Saigon` / `Friends of Figma Hanoi`.
