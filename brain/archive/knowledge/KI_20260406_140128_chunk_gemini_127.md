# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Giao thức MCP (Model Context Protocol) & Công cụ Human-in-the-Loop (`interactive-feedback-mcp` / `mcp-feedback-enhanced`) tối ưu hóa quy trình AI Coding.
- **Vấn đề / Câu hỏi:** Chi phí API cao và giảm năng suất do AI tự động đoán ý định mơ hồ, gây lãng phí nhiều lệnh gọi công cụ (premium tool calls). Thiếu chuẩn hóa kết nối giữa AI Host và các công cụ/dữ liệu bên ngoài.
- **Cách giải quyết / Logic lõi:** 
  - **Mô hình Human-in-the-Loop:** AI tạm dừng khi gặp mơ hồ, gọi MCP server để hiển thị GUI hỏi người dùng, hợp nhất nhiều bước phỏng đoán thành 1 lệnh gọi có định hướng rõ ràng.
  - **Cấu hình Client (`mcp.json` trong Cursor/IDE):**
    ```json
    {
      "mcpServers": {
        "interactive-feedback-mcp": {
          "command": "uv",
          "args": ["--directory", "/absolute/path/to/repo", "run", "server.py"],
          "timeout": 600,
          "autoApprove": ["interactive_feedback"]
        }
      }
    }
    ```
  - **Quy tắc Prompting cho AI:** Luôn gọi `interactive_feedback` khi yêu cầu mơ hồ hoặc trước khi kết thúc task. Duy trì vòng lặp cho đến khi người dùng gửi phản hồi trống.
  - **Phát triển Server MCP tùy chỉnh (FastMCP):**
    ```python
    from fastmcp import FastMCP
    mcp = FastMCP(name="MyCustomServer")

    @mcp.tool
    def greet(name: str) -> str:
        """Trả về lời chào cá nhân hóa."""
        return f"Xin chào, {name}!"

    if __name__ == "__main__":
        mcp.run()
    ```
- **Ghi chú quan trọng:**
  - **Kiến trúc MCP:** Gồm Host (điều phối), Client (kết nối 1:1), Server (cung cấp Tools/Resources/Prompts). Chuyển đổi bài toán tích hợp từ M×N sang M+N. Hỗ trợ transport `stdio` (cục bộ) và `HTTP/SSE` (từ xa).
  - **Nâng cấp hệ sinh thái:** `mcp-feedback-enhanced` (dùng Tauri/Web UI) thay thế bản gốc nhờ hỗ trợ môi trường từ xa (SSH/WSL), quản lý prompt CRUD, theo dõi phiên và phản hồi đa phương tiện.
  - **Bảo mật:** MCP không tự động enforce bảo mật. Cần kiểm tra nguồn gốc server, dùng biến môi trường cho secret, áp dụng nguyên tắc đặc quyền tối thiểu, và xác thực/sanitize đầu vào nghiêm ngặt khi phát triển server tùy chỉnh.
  - **Chiến lược triển khai:** Kết hợp nhiều MCP server (Jira, GitHub, DB, Filesystem) để biến AI thành tác nhân điều phối đa công cụ, tự động hóa quy trình end-to-end.
