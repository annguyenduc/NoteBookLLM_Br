# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Cấu hình kết nối mạng (Wi-Fi/LAN) cho máy in 3D Elegoo Neptune 4, tích hợp với Bambu Studio/OrcaSlicer và xử lý chế độ ngoại tuyến.
- **Vấn đề / Câu hỏi:** Máy in không hiển thị menu Network, lỗi cấp IP APIPA (`169.254.x.x`), kết nối sai giao thức với Bambu Studio, thông báo lỗi Cloud khi offline, và cách lưu profile nhiệt độ nhựa tùy chỉnh.
- **Cách giải quyết / Logic lõi:** 
  1. **Cấu hình Wi-Fi qua USB (`wpa_supplicant`):**
     - Format USB `FAT32` → Tạo thư mục `ELEGOO_UPDATE_DIR` → Tạo file `wpa_supplicant-wlan0.conf` (tuyệt đối xóa đuôi `.txt` ẩn).
     - Nội dung cấu hình chuẩn:
       ```ini
       ctrl_interface=/var/run/wpa_supplicant
       ctrl_interface_group=0
       update_config=1
       network={
       ssid="Tên_Wifi"
       psk="Mat_Khau"
       key_mgmt=WPA-PSK
       }
       ```
     - Quy trình: Tắt nguồn → Cắm USB & Dongle Wi-Fi → Bật nguồn → Chờ 1-2 phút → Kiểm tra IP tại `Settings -> Network`.
  2. **Xử lý lỗi mạng & Firmware cũ:**
     - IP `169.254.x.x` hoặc `0.0.0.0`: Máy không kết nối được Router. Kiểm tra băng tần (Dongle cũ chỉ hỗ trợ 2.4GHz), mật khẩu, cổng USB.
     - Không có mục `Network` trên màn hình: Dấu hiệu firmware cũ. Giải pháp: Cắm dây LAN → Truy cập IP qua trình duyệt (Giao diện Fluidd) → Cấu hình Wi-Fi trực tiếp trên Web UI.
     - Lỗi `"No root firmware"`: Do bấm nút Update khi USB chỉ chứa file config. Hệ thống tự đọc file khi khởi động, không cần thao tác thủ công.
  3. **Kết nối Bambu Studio:**
     - Neptune 4 chạy Klipper/Fluidd. Trong Bambu Studio: `Host Type` chọn `OctoPrint` → Nhập `IP` máy in → `API Key` để trống → Test/Connect.
     - Khuyến nghị: Dùng **OrcaSlicer** (fork của Bambu Studio) để hỗ trợ native Klipper và có sẵn profile Neptune 4.
  4. **Làm việc Offline & Cấu hình nhựa:**
     - Tắt pop-up Cloud: Đăng xuất tài khoản Bambu Studio → Chuyển sang chế độ Guest/Offline.
     - In offline: `Export G-code` → Lưu vào thẻ nhớ `FAT32` → In trực tiếp trên màn hình máy.
     - Lưu profile nhiệt độ: Sau khi test tháp nhiệt độ, vào `Prepare` → Chọn nhựa → Bấm icon bút (Edit preset) → Tab `Filament` → Chỉnh `Printing Temperature` (Initial/Other layers) → Save as new profile. Dùng `Generic` cho hãng không có sẵn.
- **Ghi chú quan trọng:**
  - **Tương thích Dongle Wi-Fi:** Chỉ nhận chip Realtek `RTL8188GU` hoặc `RTL8188EUS` (VD: TP-Link TL-WN725N V3). Kiểm tra `Hardware IDs` (`VID_0BDA&PID_8179` hoặc `B711`).
  - **Lỗi đuôi file ẩn:** Windows thường tự thêm `.txt`. Bắt buộc bật `View -> File name extensions` để xóa thủ công trước khi nạp vào máy.
  - **Ổn định in ấn:** Wi-Fi chập chờn dễ gây lỗi dừng giữa chừng (layer shift). Ưu tiên dùng dây LAN hoặc Repeater có cổng LAN nếu sóng yếu.
  - **LAN Only Mode:** Nếu có Router nhưng không Internet, bật chế độ này trên máy in và dùng `Access Code` để kết nối cục bộ qua Bambu Studio.
