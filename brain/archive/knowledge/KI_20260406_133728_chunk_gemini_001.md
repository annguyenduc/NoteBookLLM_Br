# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Xuất & xử lý dữ liệu hội thoại Google Gemini; Kiến trúc tự động hóa Obsidian bằng AI (Claude Code / Antigravity & MCP).
- **Vấn đề / Câu hỏi:** 
  1. Nhầm lẫn khi dùng Google Takeout: chỉ xuất được cấu hình Gems, không lấy được lịch sử chat Web.
  2. Lỗi import file `.txt` từ Takeout do định dạng thực tế là JSON phức tạp.
  3. Thiết kế luồng "Bộ não thứ hai" với Obsidian, chuyển đổi từ Claude Code sang Antigravity (MCP).
- **Cách giải quyết / Logic lõi:** 
  - **Xuất dữ liệu Gemini chính xác:**
    - *Phương án 1 (Khuyên dùng):* Takeout > `Hoạt động của tôi (My Activity)` > `Bao gồm tất cả dữ liệu hoạt động` > `Bỏ chọn tất cả` > Tích chọn `Gemini`/`Gemini Apps`. Output: `Takeout/My Activity/Gemini Apps/My Activity.html`.
    - *Phương án 2:* Takeout > Mục `Gemini` > Nhấn `Tất cả dữ liệu Gemini đi kèm` > Tick `app_conversation_history`. Output: `Takeout/Gemini/app_conversation_history.json|.html`.
    - *Phân biệt nguồn:* `Gemini` = Chat trên web; `Gemini in Workspace` = Chat nhúng trong Sheets/Docs/Drive.
  - **Xử lý định dạng file Takeout (JSON ẩn trong .txt):**
    - Đổi đuôi `.txt` → `.json` hoặc mở trực tiếp bằng trình duyệt/JSON Editor.

  - **Kiến trúc Obsidian + AI (Antigravity/MCP):**
    - *Cầu nối dữ liệu:* Cài `Obsidian MCP Server` + bật plugin `Local REST API` trong Obsidian. Cấu hình Antigravity trỏ đến MCP endpoint (JSON config).
    - *Pipeline xử lý tài liệu:* PDF → Chunking (PyMuPDF) → LLM (Gemini 1.5 Pro/Flash) trích xuất ý chính → Output Markdown → MCP đẩy tự động vào thư mục Obsidian.
    - *Agent Workflow:* Antigravity đóng vai trò Query Layer & Autonomous Agent. Hỗ trợ lệnh `/daily`, `/tldr`, tự động sinh file `.canvas` (JSON Canvas) để trực quan hóa mối liên hệ ghi chú.
- **Ghi chú quan trọng:** 
  - Antigravity dùng chuẩn **MCP (Model Context Protocol)** cho phép kết nối đa nền tảng (Drive, GitHub, Slack), context window 1-2M tokens vượt trội so với Claude Code (200k).
  - Dữ liệu Takeout chỉ tồn tại khi `Gemini Apps Activity` đang bật. Lịch sử đã xóa thủ công sẽ không khôi phục được.
  - Link "Pro AI 4 tháng 0 đồng" là cơ chế tặng quà/engagement farming, người gửi **không nhận thưởng vật chất**. Chỉ truy cập link từ domain chính thức `one.google.com` hoặc `g.co` để tránh phishing.
