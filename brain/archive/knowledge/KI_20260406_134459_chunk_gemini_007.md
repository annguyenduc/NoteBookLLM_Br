# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Thao tác Git trên di động, Quản lý & Hoàn tiền gói Google/Gemini, Tích hợp & Chấm điểm tự động trên E-learning (LMS), Cấu hình AI Agent trong Antigravity IDE, Chiến lược dữ liệu cho giảng dạy Machine Learning.
- **Vấn đề / Câu hỏi:** Làm thế nào để clone repo, xử lý hoàn tiền/xác minh sinh viên, import câu hỏi hàng loạt, thu thập kết quả từ file HTML độc lập, gọi/configure `.agents/` trong IDE, và tối ưu data training cho học sinh?
- **Cách giải quyết / Logic lõi:** 
  - **Git trên Mobile:** 
    - Terminal (Android/Termux): `pkg install git` → `git clone https://github.com/tranhieutt/for-learn-claude-cli.git`
    - Web Editor: Thay `github.com` bằng `github.dev` trong URL để mở giao diện VS Code nền web.
    - IDE Mobile: Spck, Replit (Android/iOS), Working Copy (iOS).
  - **Chính sách & Xác minh Google/Gemini:**
    - Hoàn tiền: Google Play (<48h) qua `play.google.com` > `Báo cáo sự cố`. iOS qua `reportaproblem.apple.com`. Quá hạn: Chat Live Support Google One (nhân viên có quyền duyệt ngoại lệ).
    - Re-verify Student (SheerID): Thực hiện trước 30 ngày hết hạn hoặc ngay khi hết hạn. Nếu ẩn nút, bấm `Get student offer` hoặc dùng link trong email SheerID/Google One.
  - **Import & Chấm điểm E-learning:**
    - *Aiken Format (`.txt`):* Câu hỏi 1 dòng. Đáp án: `A. Nội dung`. Dòng cuối: `ANSWER: A`
    - *CSV/Excel:* Điền theo template hệ thống (cột loại câu hỏi, nội dung, options, đáp án đúng, giải thích).
    - *SCORM:* Dùng iSpring/Articulate export `.zip` → Upload lên LMS để tự đồng bộ điểm & tiến độ.
  - **Thu thập kết quả từ HTML Standalone:**
    - Cách 1: Nhúng Google Forms qua `<iframe>`.
    - Cách 2: Google Apps Script tạo Web App API → HTML dùng `fetch()` POST dữ liệu về Google Sheets.
    - Cách 3: Đóng gói SCORM upload lên LMS.
  - **Antigravity IDE `.agents/`:**
    - Cấu trúc: `.agents/skills/`, `.agents/workflows/`, file cấu hình `SKILL.md` (JSON/YAML).
    - Cú pháp gọi: `@<tên_skill>` (áp dụng ngữ cảnh/quy tắc), `/<tên_workflow>` (chạy pipeline tự động).
    - Phím tắt: `Ctrl+L` (focus input), `Ctrl+Shift+M` (toggle Agent View), `Ctrl+Shift+P` (Command Palette).
    - Hỗ trợ MCP Server để kết nối công cụ ngoài.
  - **Chiến lược Data cho ML (Giáo dục):** Ưu tiên chất lượng & đa dạng > số lượng. Chia nhóm thu thập (5-10 mẫu/người). Dùng dataset mẫu có sẵn hoặc script Python tiền xử lý/crawl.
- **Ghi chú quan trọng:** 
  - File HTML standalone chạy offline hoàn toàn, không cần server/backend.
  - Khi hủy gói Google One, dung lượng giảm về 15GB. Cần dọn dẹp Drive/Photos trước khi hết hạn để tránh chặn nhận email.
  - Antigravity IDE tự động quét thư mục `.agents/` ở thư mục gốc (root) dự án. Đảm bảo file cấu hình đúng định dạng để Agent tự kích hoạt.
  - Xác minh SheerID không bắt buộc chờ hết hạn; nên thực hiện sớm để tránh gián đoạn dịch vụ.
