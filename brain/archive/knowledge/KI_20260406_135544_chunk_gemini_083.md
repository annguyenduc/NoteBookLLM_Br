# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Lập trình MakeCode (TypeScript) cho Minecraft Education Edition & Tối ưu hóa logic điều khiển Agent.
- **Vấn đề / Câu hỏi:** 
  1. Code vẽ hình lục giác lồng nhau bị sai hướng di chuyển, lặp code (vi phạm DRY), và `teleport` không hiệu quả.
  2. Hàm `agentBuildOctagon` bị cảnh báo `implicitly has an 'any' type` do thiếu khai báo kiểu dữ liệu tường minh cho tham số.
- **Cách giải quyết / Logic lõi:** 
  **1. Tách hàm & Chuẩn hóa logic vẽ hình lục giác:**
  ```typescript
  function VeLucGiacRong(canh: number): void {
      if (canh <= 0) return;
      // 6 cạnh đối xứng tạo hình lục giác khép kín
      for (let i = 0; i < canh; i++) { agent.move(UP, 1); agent.move(RIGHT, 1); agent.place(FORWARD); }
      for (let i = 0; i < canh; i++) { agent.move(LEFT, 1); agent.place(FORWARD); }
      for (let i = 0; i < canh; i++) { agent.move(DOWN, 1); agent.move(LEFT, 1); agent.place(FORWARD); }
      for (let i = 0; i < canh; i++) { agent.move(DOWN, 1); agent.move(RIGHT, 1); agent.place(FORWARD); }
      for (let i = 0; i < canh; i++) { agent.move(RIGHT, 1); agent.place(FORWARD); }
      for (let i = 0; i < canh; i++) { agent.move(UP, 1); agent.move(LEFT, 1); agent.place(FORWARD); }
  }
  ```
  **2. Khắc phục cảnh báo TypeScript & Logic kiểm tra tọa độ hình bát giác:**
  ```typescript
  // Thêm type annotation tường minh
  function agentBuildOctagon(n: number, k: number, colors: number[]) { ... }

  // Điều kiện toán học xác định điểm thuộc hình bát giác
  const cornerSum = cut - 1;
  ```

- **Ghi chú quan trọng:** 
  - Luôn khai báo kiểu dữ liệu (`: number`, `: number[]`) trong MakeCode/TypeScript để tránh lỗi biên dịch ngầm định và hỗ trợ gợi ý code.
  - Tuân thủ nguyên tắc DRY: Tách logic lặp lại thành hàm độc lập, dùng vòng lặp chính để truyền tham số kích thước/vị trí.
  - Khi điều khiển Agent vẽ hình học 2D/3D, cần tính toán vector di chuyển đối xứng và đảm bảo hình khép kín để tránh lệch khối.
  - *(Bổ trợ)* Mô hình kinh doanh sản phẩm số (template, checklist, stock) tận dụng AI để nghiên cứu thị trường & đóng gói combo, phù hợp cho người mới bắt đầu với vốn thấp và khả năng tự động hóa.
