# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Kiến trúc hệ thống AI Agent & Triển khai hệ thống Skill-Workflow tự trị trên Google Antigravity IDE.
- **Vấn đề / Câu hỏi:** Làm thế nào để lựa chọn đúng kho lưu trữ kỹ năng, tránh hiện tượng bão hòa ngữ cảnh (Context Saturation/Tool Bloat), và thiết lập quy trình làm việc tự động (autonomous pipeline) trên Antigravity IDE?
- **Cách giải quyết / Logic lõi:** 
  1. **Lựa chọn nền tảng:** Ưu tiên `tody-agent/codymaster` (Universal Skills Framework) thay vì `for-learn-claude-cli` (mã nguồn rò rỉ mang tính học thuật). `codymaster` cung cấp >60 kỹ năng đóng gói sẵn, tương thích native với Antigravity/Gemini CLI.
  2. **Kiến trúc phân lớp tri thức (Chống Context Saturation):**
     - `Rules` (`.agent/rules/`): Ràng buộc thụ động, luôn bật (bảo mật, coding style).
     - `Skills` (`.agents/skills/`): Phương pháp luận thực thi. Áp dụng cơ chế **Progressive Disclosure** (chỉ đọc nội dung khi ngữ cảnh khớp, tiết kiệm token).
     - `Workflows` (`.agent/workflows/`): Chuỗi tác vụ chủ động, kích hoạt bằng lệnh `/tên-workflow`, hỗ trợ gọi lồng ghép & phân nhánh.
  3. **Quy trình triển khai thực tế:**
     - **Cấu trúc thư mục chuẩn:** `.agents/skills/`, `.agent/workflows/`, `agents.md` (tại root).
     - **Cài đặt kỹ năng:**
       ```bash
       gemini extensions install https://github.com/tody-agent/codymaster
       # Hoặc clone thủ công vào <workspace-root>/.agents/skills/<tên-kỹ-năng>/
       ```
     - **Cấu hình `SKILL.md` (bắt buộc):**
       ```yaml
       ---
       name: cm-tdd
       description: Kỹ năng chuyên sâu phục vụ Lập trình Hướng Kiểm thử. Kích hoạt khi cần viết Unit Test, phân tích logic hoặc tối ưu code coverage.
       ---
       ```
       *(Phần thân Markdown chứa quy trình thực thi chi tiết, ví dụ: Red-Green-Refactor, giao thức xử lý lỗi)*
     - **Tạo Workflow điều phối:** Định nghĩa tệp `.agent/workflows/build-feature.md` với các bước tuần tự, gọi kỹ năng tương ứng, thiết lập điều kiện rẽ nhánh.
     - **Kích hoạt & Giám sát:** Gõ `/build-feature` trong Agent Chat. Hệ thống tự động phân tích, kích hoạt Progressive Disclosure, và điều phối bất đồng bộ qua Agent Manager/Sub-agents.
- **Ghi chú quan trọng:** 
  - Tuyệt đối tuân thủ định dạng YAML frontmatter cho `SKILL.md` (`name` phải viết thường, dùng dấu gạch nối).
  - Tránh nhồi nhét MCP Tools trực tiếp vào context; hãy dùng Skills làm "bộ não" điều hướng công cụ ("less infrastructure, more intelligence").
  - Workflows giới hạn an toàn ~12.000 ký tự/tệp để tránh tràn bộ nhớ ngữ cảnh.
  - Antigravity hỗ trợ cơ chế tự chữa lành (Self-Healing) và Watchdog System, cho phép chạy nhiều workflow song song mà không cần can thiệp thủ công.
