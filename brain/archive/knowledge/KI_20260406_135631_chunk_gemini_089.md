# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Lập trình Agent Minecraft (Xây hình lục giác), Nguyên lý cảm biến siêu âm HC-SR04, Đặc điểm cảm xúc & Can thiệp cho trẻ Rối loạn phổ tự kỷ (ASD).
- **Vấn đề / Câu hỏi:** 
  1. Logic thuật toán xây hình lục giác đặc cho Agent.
  2. Phân biệt chức năng mắt phát/thu và nguyên lý hoạt động của cảm biến siêu âm.
  3. Biểu hiện cảm xúc (ăn vạ/khủng hoảng tuổi lên 3) ở trẻ tự kỷ và cách phân biệt với trẻ phát triển điển hình.
- **Cách giải quyết / Logic lõi:** 
  - **Code Agent (Xây lục giác):** Chia cấu trúc thành 3 pha lặp theo chiều cao, điều chỉnh biến `currentWidth` để tạo độ dốc:
    ```javascript
    // PHA 1: Tam giác trên (Mở rộng)
    for (let index = 0; index < canh - 1; index++) {
        // ... đặt khối theo currentWidth ...
        currentWidth += 2; // Tăng 2 ô mỗi hàng
        agent.teleport(homePos, WEST); agent.move(RIGHT, 1);
    }
    // PHA 2: Hình chữ nhật giữa (Giữ nguyên)
    for (let index = 0; index < canh; index++) {
        // ... đặt khối theo currentWidth ...
        agent.teleport(homePos, WEST); agent.move(RIGHT, 1);
    }
    // PHA 3: Tam giác dưới (Thu hẹp)
    for (let k = 0; k <= canh - 2; k++) {
        currentWidth -= 2; // Giảm 2 ô mỗi hàng
        agent.move(FORWARD, 1); // Lùi vào 1 ô để căn giữa
        // ... đặt khối ...
        agent.teleport(homePos, WEST); agent.move(RIGHT, 1);
    }
    ```
  - **Cảm biến HC-SR04:** 
    - Mắt **T** (Transmitter): Phát sóng siêu âm 40kHz khi chân `Trig` nhận xung HIGH >10µs.
    - Mắt **R** (Receiver): Thu sóng phản xạ. Chân `Echo` giữ mức HIGH trong thời gian sóng đi-về.
    - Công thức tính: `Khoảng cách (cm) = (Thời gian Echo (µs) / 2) / 29.1`.
  - **Trẻ ASD & Cảm xúc:** 
    - `Tantrum` (Ăn vạ): Hành vi có mục đích (đòi hỏi/tránh né), trẻ vẫn quan sát phản ứng người lớn, dừng khi đạt yêu cầu.
    - `Meltdown` (Bùng nổ): Phản ứng mất kiểm soát do quá tải giác quan/cảm xúc hoặc thay đổi thói quen. Trẻ không thể tự dừng, cần đưa ra khỏi môi trường kích thích.
- **Ghi chú quan trọng:** 
  - Trong code Agent, việc lưu `homePos` và `teleport` về vị trí này sau mỗi hàng là bắt buộc để đảm bảo các hàng thẳng hàng, tránh lệch khối do sai số di chuyển tích lũy.
  - HC-SR04 hoạt động kém với vật liệu hấp thụ âm (vải, xốp, thảm) hoặc bề mặt cong/lồi lõm làm tán xạ sóng.
  - Với trẻ ASD, "ăn vạ" thực chất là hình thức giao tiếp thay thế do hạn chế ngôn ngữ. Can thiệp hiệu quả cần ưu tiên phòng ngừa quá tải, dạy kỹ năng giao tiếp thay thế (PECS, hình ảnh) và tuyệt đối tránh trừng phạt trong cơn `Meltdown`.
