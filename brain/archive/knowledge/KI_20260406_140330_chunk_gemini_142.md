# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Kiến trúc & Quy trình phát triển ứng dụng web Quản lý Dự án STEM (Python + Flask) tối ưu chi phí, kết hợp chiến lược prompt cho AI Coding Assistant (Cursor).
- **Vấn đề / Câu hỏi:** Làm thế nào để thiết kế database, triển khai các tính năng cốt lõi (Auth, phân quyền, nộp/chấm bài) và tối ưu chi phí hạ tầng, đồng thời xây dựng quy trình làm việc hiệu quả với Cursor AI?
- **Cách giải quyết / Logic lõi:** 
  - **Stack công nghệ tối ưu chi phí:** Backend: `Flask` + `Flask-SQLAlchemy` + `Flask-Login` + `Flask-WTF`. Database: `SQLite` (file-based, miễn phí). Frontend: `Jinja2` + `Bootstrap 5`. Deployment: `PythonAnywhere`/`Render` (Free Tier). Lưu trữ file: Thư mục `uploads/` cục bộ hoặc khuyến khích nhúng link Google Drive/YouTube cho file nặng.
  - **Thiết kế Database (SQLAlchemy Models):**
    - `User`: `id`, `username`, `email`, `password_hash`, `role` (`student`/`teacher`). Tích hợp `werkzeug.security` để băm/kiểm tra mật khẩu.
    - `Subject`: `id`, `name` (Môn học).
    - `ProjectAssignment`: `id`, `title`, `description`, `due_date`, `subject_id` (FK), `teacher_id` (FK).
    - `Submission`: `id`, `submission_date`, `file_path`, `notes`, `project_assignment_id` (FK), `student_id` (FK).
    - `Grade`: `id`, `score`, `feedback`, `grading_date`, `submission_id` (FK, unique), `teacher_id` (FK).
  - **Luồng xử lý cốt lõi (Code Logic):**
    ```python
    # 1. Cấu hình DB & App
    app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///instance/site.db'
    db = SQLAlchemy(app)
    login_manager = LoginManager(app)

    # 2. Model User (Hash password)
    class User(db.Model, UserMixin):
        # ... fields ...
        def set_password(self, pwd): self.password_hash = generate_password_hash(pwd)
        def check_password(self, pwd): return check_password_hash(self.password_hash, pwd)

    # 3. Route Upload & Lưu Submission
    @app.route('/submit/<int:pid>', methods=['GET','POST'])
    @login_required
    def submit(pid):
        if request.method == 'POST':
            file = request.files['file']
            if file:
                filename = secure_filename(file.filename)
                file.save(os.path.join(app.config['UPLOAD_FOLDER'], filename))
                new_sub = Submission(project_assignment_id=pid, student_id=current_user.id, file_path=filename)
                db.session.add(new_sub)
                db.session.commit()
    ```
  - **Chiến lược Prompt cho Cursor:** Chia nhỏ theo giai đoạn (Setup → Models → Auth → Student → Teacher → UI). Luôn cung cấp ngữ cảnh (model/code cũ) trước khi yêu cầu tính năng mới. Yêu cầu giải thích code nếu không rõ.
- **Ghi chú quan trọng:** 
  - Ưu tiên `SQLite` cho giai đoạn đầu, chỉ nâng cấp lên PostgreSQL/MySQL khi cần scale hoặc có nhiều concurrent users.
  - Form upload bắt buộc có `enctype="multipart/form-data"`.
  - Phân quyền đơn giản bằng kiểm tra `current_user.role` trong route hoặc dùng decorator `@login_required` kết hợp điều kiện.
  - Luôn chạy thử, debug từng bước sau mỗi prompt. Sử dụng `flash()` để thông báo trạng thái cho người dùng.
  - Tận dụng Free Tier của các dịch vụ cloud, tránh lưu trữ file video/dung lượng lớn trực tiếp lên server ứng dụng để giảm chi phí băng thông & storage.
