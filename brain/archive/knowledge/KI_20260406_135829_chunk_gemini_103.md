# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Sử dụng `coursera-dl` tải khóa học & Tối ưu hóa cấu hình MCP (Model Context Protocol) trong Cursor IDE.
- **Vấn đề / Câu hỏi:** 
  1. Cách xác định đúng định danh khóa học (slug) và thực thi `coursera-dl` trên Windows/Cursor IDE.
  2. Cấu hình MCP an toàn, tránh rò rỉ token và tối ưu chi phí token khi AI gọi công cụ.
- **Cách giải quyết / Logic lõi:** 
  **1. Tải khóa học với `coursera-dl`:**
  - **Lấy Slug:** Trích xuất phần cuối URL khóa học: `https://www.coursera.org/learn/[slug]` (VD: `project-management-foundations`).
  - **Lấy Cookie CAUTH:** DevTools (`F12`) > Tab `Application`/`Storage` > `Cookies` > `https://www.coursera.org` > Copy giá trị `CAUTH`.
  - **Lệnh thực thi (Tương thích PowerShell/Cursor IDE):**
    ```bash
    coursera-dl --cauth [GIÁ_TRỊ_COOKIE_CAUTH] [TÊN_SLUG_KHÓA_HỌC]
    ```
  
  **2. Tối ưu cấu hình MCP (Mô hình Hub-and-Spoke):**
  - **Vấn đề:** Định nghĩa trực tiếp nhiều tool trong `mcp.json` gây lãng phí token nền cho mỗi prompt.
  - **Giải pháp:** Tách cấu hình thành Hub (trung tâm điều phối & hỏi xác nhận) và Spokes (công cụ phụ).
  - **File `mcp.json` chính (Chỉ chứa Hub & Biến môi trường):**
    ```json
    {
      "mcpServers": {
        "feedback-hub": {
          "command": "uvx",
          "args": ["mcp-feedback-enhanced@latest", "--config-file", "~/.cursor/hub-tools.json"],
          "env": { "GITHUB_PERSONAL_ACCESS_TOKEN": "${env:GITHUB_TOKEN}" },
          "timeout": 600
        }
      }
    }
    ```
  - **File `hub-tools.json` (Chứa định nghĩa các tool phụ):**
    ```json
    {
      "mcpServers": {
        "code-quality": { "command": "python.exe", "args": ["code_quality_server.py"], "timeout": 120 },
        "filesystem": { "command": "npx", "args": ["-y", "@modelcontextprotocol/server-filesystem"], "timeout": 120 }
      }
    }
    ```
  **3. Cấu hình MCP cơ bản cho Cursor (Dùng `npx`):**
    ```json
    {
      "mcpServers": {
        "file-lister": { "command": "npx", "args": ["-y", "@cursorless/mcp-file-lister@latest"] },
        "shell-runner": { "command": "npx", "args": ["-y", "@cursorless/mcp-shell-runner@latest"] }
      }
    }
    ```
- **Ghi chú quan trọng:** 
  - 🔒 **Bảo mật:** Tuyệt đối không hardcode token/secret vào file cấu hình. Luôn dùng cú pháp `${env:TÊN_BIẾN}` để Cursor đọc từ biến môi trường hệ thống.
  - ⚡ **Hiệu quả Token:** Mô hình Hub-and-Spoke tiết kiệm token cho ~95% tác vụ thông thường. Chỉ nên "ghim" 2-3 tool hay dùng trực tiếp vào `mcp.json` chính nếu cần tốc độ phản hồi tức thì (1 lượt gọi AI).
  - 💻 **Môi trường:** `coursera-dl` chạy trên Python, tương thích hoàn toàn với Terminal tích hợp trong Cursor IDE (mặc định là PowerShell trên Windows 11).
