# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Kiến trúc chuyển đổi Python-Blockly (MakeCode), Tinh chỉnh bộ điều khiển PID, Kỹ thuật Web Scraping động & Quản lý tài nguyên Cursor IDE.
- **Vấn đề / Câu hỏi:** Làm thế nào để xây dựng bộ chuyển đổi mã Python sang khối Blockly, tinh chỉnh PID cho robot, thu thập dữ liệu từ trang web động an toàn, và xử lý khi hết tín dụng AI trên Cursor?
- **Cách giải quyết / Logic lõi:** 
  - **Chuyển đổi Python ↔ Blockly:** Dùng mô hình AST chung. Parse mã Python bằng `ast` module → Duyệt cây → Ánh xạ 1-1 sang node Blockly JSON/XML qua bảng mapping API → Xử lý cấu trúc lồng (`<statement>`, `<next>`) và biến (`<variables>`). Code ngoài phạm vi hỗ trợ chuyển thành "khối xám" (grey block).
  - **Tinh chỉnh PID:** Tuần tự: (1) `Ki=Kd=0`, tăng `Kp` đến khi dao động nhẹ rồi giảm ~30%. (2) Tăng `Kd` để triệt tiêu overshoot. (3) Tăng `Ki` nhỏ để loại bỏ sai số ổn định.
  - **Web Scraping động:** Selenium/Puppeteer render JS + BeautifulSoup parse DOM. Vượt anti-bot bằng: Proxy xoay vòng, delay ngẫu nhiên, User-Agent hợp lệ, trình duyệt anti-detect. Thiết kế module hóa để dễ bảo trì khi DOM/API thay đổi.
  - **Quản lý Cursor IDE:** Xóa custom mode qua `AI Chat > Settings > Delete`. Hết $20 credit: Chuyển sang chế độ `Auto` (mô hình chuẩn miễn phí), bật `Usage-Based Pricing`, hoặc nâng cấp gói.
  - **Code lõi (Scraping & AST Mapping):**
  ```python
  # 1. Cấu hình Selenium cơ bản
  from selenium import webdriver
  from bs4 import BeautifulSoup
  import time, random

  opts = webdriver.ChromeOptions()
  opts.add_argument("--headless")
  opts.add_argument("user-agent=Mozilla/5.0 (Windows NT 10.0; Win64; x64)...")
  driver = webdriver.Chrome(options=opts)

  # 2. Luồng scrape trang động
  def scrape_dynamic(url):
      driver.get(url)
      time.sleep(random.uniform(3, 7)) # Mô phỏng hành vi người
      soup = BeautifulSoup(driver.page_source, 'html.parser')
      return [el.text for el in soup.select('div.voucher-card')]

  # 3. Logic ánh xạ AST sang Blockly (Pseudocode)
  def ast_to_blockly(node, mapping_table):
      if node.type in mapping_table:
          block = mapping_table[node.type]
          block.fields = extract_params(node)
          if node.has_body:
              block.statements = [ast_to_blockly(c, mapping_table) for c in node.body]
          return block
  ```
- **Ghi chú quan trọng:** 
  - MakeCode Python thực chất là **Static TypeScript** mang cú pháp Python, không phải MicroPython thuần. Chỉ hỗ trợ tập con cú pháp nhất định.
  - PID: Luôn tinh chỉnh từng tham số một, ghi log kết quả. `Kp` (phản ứng hiện tại), `Ki` (tích lũy quá khứ), `Kd` (dự đoán tốc độ thay đổi).
  - Scraping: Tuân thủ **NĐ 13/2023/NĐ-CP**, tuyệt đối không thu thập PII. Kiểm tra `robots.txt` & ToS. Scraper cần cơ chế giám sát & cảnh báo lỗi tự động do cấu trúc web thay đổi liên tục.
  - Cursor: Chế độ `Auto` là giải pháp tiết kiệm nhất khi hết credit, vẫn đảm bảo workflow cơ bản không gián đoạn.
