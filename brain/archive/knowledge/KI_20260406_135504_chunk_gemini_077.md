# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Tự động hóa xây dựng & vận hành trang trại mía trong Minecraft Education Edition sử dụng MakeCode (kết hợp Code Builder & Agent).
- **Vấn đề / Câu hỏi:** Làm thế nào để tự động hóa việc xây dựng cơ chế thu hoạch mía (Piston + Observer + Redstone) và hệ thống nền móng một cách hiệu quả, tối ưu hóa phân chia tác vụ giữa lệnh khối (`/fill`) và Agent, đồng thời đảm bảo khả năng mở rộng quy mô dễ dàng?
- **Cách giải quyết / Logic lõi:** 
  - **Học thuyết phân chia tác vụ:** 
    - `blocks.fill()`: Xử lý khối lượng lớn, cấu trúc đồng nhất (san phẳng, nền đất/cát, kênh nước, tường bao). Tốc độ tức thời, không phụ thuộc hướng.
    - `Agent`: Xử lý lắp ráp tuần tự, yêu cầu định hướng khối chính xác (Piston, Observer, khối nền Redstone, dây Redstone, trồng mía).
  - **Cơ chế thu hoạch:** Observer (`y+2`) phát hiện mía mọc → kích hoạt tín hiệu Redstone → Piston (`y+1`) đẩy gãy ngọn mía → Nước cuốn item về Hopper/Chest.
  - **Code MakeCode cốt lõi (JavaScript - rút gọn):**
    ```javascript
    // 1. Nạp kho Agent theo slot cố định
    function stockAgent() {
        agent.setItem(SAND, 64, 1); agent.setItem(PISTON, 64, 2);
        agent.setItem(OBSERVER, 64, 3); agent.setItem(REDSTONE_DUST, 64, 4);
        agent.setItem(STONE, 64, 5); agent.setItem(SUGAR_CANE, 64, 6);
    }

    // 2. Xây dựng hàng loạt nhanh (Place-on-Move)
    function buildHarvestingSystem() {
        agent.teleport(world(startX, startY, startZ), NORTH);
        // Đặt Piston
        agent.setSlot(2); agent.setAssist(PLACE_ON_MOVE, true);
        agent.move(EAST, length); agent.setAssist(PLACE_ON_MOVE, false);
        
        // Đặt khối nền & Redstone (đổi slot, di chuyển ngược hướng, place DOWN)
        // ...
        
        // Đặt Observer kích hoạt ở cuối hàng
        agent.teleport(world(startX + length, startY + 2, startZ), DOWN);
        agent.place(DOWN);
    }

    // 3. Điều phối chính
    player.onChat("farm", function () {
        agent.teleportToPlayer(); stockAgent();
        buildBaseAndCollection(); // Dùng blocks.fill()
        buildHarvestingSystem();  // Dùng Agent
        plantAndFinish();         // Agent trồng mía + dựng tường kính
    });
    ```
  - **Mở rộng quy mô:** Sử dụng biến `length` để điều khiển chiều dài. Lặp lại hàm module hoặc tăng giá trị biến để nhân rộng trang trại theo chiều ngang/dọc mà không cần viết lại logic.
- **Ghi chú quan trọng:** 
  - **Định hướng khối:** Hướng của Piston/Observer phụ thuộc hoàn toàn vào hướng nhìn (`facing`) của Agent tại thời điểm gọi `agent.place()`. Cần "biên đạo" chuỗi `move` + `turn` chính xác trước khi đặt.
  - **Chuẩn bị kho đồ:** Code phụ thuộc tuyệt đối vào việc vật phẩm nằm đúng `slot`. Luôn gọi `stockAgent()` hoặc dùng lệnh `/replaceitem` trước khi chạy.
  - **Gỡ lỗi (RCA):** 
    - *Piston không kích hoạt:* Kiểm tra hướng mặt "nhìn" của Observer và đường dẫn Redstone có bị đứt/không nối khối rắn không.
    - *Agent báo thiếu khối:* Kiểm tra lại `slot` và số lượng vật phẩm trong kho.
    - *Mía rơi sót:* Đảm bảo kênh nước có tường kính chắn hai bên để 100% item trôi về Hopper.
  - **Tối ưu hiệu suất:** `agent.setAssist(PLACE_ON_MOVE, true)` giúp Agent vừa di chuyển vừa đặt khối, giảm đáng kể số dòng code và thời gian thực thi so với vòng lặp `place` + `move` rời rạc.
