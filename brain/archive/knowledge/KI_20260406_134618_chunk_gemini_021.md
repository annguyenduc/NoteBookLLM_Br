# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Ứng dụng Toán học & Data Science trong Hệ thống Nhúng (Yolo Uno/ESP32-S3), Kiến trúc Edge-Server, và Quy trình Phát triển Dự án STEM/Hackathon.
- **Vấn đề / Câu hỏi:** Làm thế nào để tích hợp giải tích (đạo hàm/tích phân) vào xử lý dữ liệu cảm biến, phân biệt quy trình Khoa học/Kỹ thuật, và triển khai nhanh hệ thống IoT/AI cho cuộc thi?
- **Cách giải quyết / Logic lõi:** 
  - **Toán học trong Data Science:** 
    - *Đạo hàm ($dH/dt$):* Dùng trong Gradient Descent (tối ưu Loss Function) và tính tốc độ biến thiên để dự báo thời gian đạt ngưỡng nguy hiểm: $T_{warning} = \frac{H_{critical} - H_{current}}{dH/dt}$.
    - *Tích phân:* Nền tảng tính xác suất (PDF) và cộng dồn tín hiệu gia tốc/vận tốc để suy ra vị trí (Time-series).
  - **Kiến trúc hệ thống Edge-AI:** 
    `Cảm biến (Yolo Uno/ESP32-S3) -> Gửi JSON qua Serial -> Python (PySerial/Pandas) làm sạch & lưu CSV -> Streamlit/Dash hiển thị Dashboard & Cảnh báo.`
  - **Xử lý tín hiệu & TinyML:** Ưu tiên camera thường (OV2640) để xử lý ma trận pixel (Tensor) thay vì camera AI đóng gói sẵn. Áp dụng lọc nhiễu trước khi phân tích.
  - **Code Snippets (Core Logic):**
    ```python
    # 1. Lọc nhiễu cảm biến (Moving Average)
    def moving_average(data, window=5):
        return sum(data[-window:]) / window

    # 2. Dự báo thời gian cảnh báo (Ứng dụng đạo hàm)
    def calc_warning_time(h_crit, h_curr, dh_dt):
        return (h_crit - h_curr) / dh_dt if dh_dt != 0 else float('inf')

    # 3. Data Pipeline cơ bản (Python)
    import pandas as pd, serial, json, time
    ser = serial.Serial('COM3', 115200)
    df = pd.DataFrame(columns=['timestamp', 'value'])
    
    while True:
        if ser.in_waiting:
            raw = json.loads(ser.readline().decode())
            df = pd.concat([df, pd.DataFrame([{'timestamp': time.time(), 'value': raw['sensor']}])], ignore_index=True)
            df.to_csv('data.csv', index=False)
    ```
- **Ghi chú quan trọng:** 
  - **Phân biệt quy trình:** *Scientific Method* trả lời "Tại sao?" (kiểm chứng giả thuyết, kiểm soát biến số). *EDP* trả lời "Làm thế nào?" (lặp lại thiết kế, ràng buộc chi phí/kỹ thuật). STEM là nội dung liên môn, PBL là phương pháp tổ chức lớp học.
  - **Chiến lược Hackathon:** Dùng AI Agent sinh code OpenSCAD (thiết kế tham số), MicroPython (Edge), Python/Pandas/Streamlit (Server). Luôn yêu cầu AI viết Unit Test và lọc nhiễu (Kalman/Moving Average) trước khi đưa vào mô hình.
  - **Hardware:** Yolo Uno (ESP32-S3) hỗ trợ Vector Instructions, chạy được TensorFlow Lite. Camera thường giúp học sinh tiếp cận quy trình Data Science thực thụ (Preprocessing, Normalization, Training, Deploy) thay vì chỉ tích hợp hệ thống.
