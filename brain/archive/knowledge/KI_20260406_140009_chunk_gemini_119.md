# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Tích hợp và sử dụng GitHub Code Search API (REST) để tự động hóa tìm kiếm mã nguồn.
- **Vấn đề / Câu hỏi:** Cách truy vấn mã nguồn GitHub qua API trực tiếp, xử lý xác thực an toàn, xây dựng cú pháp tìm kiếm chính xác, phân trang kết quả và quản lý giới hạn tốc độ/lỗi.
- **Cách giải quyết / Logic lõi:** 
  - **Endpoint & Method:** `GET https://api.github.com/search/code`
  - **Xác thực:** Dùng Fine-grained PAT với quyền `contents:read`. Header bắt buộc: `Authorization: Bearer <TOKEN>`, `User-Agent: <App_Name>`, `Accept: application/vnd.github.v3.text-match+json`
  - **Cú pháp truy vấn (`q`):** Bắt buộc có từ khóa + bộ lọc phạm vi (`user:`, `org:`, hoặc `repo:`). Kết hợp qualifier: `in:file`, `language:`, `path:`, `extension:`, `size:`, `filename:`.
  - **Code mẫu lõi (Python `requests`):**
    ```python
    import requests
    url = "https://api.github.com/search/code"
    headers = {
        "Authorization": "Bearer YOUR_FINE_GRAINED_PAT",
        "User-Agent": "MyCodeSearchApp",
        "Accept": "application/vnd.github.v3.text-match+json"
    }
    params = {"q": "addClass in:file language:js repo:jquery/jquery"}
    res = requests.get(url, headers=headers, params=params)
    data = res.json() # Xử lý data['items'] và data['total_count']
    ```
  - **Phân trang:** Giới hạn cứng 1000 kết quả/tìm kiếm. Dùng `per_page` (tối đa 100) + `page`, hoặc lặp URL từ header `Link` (`rel="next"`).
  - **Rate Limit & Error Handling:** Giám sát header `x-ratelimit-remaining`. Xử lý `429` bằng cơ chế back-off. Lưu ý `404` có thể được trả về thay `403` cho tài nguyên private (cơ chế bảo mật ẩn danh).
- **Ghi chú quan trọng:** 
  - API chỉ quét nhánh mặc định (`main`/`master`) và tự động bỏ qua tệp > 384KB.
  - MCP (Model Context Protocol) là giao thức kết nối LLM với công cụ bên ngoài, **không** thay thế được GitHub Code Search API.
  - Luôn kiểm tra trường `incomplete_results: true` trong JSON để biết kết quả có bị cắt bớt do timeout không.
  - Ưu tiên Fine-grained PAT thay vì Classic PAT để tuân thủ nguyên tắc đặc quyền tối thiểu (Least Privilege) và tăng bảo mật.
