# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Lập trình Minecraft Education Edition (Python), Thuật toán dựng hình khối, Kịch bản tương tác Tynker, và Nền tảng CNTT/Phần cứng máy tính.
- **Vấn đề / Câu hỏi:** 
  1. Lỗi `unknown object type` khi truy cập thuộc tính `.x/.y/.z` và không tương thích thư viện `mcpi` trên Minecraft Education Edition.
  2. Logic tính toán tọa độ để xây kim tự tháp đáy vuông.
  3. Cấu trúc khối lệnh tạo video quảng cáo tương tác trên Tynker.
  4. Kiến thức cốt lõi về phần cứng, hệ nhị phân, mã hóa và khái niệm trừu tượng hóa trong CNTT.
- **Cách giải quyết / Logic lõi:** 
  - **Sửa lỗi API Minecraft:** Thay thế thư viện `mcpi` bằng API chuẩn của Code Builder. Dùng hàm trợ giúp thay vì truy cập thuộc tính trực tiếp:
    ```python
    # Thay vì: pos.x, pos.y, pos.z
    x = positions.x(pos)
    y = positions.y(pos)
    z = positions.z(pos)
    ```
  - **Thuật toán Kim tự tháp (Đáy `base_size` lẻ):**
    ```python
    num_floors = (base_size + 1) // 2
    for floor in range(num_floors):
        current_size = base_size - (floor * 2)  # Giảm 2 khối mỗi tầng
        current_y = y + floor                   # Tăng độ cao
        start_x = x + floor                     # Dịch vào trong 1 khối trục X
        start_z = z + floor                     # Dịch vào trong 1 khối trục Z
        # Vòng lặp lồng nhau đặt khối tại (start_x + i, current_y, start_z + j)
    ```
  - **Tynker Scripting:** Sử dụng mô hình sự kiện (`when stage starts`, `when I receive [tên_tin]`), chuyển cảnh (`switch background`), điều khiển sprite (`glide`, `go to x: y:`), và đồng bộ âm thanh/hình ảnh để tạo luồng tương tác.
  - **Kiến thức CNTT & Phần cứng:**
    - **RAM:** Bộ nhớ ngắn hạn (volatile). Loại phổ biến: `DRAM`, `DDR SDRAM`. (`DIMM` là chuẩn chân cắm vật lý).
    - **Mainboard:** Định tuyến nguồn, kết nối linh kiện, hỗ trợ khe mở rộng (PCIe).
    - **Lưu trữ:** `HDD` có đĩa quay/cánh tay cơ học (dễ hỏng vật lý). `SSD` nhanh hơn, bền hơn. Nâng cấp giúp tăng tốc boot/load và dung lượng.
    - **Hệ nhị phân & Mã hóa:** Bit chỉ nhận `0` hoặc `1`. Transistor vật lý triển khai **Logic Gates**. Character Encoding ánh xạ ký tự sang nhị phân. Compilers dịch ngôn ngữ bậc cao sang mã máy.
    - **Abstraction (Trừu tượng hóa):** Ẩn chi tiết phức tạp, chia nhỏ hệ thống thành các khái niệm đơn giản để dễ quản lý.
- **Ghi chú quan trọng:** 
  - Minecraft Education Edition dùng môi trường `Code Builder` tích hợp sẵn, **không hỗ trợ** `mcpi` (chỉ dùng cho Pi Edition/Java Server có plugin).
  - Lỗi hình chiếu mờ/pixelated thường do **mismatch độ phân giải** hoặc thiếu driver hiển thị, không phải lỗi phần cứng máy chiếu.
  - Đơn vị tốc độ truyền dữ liệu: `MB/s` (Megabyte/giây - lưu trữ), `Mb/s` (Megabit/giây - mạng).
