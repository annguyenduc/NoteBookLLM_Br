# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Lập trình Agent Minecraft Education (MakeCode) xây dựng cấu trúc hình lục giác thẳng đứng có viền và tâm đặc.
- **Vấn đề / Câu hỏi:** Làm sao xây chính xác hình lục giác "tâm mặt trời" (viền rỗng, tâm đặc) mà không bị lệch khối, sai hình học hoặc lỗi kiểu dữ liệu hướng di chuyển (`SixDirection` vs `CompassDirection`)?
- **Cách giải quyết / Logic lõi:** 
  - **Thuật toán chồng lớp (Overlay):** Thay vì tính toán phức tạp để vẽ viền rỗng rồi lấp tâm, dùng phương pháp xây 2 hình lục giác đặc chồng lên nhau: Hình lớn (màu viền) + Hình nhỏ hơn 1 cạnh (màu tâm) đặt chính giữa. Kết quả tự động tạo viền dày 1 khối bao quanh tâm.
  - **Hàm xây cột đối xứng:** Xây từ đường giữa (`midline`) lên trên và xuống dưới để đảm bảo hình luôn cân đối.
  - **Căn giữa hình học:** Chiều rộng tổng của lục giác cạnh `n` là `3*n - 2`. Dịch Agent sang trái `Math.floor(totalWidth / 2)` từ tâm để căn chỉnh.
  - **Code lõi:**
    ```typescript
    // 1. Xây cột đối xứng thẳng đứng
    function buildVerticalColumn(height: number, blockType: number) {
        agent.setSlot(1); agent.setItem(blockType, 64, 1);
        const midline = agent.getPosition();
        for (let i = 0; i < Math.ceil(height / 2); i++) { agent.place(FORWARD); agent.move(UP, 1); }
        agent.teleport(midline, WEST);
        for (let i = 0; i < Math.floor(height / 2); i++) { agent.move(DOWN, 1); agent.place(FORWARD); }
        agent.teleport(midline, WEST);
    }

    // 2. Duyệt cột tạo hình lục giác đặc
    function buildVerticalHexagon(canh: number, blockType: number) {
        let currentHeight = canh;
        for (let i = 0; i < canh - 1; i++) { buildVerticalColumn(currentHeight, blockType); agent.move(RIGHT, 1); currentHeight += 2; }
        for (let i = 0; i < canh; i++) { buildVerticalColumn(currentHeight, blockType); agent.move(RIGHT, 1); }
        for (let i = 0; i < canh - 1; i++) { currentHeight -= 2; buildVerticalColumn(currentHeight, blockType); agent.move(RIGHT, 1); }
    }

    // 3. Lệnh thực thi & Căn giữa
    player.onChat("tam_mat_troi", function (canh: number) {
        if (!(canh > 2 && canh % 2 == 0)) { player.say("Lỗi: Cạnh phải là số chẵn > 2."); return; }
        const buildCenter = agent.getPosition(); // Hoặc startPos đã lưu
        
        // Bước 1: Viền lớn
        let totalWidthVien = 3 * canh - 2;
        agent.teleport(buildCenter, WEST);
        agent.move(LEFT, Math.floor(totalWidthVien / 2));
        buildVerticalHexagon(canh, RIA_COLOR);

        // Bước 2: Tâm nhỏ (chồng lên, nhô ra 1 khối về phía trước)
        let canhTam = canh - 1;
        let totalWidthTam = 3 * canhTam - 2;
        agent.teleport(buildCenter, WEST);
        agent.move(LEFT, Math.floor(totalWidthTam / 2));
        agent.move(FORWARD, 1); 
        buildVerticalHexagon(canhTam, TAM_COLOR);
    });
    ```
- **Ghi chú quan trọng:** 
  - **Ràng buộc đầu vào:** Cạnh phải là số chẵn `> 2` để công thức `Math.ceil/floor` và logic tăng/giảm chiều cao `±2` hoạt động đối xứng tuyệt đối.
  - **Sửa lỗi Type Mismatch:** Lỗi `Argument of type 'SixDirection' is not assignable to parameter of type 'CompassDirection'` xảy ra khi khai báo sai kiểu cho hàm wrapper. Giải pháp: Gọi trực tiếp `agent.move()` với các hằng số `UP, DOWN, LEFT, RIGHT` (thuộc `SixDirection`) hoặc kiểm tra kỹ signature hàm trong MakeCode.
  - **Tối ưu:** Phương pháp "chồng hình đặc" giảm thiểu 90% lỗi tính toán tọa độ viền rỗng, dễ bảo trì và mở rộng cho các cấu trúc đa lớp.
