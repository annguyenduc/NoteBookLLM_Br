# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Khắc phục lỗi kích hoạt môi trường ảo Python trên PowerShell & Phương pháp tải nội dung khóa học Coursera (Chính thức & Công cụ CLI/GUI).
- **Vấn đề / Câu hỏi:** 
  1. Lỗi `CommandNotFoundException` khi chạy lệnh `coursera_env\Scripts\activate` trong PowerShell.
  2. Nhu cầu tải hàng loạt video, tài liệu, quiz từ Coursera về máy để học offline, kèm theo các công cụ hỗ trợ và ràng buộc pháp lý.
- **Cách giải quyết / Logic lõi:** 
  **1. Sửa lỗi kích hoạt venv trên PowerShell:**
  - Nguyên nhân: Sai thư mục làm việc, đường dẫn tương đối không khớp, hoặc Execution Policy chặn script.
  - Lệnh kích hoạt chuẩn theo môi trường:
    ```powershell
    # PowerShell
    ./<tên_env>/Scripts/activate
    # CMD
    .\<tên_env>\Scripts\activate.bat
    # Bash/Git Bash
    source <tên_env>/bin/activate
    ```
  - Khắc phục: `cd` đúng thư mục chứa env hoặc dùng đường dẫn tuyệt đối. Nếu bị chặn policy: `Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass`
  
  **2. Tải khóa học Coursera:**
  - *Chính thức:* App di động hỗ trợ tải hàng loạt (lưu trong sandbox). Web chỉ cho tải từng file riêng lẻ (video, `.vtt`, `.txt`).
  - *Công cụ CLI (Khuyến nghị):* `coursera-helper` (fork hiện đại của `coursera-dl`).
    ```bash
    pip install --upgrade coursera-helper
    # Xác thực bằng Cookie CAUTH (lấy từ DevTools > Application/Storage > Cookies > https://www.coursera.org/)
    coursera-helper --cauth <GIÁ_TRỊ_CAUTH> <TÊN_SLUG_KHÓA_HỌC>
    # Tùy chọn nâng cao
    coursera-helper --cauth <CAUTH> <SLUG> --subtitle-language en,vi --video-resolution 720p --download-quizzes
    ```
  - *Tiện ích mở rộng:* `CSA - Asst. for saving files from Coursera` (Chrome) - Giao diện GUI, tự động đặt tên file logic, hỗ trợ tải phụ đề đa ngôn ngữ.
- **Ghi chú quan trọng:** 
  - Tuân thủ ToS Coursera: Chỉ tải cho mục đích **cá nhân, phi thương mại**. Nghiêm cấm phân phối lại.
  - Cookie `CAUTH` có hạn ~50 ngày, cần trích xuất lại khi hết hạn.
  - `coursera-dl` gốc đã lỗi thời do Coursera thay đổi API. Ưu tiên `coursera-helper` hoặc `dl_coursera` (dùng `cookies.txt`).
  - Các chứng chỉ Google trên Coursera được cấu trúc thành chuỗi khóa học thành phần (ví dụ: Data Analytics 8 khóa, Cybersecurity 8 khóa, BI 3 khóa...). Link chính thức có thể tra cứu trực tiếp trên trang Coursera.
