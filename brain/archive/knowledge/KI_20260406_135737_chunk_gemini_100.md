# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Hệ sinh thái công cụ tải khóa học Coursera ngoại tuyến & Hướng dẫn kỹ thuật thiết lập (2024-2025)
- **Vấn đề / Câu hỏi:** Tính năng tải chính thức của Coursera bị giới hạn (thao tác thủ công, chất lượng thấp, mất cấu trúc thư mục). Các công cụ bên thứ ba thường xuyên lỗi do Coursera liên tục cập nhật API (`onDemandCourseMaterials.v1` → `v2`) và cấu trúc Front-end. Người dùng cần giải pháp ổn định, quy trình xác thực an toàn và cảnh báo rủi ro bảo mật/pháp lý.
- **Cách giải quyết / Logic lõi:** 
  1. **Phân loại & Lựa chọn công cụ:**
     - `Coursera-Downloader` (GUI): Ưu tiên cho người mới, Windows, tự động giữ cấu trúc tuần/chủ đề, hỗ trợ resume.
     - `coursera-helper` (CLI): Fork ổn định nhất, đa nền tảng, hỗ trợ Python 3.8-3.11, tải được video, quiz, Jupyter Notebook.
     - *Cần tránh:* `coursera-dl` gốc (lỗi `HTMLParser.unescape` trên Python 3.9+, lỗi API 404/400), `yt-dlp` (yêu cầu trích xuất thủ công link `.m3u8`, không phù hợp người mới).
  2. **Thiết lập môi trường Python & Pip (Windows):**
     ```bash
     # Kiểm tra phiên bản sau khi cài đặt (nhớ tick "Add Python to PATH")
     python --version
     pip --version
     # Cài đặt công cụ CLI
     pip install coursera-helper
     ```
  3. **Xác thực & Thực thi tải (Logic CAUTH Cookie):**
     - Trích xuất cookie `CAUTH` từ DevTools: Chrome (`Application` > `Cookies` > `https://www.coursera.org/`) hoặc Firefox (`Storage` > `Cookies`).
     - Thực thi lệnh tải:
     ```bash
     # Cú pháp cơ bản
     coursera-helper --cauth "dán_chuỗi_cookie_dài_vào_đây" ten-khoa-hoc
     # Tùy chỉnh độ phân giải
     coursera-helper --cauth "cookie" ten-khoa-hoc --video-resolution 720p
     ```
  4. **Đồng bộ phụ đề (.srt):** Đặt tên file `.srt` trùng khớp hoàn toàn với file `.mp4` (chỉ khác phần mở rộng) trong cùng thư mục để VLC Media Player tự động nhận diện.
- **Ghi chú quan trọng:** 
  - **Bảo mật:** Cookie `CAUTH` là chìa khóa phiên đăng nhập. Chỉ cung cấp cho tool mã nguồn mở uy tín (GitHub). Tuyệt đối không dùng cho dịch vụ web lạ để tránh Session Hijacking.
  - **Pháp lý & ToS:** Coursera cho phép tải cho mục đích cá nhân nhưng nghiêm cấm hành vi "vượt qua biện pháp xác thực/bảo mật" hoặc "lạm dụng dịch vụ". Sử dụng tool tự động tiềm ẩn nguy cơ bị khóa tài khoản.
  - **Tính ổn định:** Công cụ tải Coursera cực kỳ nhạy cảm với bản cập nhật nền tảng. Khi gặp lỗi 404/400, nguyên nhân thường do API thay đổi. Cần theo dõi repo GitHub để cập nhật phiên bản fork mới nhất hoặc lấy lại cookie `CAUTH` mới.
