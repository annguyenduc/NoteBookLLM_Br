# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Lập trình Agent Minecraft (MakeCode/TypeScript) để xây dựng hình lục giác đứng đối xứng và hình mặt trời có tia nắng căn chỉnh chính xác.
- **Vấn đề / Câu hỏi:** 
  1. Chuyển đổi logic xây hình từ mặt phẳng ngang (X-Z) sang mặt phẳng đứng (Y-Z).
  2. Khắc phục lỗi lệch khối ở nửa bên phải và tạo hình lục giác đối xứng hoàn toàn qua tâm (không bị "cụt" đáy).
  3. Gắn các tia nắng vào lõi lục giác mà không bị hở/lệch do sai số tính toán tọa độ thủ công.
- **Cách giải quyết / Logic lõi:** 
  - **Xây cột đối xứng qua trục ngang:** Thay vì xây từ dưới lên, dùng hàm `buildMirroredColumn` chia chiều cao thành nửa trên và nửa dưới, xây từ đường trung tâm (`midline`) ra hai phía.
  - **Kỹ thuật "Điểm Neo" (Anchor) để căn chỉnh:** Thay vì tính toán tọa độ tia nắng bằng công thức toán học (dễ sai lệch trong môi trường khối rời rạc), lưu trực tiếp `agent.getPosition()` vào mảng `anchors` khi Agent đi qua các góc/cạnh trong quá trình xây lõi. Sau đó dịch chuyển Agent đến từng điểm neo để vẽ tia.
  - **Quy tắc tỷ lệ khối:** Chiều cao cột phải tăng/giảm `±2` mỗi bước dịch ngang `1` để duy trì góc nghiêng ~60° đặc trưng của lục giác trên lưới khối.
  - *Code lõi:*
    ```typescript
    // 1. Hàm xây cột đối xứng & lưu điểm neo
    function buildAndAnchorColumn(height: number, isCorner: boolean, isMiddle: boolean) {
        const midline = agent.getPosition();
        // Nửa trên
        for (let i = 0; i < Math.ceil(height / 2); i++) { agent.place(FORWARD); agent.move(UP, 1); }
        if (isMiddle) anchors.push(agent.getPosition()); // Lưu neo trên
        
        agent.teleport(midline, WEST); // Về đường giữa
        // Nửa dưới
        for (let i = 0; i < Math.floor(height / 2); i++) { agent.move(DOWN, 1); agent.place(FORWARD); }
        if (isMiddle) anchors.push(agent.getPosition()); // Lưu neo dưới
        if (isCorner) anchors.push(agent.getPosition()); // Lưu neo góc
        agent.teleport(midline, WEST);
    }

    // 2. Hàm vẽ tia từ điểm neo
    function buildRay(length: number, x_step: number, y_step: number) {
        for (let i = 0; i < length; i++) {
            if (x_step > 0) agent.move(RIGHT, x_step);
            if (x_step < 0) agent.move(LEFT, -x_step);
            if (y_step > 0) agent.move(UP, y_step);
            if (y_step < 0) agent.move(DOWN, -y_step);
            agent.place(BACK, 1); // Đặt khối lùi lại để tạo vệt liền mạch
        }
    }
    ```
- **Ghi chú quan trọng:**
  - **Sửa lỗi lệch mặt phẳng:** Tuyệt đối không dùng `agent.move(DOWN, 1)` trước khi xây các cột thấp dần ở nửa bên phải. Điều này khiến toàn bộ nửa phải bị tụt xuống 1 khối, phá vỡ cấu trúc.
  - **Ràng buộc đầu vào:** Tham số `cạnh` (số khối) nên là **số chẵn > 2**. Điều này đảm bảo việc xác định điểm giữa cạnh (để lưu neo) và tính đối xứng qua tâm được chính xác tuyệt đối, tránh lỗi làm tròn `Math.floor/ceil`.
  - **Quản lý trạng thái Agent:** Luôn dùng `agent.teleportToPlayer()` hoặc lưu `buildOrigin` để reset vị trí trước mỗi lần chạy lệnh. Xóa mảng `anchors = []` ở đầu hàm để tránh tích lũy tọa độ từ lần chạy trước.
  - **Hướng đặt khối:** Dùng `agent.place(BACK, 1)` kết hợp di chuyển để tạo vệt tia nắng liền mạch mà không cần tính toán vị trí đặt khối phức tạp.
