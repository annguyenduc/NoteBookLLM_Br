# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Lập trình JavaScript cho Agent (Minecraft Education) & Dinh dưỡng trẻ 3 tuổi.
- **Vấn đề / Câu hỏi:** 
  1. Lỗi ép kiểu `Argument of type 'number' is not assignable to parameter of type 'string'` khi xử lý tham số lệnh chat.
  2. Thuật toán điều khiển Agent xây hình lục giác (chuvi & khối đặc) và lăng trụ lục giác.
  3. Lịch uống sữa & nguyên tắc bữa tối chuẩn cho trẻ 3 tuổi.
- **Cách giải quyết / Logic lõi:** 
  **1. Khắc phục lỗi kiểu dữ liệu:**
  - `player.onChat` trong Minecraft Education tự động truyền tham số dưới dạng số/chuỗi tương thích. **Bỏ `parseInt()`** để tránh lỗi strict type.
  - Khi hiển thị số ra màn hình, dùng toán tử `+` để tự động ép kiểu sang string: `player.say("Chiều cao: " + chieuCao);`

  **2. Thuật toán Lăng trụ Lục giác (Chu vi - Blocky):**
  - Logic: 3 vòng lặp lồng nhau `(Chiều cao → 6 cạnh → Số block/cạnh)`. Agent chỉ quay được 90° nên hình sẽ dạng bậc thang.
  ```javascript
  for (let h = 0; h < chieuCao; h++) {
      for (let side = 0; side < 6; side++) {
          for (let i = 0; i < canh; i++) {
              agent.place(DOWN);
              if (i < canh - 1) agent.move(FORWARD, 1); // Tránh di chuyển thừa ở góc
          }
          agent.turn(RIGHT); // Rẽ 90° sang cạnh tiếp theo
      }
      if (h < chieuCao - 1) agent.move(UP, 1); // Lên tầng (bỏ qua tầng cuối)
  }
  ```

  **3. Thuật toán Lục giác đặc (Theo hàng - Row-based):**
  - Logic: Chia hình thành 2 nửa dựa trên `radius` (bán kính). Nửa trên mở rộng hàng, nửa dưới thu hẹp hàng.
  - Công thức độ dài hàng: `rowLength = radius + i` (trên), `rowLength = (radius * 2 - 2) - i` (dưới).
  - Di chuyển Agent giữa các hàng: `BACK` (về đầu) → `RIGHT` (sang hàng mới) → `BACK`/`FORWARD` (thụt lề/nhô ra 1 block tạo độ nghiêng).
  ```javascript
  // Nửa trên (mở rộng)
  for (let i = 0; i < radius; i++) {
      let rowLength = radius + i;
      for (let j = 0; j < rowLength; j++) {
          agent.place(DOWN);
          if (j < rowLength - 1) agent.move(FORWARD, 1);
      }
      if (i < radius - 1) {
          agent.move(BACK, rowLength - 1);
          agent.move(RIGHT, 1);
          agent.move(BACK, 1); // Thụt lề tạo góc
      }
  }
  // Nửa dưới (thu hẹp) thực hiện tương tự nhưng đảo hướng di chuyển & công thức rowLength
  ```

  **4. Dinh dưỡng trẻ 3 tuổi:**
  - Lượng sữa: **400–600ml/ngày**, chia 3 cữ (Sáng: 150-200ml, Chiều: 150-200ml, Tối trước ngủ: 100-150ml).
  - Bữa tối: Đủ 4 nhóm chất, chế biến mềm/nhạt, đa dạng thực phẩm, trình bày bắt mắt.
- **Ghi chú quan trọng:** 
  - Luôn dùng điều kiện `if (index < max - 1)` trước lệnh `move()` để tránh Agent di chuyển thừa 1 bước ở cuối vòng lặp, giúp hình khối kín và chính xác.
  - Sau mỗi tầng/hàng, **bắt buộc reset vị trí Agent** về điểm xuất phát chuẩn trước khi sang bước lặp tiếp theo.
  - Không cho trẻ uống sữa ngay trước/trong bữa chính để tránh no giả, gây biếng ăn. Ưu tiên hấp/luộc/hầm để giữ dinh dưỡng & độ mềm.
