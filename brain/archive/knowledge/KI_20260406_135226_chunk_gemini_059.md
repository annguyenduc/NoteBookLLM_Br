# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Lập trình mạch OhStem YOLO UNO (ESP32-S3) bằng MicroPython và C++/Arduino trên VS Code.
- **Vấn đề / Câu hỏi:** Thiết lập môi trường phát triển, nạp firmware, cấu hình bo mạch tùy chỉnh trên PlatformIO, và lập trình điều khiển ngoại vi/IoT/TinyML.
- **Cách giải quyết / Logic lõi:** 
  1. **Nạp Firmware MicroPython:** Dùng `esptool.py`. Vào chế độ Bootloader (giữ `BOOT0` + nhấn `RESET`), xóa flash, nạp firmware generic ESP32-S3 (hỗ trợ SPIRAM) vào địa chỉ `0x0`.
     ```bash
     esptool.py --chip esp32s3 --port PORTNAME erase_flash
     esptool.py --chip esp32s3 --port PORTNAME --baud 460800 write_flash -z 0x0 FIRMWARE_NAME.bin
     ```
  2. **Cấu hình PlatformIO (C++):** Tạo file `boards/yolo_uno.json` định nghĩa thông số phần cứng. Cấu hình `platformio.ini` để trỏ đến file này:
     ```ini
     [env:yolo_uno]
     platform = espressif32
     board = yolo_uno
     framework = arduino
     boards_dir = boards
     monitor_speed = 115200
     ```
  3. **Code mẫu cốt lõi (Bỏ boilerplate):**
     - *LED Blink (GPIO 13):* 
       `MicroPython:` `led = Pin(13, Pin.OUT); led.on()/off()`
       `C++:` `pinMode(LED_BUILTIN, OUTPUT); digitalWrite(LED_BUILTIN, HIGH/LOW)`
     - *NeoPixel RGB (GPIO 48):* 
       `MicroPython:` `np = neopixel.NeoPixel(Pin(48), 1); np[0]=(R,G,B); np.write()`
       `C++:` `pixels.setPixelColor(0, pixels.Color(R,G,B)); pixels.show()`
     - *ADC (GPIO 1):* 
       `MicroPython:` `ADC(Pin(1)).atten(ADC.ATTN_11DB).read()`
       `C++:` `analogRead(1)`
     - *Wi-Fi Scan:* 
       `MicroPython:` `network.WLAN(network.STA_IF).active(True).scan()`
       `C++:` `WiFi.mode(WIFI_STA); WiFi.scanNetworks()`
- **Ghi chú quan trọng:** 
  - YOLO UNO dùng ESP32-S3 (240MHz, 4MB Flash, 8MB PSRAM), tích hợp Wi-Fi/BT và 12 cổng Grove.
  - PlatformIO yêu cầu khai báo tường minh `#include <Arduino.h>` và quản lý dependency qua `lib_deps` trong `platformio.ini`.
  - **AI/TinyML:** MCU không chạy được YOLOv8. Nên dùng Edge Impulse để huấn luyện mô hình FOMO (tối ưu cho vi điều khiển) hoặc xuất `.tflite` kết hợp `tflite-micro` trong MicroPython.
  - **Mẹo Windows 10:** Đổi hình nền khi chưa active bằng cách chuột phải vào file ảnh → chọn `Set as desktop background`.
