# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Khắc phục lỗi treo UI Screen, xử lý mất IP và quy trình cập nhật Firmware cho máy in 3D Elegoo Neptune 4.
- **Vấn đề / Câu hỏi:** Màn hình Neptune 4 bị đơ cứng tại giao diện "About Setting", không phản hồi cảm ứng, không tìm thấy IP mạng. Lỗi này tạo vòng lặp "con gà - quả trứng": Mainboard cần xác nhận "Confirm" trên UI để update, nhưng UI đang đơ nên không thể thao tác.
- **Cách giải quyết / Logic lõi:** 
  1. **Nguyên tắc cập nhật bắt buộc:** Nạp Firmware Màn hình TRƯỚC (chạy tự động qua Bootloader phần cứng, không cần UI), sau đó mới nạp Mainboard.
  2. **Định dạng thiết bị lưu trữ (FAT32 + 4096 bytes):** Bootloader Neptune 4 yêu cầu khắt khe `Allocation unit size: 4096 bytes`. Windows thường ẩn tùy chọn này trên ổ >32GB.
     - *Lệnh CMD nhanh:* `format F: /FS:FAT32 /A:4096 /Q`
     - *Xử lý USB lớn bằng DiskPart:*
       ```cmd
       diskpart
       list disk
       select disk X  (Thay X bằng số ổ USB)
       clean
       create partition primary size=4096
       active
       format fs=fat32 unit=4096 quick
       assign
       exit
       ```
  3. **Cấu trúc file & Thao tác vật lý:**
     - *Màn hình:* Copy nguyên thư mục `ZNP_ROBIN_NANO` vào gốc thẻ MicroSD. Tháo 4 ốc lưng màn hình, cắm thẻ vào khe trên mạch PCB. Bật nguồn → máy tự update → báo `"Update Success"`.
     - *Mainboard:* Các file update phải nằm trong thư mục `ELEGOO_UPDATE_DIR` ở gốc USB. Cắm USB vào máy, trên UI bấm `"Confirm"`.
  4. **Xử lý mất IP/Không kết nối mạng:**
     - Truy cập qua hostname mặc định: `http://mainsail.local`, `http://fluidd.local`, `http://elegoo.local`
     - Quét mạng nội bộ: `arp -a` (CMD), dùng `Advanced IP Scanner` hoặc app `Fing`.
     - Kiểm tra phần cứng: Vệ sinh/cắm lại cáp màn hình, kiểm tra module EMMC trên mainboard (tháo lắp lại nếu bị lỏng do rung động).
- **Ghi chú quan trọng:** 
  - Lỗi đơ màn hình thường do mismatch phiên bản UI/Mainboard hoặc file hệ thống Klipper bị corrupt.
  - Sau khi update firmware, máy sẽ reset toàn bộ cấu hình, bắt buộc phải thực hiện lại Bed Leveling.
  - Nếu module EMMC hỏng vật lý hoặc file hệ thống không thể khôi phục qua USB/SD, cần liên hệ bảo hành hoặc thay thế module.
