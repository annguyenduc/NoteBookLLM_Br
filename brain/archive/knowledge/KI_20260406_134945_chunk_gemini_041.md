# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Tích hợp kết nối WiFi cho Elegoo Neptune 4 (Bản Standard) và thiết lập đồng bộ mạng với OrcaSlicer qua hệ sinh thái Klipper/Moonraker.
- **Vấn đề / Câu hỏi:** Neptune 4 bản tiêu chuẩn bị cắt giảm module WiFi, gây gián đoạn quy trình gửi G-code và giám sát từ xa. Người dùng cần giải pháp phần cứng tương thích, cấu hình Linux Armbian chính xác và khắc phục xung đột cổng mạng khi tích hợp OrcaSlicer.
- **Cách giải quyết / Logic lõi:** 
  1. **Lựa chọn phần cứng:** Ưu tiên USB WiFi Dongle chipset `RTL8188EUS` (VD: TP-Link TL-WN725N V1/V2) hoặc dùng cầu nối Ethernet-to-WiFi (Repeater có cổng LAN) để bypass driver Linux, đảm bảo ổn định tuyệt đối.
  2. **Cấu hình WiFi qua SSH:** Truy cập `ssh mks@<IP>` (pass: `makerbase`). Chỉnh sửa file cấu hình mạng:
     ```ini
     # File: /etc/wpa_supplicant/wpa_supplicant-wlan0.conf
     ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
     update_config=1
     country=VN
     network={
         ssid="TÊN_WIFI"
         psk="MẬT_KHẨU"
         key_mgmt=WPA-PSK
         priority=1
     }
     ```
  3. **Khắc phục lỗi mất kết nối sau reboot:** Xóa file cấu hình rác gây xung đột: `sudo rm /etc/wpa_supplicant/wpa_supplicant.conf` (chỉ giữ lại file `-wlan0.conf`), sau đó restart dịch vụ: `sudo systemctl restart networking`.
  4. **Đồng bộ OrcaSlicer:** 
     - Host Type: Chọn `Octo/Klipper`.
     - IP/Hostname: Nhập IP máy in. Nếu nút Upload lỗi, thêm cổng API: `192.168.1.x:7125`.
     - Device UI: Chỉ nhập `http://192.168.1.x` (cổng 80 cho Fluidd), tuyệt đối không thêm `:7125` để tránh crash giao diện.
- **Ghi chú quan trọng:** 
  - Gán IP tĩnh qua **DHCP Reservation** trên Router thay vì cấu hình trên máy in để tránh sai subnet mask gây mất kết nối.
  - Tránh cắm đồng thời Webcam và WiFi Dongle trên cùng bus USB để tránh nghẽn băng thông USB 2.0.
  - **Phương án không cần SSH:** Đặt file `wpa_supplicant-wlan0.conf` vào thư mục `ELEGOO_UPDATE_DIR` trên USB FAT32, cắm vào máy và reboot. Hệ thống sẽ tự động copy và cấp quyền.
  - **Chiến lược vận hành:** Tránh in gia công số lượng lớn (OPEX cao). Tối ưu doanh thu bằng cách bán tài sản số (Profile OrcaSlicer `.orca_printer`, giáo án STEM, mô hình 3D) và dịch vụ đào tạo/workshop, tận dụng Neptune 4 làm công cụ demo thay vì máy sản xuất.
