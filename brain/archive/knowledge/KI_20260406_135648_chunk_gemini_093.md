# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Tự động hóa xây dựng & trang trí khối Rubik 10x10x10 trong Minecraft Education Edition bằng MakeCode (Agent).
- **Vấn đề / Câu hỏi:** 
  - Agent bị kẹt khi tự xây khối đặc 10x10x10 do logic di chuyển sai.
  - Lỗi không tương thích API (`player.runCommandAsync` không tồn tại, `gameplay.runCommand` không ổn định trên một số phiên bản).
  - Sai lệch tọa độ & hướng khi Agent xây 6 mặt, gây xây chồng chéo hoặc lệch vị trí.
  - Player và Agent phụ thuộc vị trí lẫn nhau, gây gián đoạn khi người chơi di chuyển.
- **Cách giải quyết / Logic lõi:** 
  - **Tạo khung nhanh:** Thay vì Agent xếp từng block, dùng `blocks.fill()` hoặc `player.runCommand("fill ...")` để tạo khối rỗng 10x10x10 ngay lập tức, giải phóng không gian cho Agent di chuyển.
  - **Chuẩn hóa API:** Sử dụng `player.runCommand()` là lệnh ổn định và tương thích nhất trong MakeCode để thực thi lệnh game.
  - **Xây dựng từ bên ngoài:** Định vị Agent ở ngoài khối, dùng 3 hàm xử lý block tùy hướng mặt:
    ```typescript
    // Mặt bên: xây về phía trước
    function replace_2x2_in_front() { /* destroy/place FORWARD, move UP/RIGHT/LEFT */ }
    // Mặt trên: xây xuống dưới
    function replace_2x2_below() { /* destroy/place DOWN, move FORWARD/RIGHT/LEFT */ }
    // Mặt dưới: xây lên trên
    function replace_2x2_above() { /* destroy/place UP, move FORWARD/RIGHT/LEFT */ }
    ```
  - **Điều hướng & Vòng lặp:** Teleport Agent đến góc ngoài từng mặt, set hướng (`NORTH/SOUTH/EAST/WEST/UP/DOWN`). Dùng vòng lặp 3x3 để đặt 9 ô màu 2x2. Tránh xung đột biến vòng lặp lồng nhau (đổi `index` thành `i`, `j`).
  - **Tách biệt tọa độ:** Lưu độc lập `playerStartPos` và `agentStartPos`. Dùng `agent.teleport(agentStartPos, DIRECTION)` để reset Agent trước mỗi lần xây, cho phép Player tự do di chuyển.
- **Ghi chú quan trọng:** 
  - Luôn kiểm tra điều kiện `if (!startPos) { player.say("..."); return; }` để tránh lỗi `null reference` khi chạy lệnh.
  - Quy trình vận hành tối ưu: `start` (khởi tạo tọa độ) → `build_hollow_cube` (tạo khung rỗng) → `rubik` (Agent tự động trang trí 6 mặt từ ngoài vào).
  - Agent cần được trang bị đủ block màu vào slot 1 (`agent.setItem(COLOR, 4, 1)`) trước khi chạy hàm xây.
