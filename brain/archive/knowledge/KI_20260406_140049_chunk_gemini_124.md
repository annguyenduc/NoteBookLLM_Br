# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Quản lý môi trường Python, xử lý lỗi PATH/CMD trên Windows, công cụ `uv` và dọn dẹp thư viện toàn cục.
- **Vấn đề / Câu hỏi:** 
  - Lỗi `'uv' is not recognized` khi thực thi lệnh.
  - Cách quản lý, kiểm tra và dọn dẹp cache của `uv` và `pip`.
  - Lỗi không chuyển được ổ đĩa/thư mục trong Windows CMD (`cd D:\...`).
  - Cách xóa an toàn hàng loạt thư viện Python khỏi môi trường global mà không làm hỏng hệ thống.
- **Cách giải quyết / Logic lõi:** 
  - **Khắc phục lỗi `uv` & PATH:** 
    - Kiểm tra vị trí thực thi: `where uv`
    - Cài đặt khuyến nghị (tránh xung đột, tự động cập nhật PATH):
      ```bash
      pip install pipx
      pipx ensurepath
      pipx install uv
      ```
    - Nếu cài thủ công, thêm đường dẫn chứa `uv.exe` (thường là `%LOCALAPPDATA%\uv\bin\` hoặc `Scripts\` của Python) vào biến môi trường `PATH`.
  - **Quản lý Cache:**
    - `uv`: `uv cache dir` (xem đường dẫn), `uv cache prune` (xóa mục cũ), `uv cache clean` (xóa toàn bộ).
    - `pip`: `pip cache dir`, `pip cache info`, `pip cache remove <tên_gói>`, `pip cache purge`.
    - *Lưu ý:* `uv` sử dụng cache độc lập, không dùng chung với `pip`.
  - **Chuyển thư mục trong CMD:** 
    - CMD Windows yêu cầu chuyển ổ đĩa trước, sau đó mới đổi thư mục:
      ```cmd
      D:
      cd Python
      ```
    - Hoặc dùng cờ `/d` để chuyển đồng thời: `cd /d D:\Python`
  - **Logic Script xóa thư viện Global (Batch):**
    ```bat
    @echo off
    set "packages_to_uninstall=gói1 gói2 gói3 ..."
    for %%P in (%packages_to_uninstall%) do (
        pip uninstall -y %%P
    )
    pip list
    ```
- **Ghi chú quan trọng:** 
  - Luôn ưu tiên dùng môi trường ảo (`.venv`) cho từng dự án. Chỉ giữ các công cụ CLI (`pip`, `setuptools`, `wheel`, `black`, `pytest`, `uv`, `typer`...) ở môi trường global.
  - **Tuyệt đối không xóa** `pip`, `setuptools`, `wheel` khỏi global.
  - Sau khi chỉnh sửa `PATH` hoặc chạy `pipx ensurepath`, bắt buộc đóng và mở lại CMD/PowerShell để hệ thống nhận biến môi trường mới.
  - `uv` (phát triển bởi Astral) là công cụ thay thế tốc độ cao cho `pip`, `venv`, `pip-tools`. Các lệnh tương đương: `uv venv`, `uv pip install <pkg>`, `uv python list`.
