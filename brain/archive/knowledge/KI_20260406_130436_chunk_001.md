# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Xuất & xử lý dữ liệu hội thoại Gemini từ Google Takeout, Khắc phục lỗi định dạng file, Kiến trúc tích hợp AI (Claude/Antigravity) với Obsidian.
- **Vấn đề / Câu hỏi:** 
  1. Mục `Gemini` trong Takeout chỉ xuất cấu hình Gems, không có lịch sử chat.
  2. File `.txt` tải về báo lỗi `trouble reading the format` khi import vào công cụ bên thứ ba.
  3. Làm sao để AI tự động hóa việc quản lý, phân loại và xử lý tài liệu trong Obsidian Vault?
- **Cách giải quyết / Logic lõi:** 
  **1. Đường dẫn xuất dữ liệu chính xác:**
  - Không tích mục `Gemini`. Chọn `Hoạt động của tôi (My Activity)` → `Bỏ chọn tất cả` → Tích riêng `Gemini` hoặc `Gemini Apps`.
  - Dữ liệu chat nằm tại: `Takeout/My Activity/Gemini Apps/My Activity.html` hoặc `Takeout/Gemini in Workspace/Conversation History/`.
  
  **2. Xử lý lỗi định dạng & Trích xuất nội dung:**
  - File `.txt` của Takeout thực chất là JSON lồng nhau. Công cụ import thường yêu cầu `.csv`/`.json` thuần nên báo lỗi.
  - *Fix nhanh:* Đổi đuôi `.txt` → `.json` và mở bằng JSON Viewer.
  - *Fix tự động (Python):* Lọc trực tiếp trường `prompt` và `response` từ cấu trúc `conversation_turns`:
    ```python
    import json, os

    def extract_gemini_chat(file_path):
        with open(file_path, 'r', encoding='utf-8') as f:
            data = json.load(f)
        
        for turn in data.get("conversation_turns", []):
            user_prompt = turn.get("user_turn", {}).get("prompt", "")
            ai_response = turn.get("system_turn", {}).get("text", {}).get("data", "")
            if user_prompt and ai_response:
                print(f"Q: {user_prompt}\nA: {ai_response}\n---")
    ```

  **3. Kiến trúc "Second Brain" Obsidian + AI:**
  - **Claude Code:** Tương tác qua `Obsidian CLI` (bật trong `Settings > General > Command line interface`). AI dịch lệnh tự nhiên thành CLI để tạo/sửa file Markdown.
  - **Antigravity (Google/MCP):** Dùng `Model Context Protocol`. Cài `obsidian-mcp-server` hoặc plugin `Local REST API` để Antigravity đọc/ghi Vault trực tiếp.
  - **Cấu trúc Vault chuẩn:** `Inbox` (ý tưởng thô) → `Projects` (dự án có hạn) → `Areas` (trách nhiệm dài hạn) → `Resources` (tài liệu tham khảo) → `Archive`.
  - **PDF Pipeline:** Chunking (PyMuPDF) → LLM trích xuất ý chính → Xuất Markdown sạch → Đẩy vào Obsidian qua CLI/MCP.
- **Ghi chú quan trọng:** 
  - Dữ liệu Takeout chỉ tồn tại khi tính năng `Hoạt động trên ứng dụng Gemini` đang BẬT trong cài đặt bảo mật Google.
  - Antigravity tận dụng Context Window lớn (1M-2M tokens) của Gemini, cho phép đọc tài liệu dài mà không cần chia nhỏ (chunking) nhiều như Claude.
