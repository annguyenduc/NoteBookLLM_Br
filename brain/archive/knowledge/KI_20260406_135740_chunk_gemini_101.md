# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Phương pháp & công cụ tải xuống nội dung khóa học Coursera để học offline (Cập nhật 2024-2025).
- **Vấn đề / Câu hỏi:** Làm thế nào để tải hàng loạt video, phụ đề và tài liệu Coursera về máy tính một cách ổn định, dễ dùng cho người mới và tuân thủ chính sách nền tảng?
- **Cách giải quyết / Logic lõi:** 
  - **Phương pháp chính thức:** Ứng dụng di động Coursera cho phép tải hàng loạt theo module (nhưng file bị khóa trong sandbox, chỉ xem được trong app). Web chỉ hỗ trợ tải thủ công từng file.
  - **Công cụ dòng lệnh `cs-dlp` (Fork của `coursera-dl`):** Giải pháp mạnh mẽ nhất cho người dùng kỹ thuật, hỗ trợ tải toàn bộ khóa học, tùy chỉnh chất lượng, phụ đề và resume.
    ```bash
    # Cài đặt (Lưu ý: Repo gốc hiện đã bị ẩn/xóa)
    git clone https://github.com/raffaem/cs-dlp
    cd cs-dlp
    pip install .

    # Lệnh tải cơ bản
    cs-dlp --cookies /path/to/cookies.txt <course-slug> --path "D:\Coursera" --subtitle-language vi,en --video-resolution 720p --resume
    ```
  - **Cơ chế xác thực bằng Cookie:** Do API login của Coursera thay đổi, xác thực bằng mật khẩu thường lỗi `400/403`. Giải pháp ổn định là xuất cookie phiên đăng nhập (định dạng `Netscape`/`cookies.txt`) chứa token `CAUTH` qua extension trình duyệt (ví dụ: Cookie-Editor).
  - **Giải pháp thay thế GUI (Khuyến nghị hiện tại):** Tiện ích `CSA - Asst. for saving files from Coursera` (Chrome). Hỗ trợ đặt tên file thông minh, theo dõi trạng thái tải, không yêu cầu dòng lệnh và được bảo trì tích cực.
- **Ghi chú quan trọng:** 
  - ⚠️ **Trạng thái `cs-dlp`:** Repository `raffaem/cs-dlp` hiện không còn truy cập công khai (`fatal: repository not found`). Người dùng cần tìm fork thay thế hoặc chuyển sang dùng Extension `CSA`/App di động.
  - 🔐 **Xử lý lỗi thường gặp:** 
    - `AuthenticationFailed/403`: Cookie hết hạn → Đăng xuất/đăng nhập lại Coursera và xuất file `cookies.txt` mới.
    - `Course Not Found`: Sai slug hoặc chưa enroll/chấp nhận Honor Code trên web.
    - `'cs-dlp' is not recognized` (Windows): Thiếu đường dẫn `Python\Scripts` trong biến môi trường `PATH`.
  - ⚖️ **Tuân thủ ToS & Quản lý rủi ro:** Coursera cho phép tải xuống cho mục đích **cá nhân, phi thương mại**. Cấm tuyệt đối việc chia sẻ, bán lại hoặc dùng tool gian lận (auto-quiz). Rủi ro khóa tài khoản rất thấp nếu tải với tốc độ hợp lý, dùng cookie xác thực và không phân phối lại nội dung.
