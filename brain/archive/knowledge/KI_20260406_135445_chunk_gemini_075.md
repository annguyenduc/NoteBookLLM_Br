# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Chuyển đổi mã Python sang định dạng JSON tương thích với nền tảng `app.ohstem.vn` (Google Blockly).
- **Vấn đề / Câu hỏi:** Làm thế nào để tự động phân tích script Python, ánh xạ sang các khối Blockly, tạo ID chuẩn và xuất file JSON import được vào OhStem?
- **Cách giải quyết / Logic lõi:** 
  1. **Phân tích cú pháp:** Dùng module `ast` của Python để chuyển mã nguồn thành cây cú pháp trừu tượng (AST).
  2. **Duyệt & Ánh xạ:** Kế thừa `ast.NodeVisitor`, ghi đè các phương thức `visit_...` để chuyển node Python thành đối tượng JSON của Blockly.
  3. **Tạo ID chuẩn:** Mô phỏng `Blockly.utils.idGenerator.genUid()` sinh chuỗi 20 ký tự ngẫu nhiên từ bộ 92 ký tự (`soup_`).
  4. **Xây dựng JSON:** Ghép mảng `variables` và đối tượng `blocks`. Liên kết các khối tuần tự qua thuộc tính `next`.
  **Code lõi (Python):**
  ```python
  import secrets, string, ast

  def generate_blockly_uid():
      soup = string.ascii_letters + string.digits + "!@#$%^&*()_+-={};':\"\\|,.<>/?~`"
      return ''.join(secrets.choice(soup) for _ in range(20))

  class BlocklyJsonBuilder(ast.NodeVisitor):
      def __init__(self):
          self.variables = {}
          self.top_level_blocks = []
          self.last_top_block = None

      def _add_top_block(self, block_json):
          if self.last_top_block:
              self.last_top_block['next'] = {'block': block_json}
          else:
              self.top_level_blocks.append(block_json)
          self.last_top_block = block_json

      def _get_variable_id(self, name):
          if name not in self.variables:
              self.variables[name] = {"name": name, "id": generate_blockly_uid()}
          return self.variables[name]['id']

      def visit_Assign(self, node):
          # Xử lý gán biến đơn giản (x = value)
          if len(node.targets) == 1 and isinstance(node.targets[0], ast.Name):
              var_id = self._get_variable_id(node.targets[0].id)
              # ... tạo block 'variables_set' và gọi self._add_top_block()

      def get_final_json(self):
          return {"variables": list(self.variables.values()), 
                  "blocks": {"languageVersion": 0, "blocks": self.top_level_blocks}}
  ```
- **Ghi chú quan trọng:** 
  - **Nền tảng lõi:** `app.ohstem.vn` là bản tùy chỉnh của Google Blockly. Mọi quy tắc ID và cấu trúc JSON phải tuân thủ tuyệt đối chuẩn Blockly.
  - **Bộ ký tự `soup_`:** Gồm 92 ký tự (chữ, số, ký tự đặc biệt). ID dài cố định 20 ký tự. Cần xử lý escape/trích dẫn đúng cách khi nhúng vào JSON/JS để tránh lỗi cú pháp.
  - **Phân biệt thực thể:** Biến và khối lệnh dùng chung cơ chế tạo ID. Sự khác biệt được xác định bằng vị trí ngữ nghĩa trong JSON (`variables` vs `blocks`).
  - **Giới hạn chuyển đổi:** Chỉ hỗ trợ tập con cú pháp Python ánh xạ trực tiếp được sang Blockly (gán, vòng lặp, điều kiện, gọi hàm cơ bản). Cấu trúc phức tạp cần cơ chế bỏ qua hoặc báo lỗi.
  - **Liên kết khối:** Sử dụng thuộc tính `next` để nối các khối cấp cao nhất thành chuỗi thực thi tuần tự.
