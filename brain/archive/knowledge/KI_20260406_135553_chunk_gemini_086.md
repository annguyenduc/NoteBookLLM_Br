# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Lập trình Agent Minecraft (MakeCode) - Xây dựng cấu trúc hình học 3D (Lục giác thẳng đứng có viền rỗng/tâm đặc & tia nắng tỏa tròn).
- **Vấn đề / Câu hỏi:** Làm thế nào để xây dựng chính xác hình lục giác đứng và các tia nắng mà không bị lệch vị trí do lỗi tích lũy khi di chuyển tương đối (`agent.move`)?
- **Cách giải quyết / Logic lõi:** 
  - **Nguyên tắc điểm gốc cố định:** Luôn lưu vị trí khởi tạo `buildCenter = agent.getPosition()` và dùng nó làm chuẩn tuyệt đối cho mọi phép tính tọa độ.
  - **Thuật toán xây cột đối xứng:** Chia chiều cao cột thành nửa trên (`Math.ceil`) và nửa dưới (`Math.floor`), xây từ giữa ra hai đầu để đảm bảo đối xứng.
    ```typescript
    function buildVerticalColumn(height: number, blockType: number) {
        const midline = agent.getPosition();
        for (let i = 0; i < Math.ceil(height / 2); i++) { agent.place(FORWARD); agent.move(UP, 1); }
        agent.teleport(midline, WEST);
        for (let i = 0; i < Math.floor(height / 2); i++) { agent.move(DOWN, 1); agent.place(FORWARD); }
        agent.teleport(midline, WEST);
    }
    ```
  - **Tạo viền rỗng & tâm đặc:** Xây chồng 2 hình lục giác đặc. Hình lớn (màu viền) trước, sau đó tính toán lại vị trí căn giữa và xây hình nhỏ hơn (`canhTam = canh - 1`, màu tâm) đè lên.
  - **Xây tia nắng từ gốc:** Tính toán tọa độ 12 điểm trên chu vi dựa trên `buildCenter`, `teleport` Agent đến từng điểm rồi vẽ tia.
    ```typescript
    const R = Math.floor((3 * canh - 2) / 2);
    // Teleport tuyệt đối từ điểm gốc trước khi vẽ mỗi tia
    agent.teleport(positions.add(buildCenter, pos(R, 0, 0)), WEST);
    buildVerticalRay(length, 1, 0); 
    ```
- **Ghi chú quan trọng:** 
  - `canh` (số cạnh) bắt buộc là **số chẵn > 2** để thuật toán chia đôi chiều cao hoạt động đối xứng.
  - Công thức chiều rộng tổng thể của lục giác: `totalWidth = 3 * canh - 2`.
  - **Chống sai số tích lũy:** Tuyệt đối tránh dùng chuỗi `agent.move()` liên tiếp cho các bước định vị quan trọng. Luôn dùng `agent.teleport()` kết hợp `positions.add(buildCenter, pos(x,y,z))`.
  - Hàm vẽ tia cần xử lý hướng di chuyển động: `if (x_step > 0) agent.move(RIGHT, 1); else if (x_step < 0) agent.move(LEFT, 1);` trước khi `agent.place(FORWARD)`.
