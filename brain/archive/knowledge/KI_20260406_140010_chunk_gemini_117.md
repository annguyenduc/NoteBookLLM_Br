# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Tích hợp MCP (Model Context Protocol) và quy trình phát triển dự án Python cho Minecraft Education Edition (MCEE) & MakeCode.
- **Vấn đề / Câu hỏi:** MCP có tương thích trực tiếp với MCEE không? Đâu là IDE, kiến trúc và quy trình tối ưu để giáo viên tạo & triển khai dự án học tập Python trên nền tảng này?
- **Cách giải quyết / Logic lõi:** 
  - **Giới hạn tương thích:** Các MCP server mã nguồn mở hiện có (`rgbkrk/rcon-mcp`, `gerred/mcpmc`) chỉ hoạt động với **Minecraft: Java Edition** (qua giao thức RCON hoặc thư viện Mineflayer). MCEE (nền tảng Bedrock) có môi trường đóng, **không hỗ trợ kết nối MCP trực tiếp** vào game engine.
  - **IDE chính thức:** Microsoft MakeCode (trong Code Builder) là IDE duy nhất được hỗ trợ chính thức cho MCEE. Microsoft sẽ ngừng hỗ trợ Tynker & Python Notebooks từ 07/2025.
  - **Quy trình phát triển (Giáo viên):** Soạn thảo, quản lý phiên bản & debug mã trên IDE chuyên nghiệp (VS Code/PyCharm) → Sao chép mã đã tinh chỉnh vào trình soạn thảo Python của MakeCode trong MCEE để học sinh chạy trực tiếp.
  - **Mẫu MCP Server (Python) hỗ trợ giáo dục:**
    ```python
    from mcp.server import Server
    server = Server("minecraft-edu-helper")

    @server.tool()
    async def generate_custom_block(block_name: str, description: str, params: list) -> str:
        # Logic sinh code TypeScript/JSON cho MakeCode extension
        pass

    @server.tool()
    async def suggest_lesson_plan(grade: str, topic: str, duration: int) -> str:
        # Logic đề xuất giáo án & hoạt động coding theo curriculum
        pass
    ```
  - **Cấu hình MCP Hub (JSON):**
    ```json
    {
      "mcpServers": {
        "minecraft-edu": {
          "command": "python",
          "args": ["server.py"],
          "env": { "MAKECODE_API_KEY": "your_key" }
        }
      }
    }
    ```
  - **Chiến lược giảng dạy 2 tầng:**
    1. *Cơ bản:* Dùng MakeCode (chuyển đổi Block ↔ Python) để dạy tư duy tính toán, cú pháp, vòng lặp, hàm & điều khiển Agent.
    2. *Nâng cao:* Dùng `rcon-mcp` + Minecraft Java Edition + Docker + LLM để dạy kiến trúc Client-Server, API, giao thức mạng và Prompt Engineering.
- **Ghi chú quan trọng:** 
  - Thư viện `mcpi` (Minecraft Pi API) **hoàn toàn không tương thích** với MCEE.
  - MCP hoạt động dựa trên 4 primitives cốt lõi: `Tools` (thực thi hành động), `Resources` (cung cấp dữ liệu ngữ cảnh), `Prompts` (mẫu lệnh/workflow), `Sampling` (cho phép server yêu cầu LLM suy luận).
  - Khi phát triển MCP cho MCEE, nên tập trung vào các công cụ hỗ trợ gián tiếp (sinh code MakeCode, lập kế hoạch bài học, debug logic, theo dõi tiến độ) thay vì cố gắng kết nối trực tiếp vào runtime của game.
