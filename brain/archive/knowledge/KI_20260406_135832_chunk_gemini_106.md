# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Tối ưu hóa chi phí & Quy trình phát triển ứng dụng Web với Cursor IDE (Mô hình định giá Compute-based).
- **Vấn đề / Câu hỏi:** Cursor chuyển gói Pro từ định giá theo số lượng yêu cầu sang tiêu thụ tính toán (compute-based), gây khó dự đoán chi phí và dễ cạn kiệt tín dụng. Làm sao để quản lý ngân sách AI hiệu quả, tránh lãng phí và xây dựng ứng dụng To-Do List tối ưu?
- **Cách giải quyết / Logic lõi:** 
  1. **Chiến lược quản lý chi phí & Prompt Engineering:**
     - Chia nhỏ tác vụ lớn thành các prompt đơn giản, kiểm soát & test từng bước.
     - Phân bổ mô hình AI: Dùng mô hình rẻ (GPT-4.1, Gemini) cho HTML/CSS cơ bản; dành mô hình mạnh (Sonnet 4) cho logic phức tạp.
     - Tránh chế độ `Auto` cho công việc chuyên nghiệp do chất lượng không ổn định và thiếu minh bạch.
  2. **Code lõi To-Do List (JS + LocalStorage):**
     ```javascript
     // Quản lý trạng thái & LocalStorage
     const STORAGE_KEY = 'todoList';
     const saveTasks = (tasks) => localStorage.setItem(STORAGE_KEY, JSON.stringify(tasks));
     const loadTasks = () => JSON.parse(localStorage.getItem(STORAGE_KEY)) || [];

     // Logic thêm task (tối ưu token)
     function addTask() {
       const input = document.querySelector('#task-input');
       const text = input.value.trim();
       if (!text) return;
       
       const tasks = loadTasks();
       tasks.push({ id: Date.now(), text, completed: false });
       saveTasks(tasks);
       renderTasks(); // Hàm render giao diện
       input.value = '';
     }
     ```
  3. **Gỡ lỗi & Tái cấu trúc tiết kiệm:**
     - Copy lỗi từ Browser Console -> Dán vào Chat (`Ctrl+L`) -> Hỏi nguyên nhân & fix. Cung cấp log cụ thể giúp AI trả lời chính xác với ít token hơn.
     - Tái cấu trúc từng hàm nhỏ bằng `Ctrl+K`, commit Git ngay sau khi code chạy ổn định.
  4. **Tích hợp Git (Lưới an toàn tài chính):**
     ```bash
     git init
     git add .
     git commit -m "feat: hoàn thành CRUD task & localStorage"
     ```
     Dùng `@Commit` hoặc `@Branch` trong prompt để AI nắm bắt ngữ cảnh thay đổi, giảm prompt lại.
- **Ghi chú quan trọng:** 
  - **Phím tắt cốt lõi:** `Ctrl+L` (Chat đối thoại), `Ctrl+I` (Agent/Composer xử lý đa tệp), `Ctrl+K` (Inline Edit tại chỗ), `Tab` (Accept suggestion).
  - **Cung cấp ngữ cảnh:** Dùng `@tên_tệp`, `@thư_mục`, `@Docs`, `@Web` để AI truy xuất chính xác, giảm thiểu hallucination & lãng phí tín dụng.
  - **Tư duy phát triển:** "Bạn là Kiến trúc sư, AI là Thợ xây". Luôn review code AI sinh ra, không chấp nhận mù quáng.
  - **Cảnh báo định giá:** Gói Pro $20 tương đương ~225 request Sonnet 4 hoặc ~650 request GPT-4.1. Hết tín dụng sẽ bị ép chuyển sang `Auto` hoặc tính phí vượt mức. Cần theo dõi `cursor.com/dashboard` thường xuyên.
