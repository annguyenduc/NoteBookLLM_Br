# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Giao thức Ngữ cảnh Mô hình (MCP) trong Cursor IDE
- **Vấn đề / Câu hỏi:** Cách tích hợp, cấu hình, vận hành và khắc phục sự cố MCP để mở rộng khả năng của Cursor Agent (kết nối DB, GitHub, Notion, tự động hóa tác vụ) và cơ chế tính phí request.
- **Cách giải quyết / Logic lõi:** 
  - **Kiến trúc & Transport:** 
    - `stdio`: Chạy cục bộ, Cursor tự quản lý tiến trình. Cấu hình qua `command`, `args`, `env`. Phù hợp dev cá nhân.
    - `SSE`: Chạy cục bộ hoặc remote, người dùng tự quản lý server. Cấu hình qua `url`. Phù hợp team/dịch vụ mạng.
  - **Cấu hình `mcp.json`:** Đặt tại `.cursor/mcp.json` (theo dự án) hoặc `~/.cursor/mcp.json` (toàn cục).
  - **Snippet cấu hình cơ bản (Node.js/stdio):**
    ```json
    {
      "mcpServers": {
        "my-server": {
          "command": "npx",
          "args": ["-y", "my-mcp-package"],
          "env": { "API_KEY": "your_key" }
        }
      }
    }
    ```
  - **Cấu hình WSL (khắc phục lỗi môi trường shell):**
    ```json
    {
      "mcpServers": {
        "wsl-node": {
          "command": "wsl",
          "args": ["bash", "-c", "source /home/USER/.nvm/nvm.sh && npx my-server"],
          "env": { "KEY": "val" }
        }
      }
    }
    ```
  - **Yolo Mode (Tự động hóa):** Bật trong Settings. Dùng `Prompt`, `Allow list` (vd: `npm test`, `mkdir`), `Deny list` (vd: `rm -rf`, `git push`) để tạo sandbox an toàn cho AI tự chạy tool.
  - **Image Injection:** MCP server phải trả về ảnh dạng chuỗi `base64` trong phản hồi tool để Cursor tự động giải mã, hiển thị và cho LLM phân tích đa phương thức.
- **Ghi chú quan trọng:** 
  - Tool calls qua MCP **tính vào quota request** của gói Cursor. Mức tiêu thụ phụ thuộc vào gói dịch vụ và loại model.
  - Cursor mặc định yêu cầu **phê duyệt thủ công (Tool Approval)** trước khi thực thi tool.
  - **Giới hạn hiện tại:** Chỉ nhận diện tối đa 40 tools đầu tiên; `stdio` không hoạt động ổn định trong môi trường Remote/SSH; tính năng "Resources" chưa được hỗ trợ.
  - **Bảo mật:** Luôn truyền API key/token qua trường `env`. Thêm `mcp.json` vào `.gitignore` nếu chứa key thật.
  - **Lỗi UI:** Từ v0.47.0, nút thêm server có thể mở trực tiếp file JSON thay vì modal. Cấu hình thủ công trong file vẫn hoạt động bình thường.
