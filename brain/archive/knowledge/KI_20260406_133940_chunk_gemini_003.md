# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Khung lý thuyết TPACK trong giáo dục & Kiến trúc triển khai AI Agent (Skill-Workflow) trên nền tảng Antigravity IDE.
- **Vấn đề / Câu hỏi:** 
  1. TPACK là gì, cấu trúc ra sao và được ứng dụng thế nào tại Việt Nam?
  2. Làm thế nào để nâng cấp bộ kỹ năng cho AI Agent (`codymaster`) và thiết lập luồng công việc tự trị trên Antigravity IDE?
  3. Nên chọn kho `tranhieutt/for-learn-claude-cli` hay `tody-agent/codymaster` để tích hợp?
- **Cách giải quyết / Logic lõi:** 
  - **TPACK Framework:** Là mô hình giao thoa 3 kiến thức: Nội dung môn học (CK), Phương pháp sư phạm (PK), Công nghệ (TK). Giá trị cốt lõi nằm ở các vùng giao thoa (PCK, TCK, TPK) và tâm điểm TPACK. Tại VN, được áp dụng trong đào tạo ĐH Sư phạm, lồng ghép vào chuẩn nghề nghiệp giáo viên (Bộ GD&ĐT/ETEP), và làm khung năng lực thiết kế học liệu STEM/E-learning.
  - **Kiến trúc Antigravity IDE & Quản lý ngữ cảnh:** Giải quyết "Context Saturation" (bão hòa ngữ cảnh) bằng 3 lớp biệt lập:
    - `Rules` (`.agent/rules/`): Ràng buộc thụ động, luôn bật (bảo mật, coding convention).
    - `Skills` (`.agents/skills/`): Phương pháp luận thực thi ("Bộ não"), dùng cơ chế **Progressive Disclosure** (chỉ nạp khi ngữ cảnh khớp) để tiết kiệm token.
    - `Workflows` (`.agent/workflows/`): Chuỗi tác vụ vĩ mô, kích hoạt bằng `/workflow-name`, hỗ trợ phân nhánh & gọi lồng ghép.
  - **Lựa chọn kho lưu trữ:** `codymaster` là Universal AI Agent Skills Framework, tương thích native với Antigravity/Gemini CLI, dùng định dạng `SKILL.md`. `for-learn-claude-cli` là mã nguồn rò rỉ của Claude Code, chỉ dùng cho nghiên cứu/bảo mật, không dùng để tích hợp skill.
  - **Quy trình triển khai Skill-Workflow:**
    1. Cấu trúc thư mục: `.agents/skills/`, `.agent/workflows/`, `agents.md`.
    2. Import: Dùng CLI hoặc clone thủ công vào thư mục dự án.
    3. Định dạng `SKILL.md` (YAML frontmatter + Markdown body).
    4. Tạo Workflow điều phối chuỗi tác vụ.
    5. Kích hoạt & giám sát qua Agent Manager.
  - **Code/Config mẫu:**
    ```bash
    # Cài đặt extension toàn cục cho Antigravity/Gemini CLI
    gemini extensions install https://github.com/tody-agent/codymaster
    ```
    ```yaml
    # Cấu trúc bắt buộc của tệp SKILL.md
    ---
    name: cm-tdd
    description: Kỹ năng chuyên sâu phục vụ Lập trình Hướng Kiểm thử. Kích hoạt khi cần viết Unit Test, phân tích code coverage hoặc tối ưu logic.
    ---
    # Phần thân Markdown: Quy trình thực thi, giao thức lỗi, công cụ gọi...
    ```
- **Ghi chú quan trọng:** 
  - Luôn ưu tiên `codymaster` cho môi trường thực tế do tính chuẩn hóa, an toàn pháp lý và tương thích cao. Tránh dùng repo rò rỉ (`for-learn-claude-cli`) vì rủi ro bản quyền và không hỗ trợ cơ chế extension.
  - Cơ chế **Progressive Disclosure** của Antigravity giúp AI chỉ đọc `SKILL.md` khi cần, tránh lãng phí token và giảm "Context Rot".
  - Workflow giới hạn an toàn ~12.000 ký tự/tệp để tránh tràn bộ nhớ ngữ cảnh. Hỗ trợ gọi lồng ghép (`/workflow-1` gọi `/workflow-2`) để xây dựng pipeline tự trị.
