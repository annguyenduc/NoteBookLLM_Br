# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Thiết kế kiến trúc Multi-Agent tự học lập trình & Quản lý tài nguyên nền tảng AI
- **Vấn đề / Câu hỏi:** Làm thế nào để cấu hình Rule, Workflow và Agent Manager trong framework "Antigravity" để tự động hóa việc học từ thư mục khóa học (`Cotai_AI_Course` chứa file `.ipynb`)? Cách kiểm tra quota và tối ưu model?
- **Cách giải quyết / Logic lõi:** 
  **1. Kiến trúc Agent & System Prompt:**
  - `Agent Manager` (Điều phối): Quét cấu trúc thư mục, phân phối task tuần tự. Không tự học.
  - `Agent Learner` (Học viên): Tóm tắt lý thuyết, viết code giải bài tập. *Rule:* Tuyệt đối không được truy cập file `DapAn`.
  - `Agent Reviewer` (Trợ giảng): So sánh code Learner với `DapAn`. *Rule:* Chỉ trả về `[PASS]` hoặc `[FAIL]` + gợi ý lỗi. Không đưa code mẫu.
  
  **2. Workflow (State Machine):**
  `Manager` quét `BaiGiang` -> `Learner` tóm tắt -> `Manager` gửi `BaiTap` + Knowledge -> `Learner` viết code -> `Manager` gửi code + `DapAn` cho `Reviewer` -> Nếu `FAIL`: lặp lại bước làm bài với gợi ý. Nếu `PASS`: chuyển bài. Xử lý edge-case: thiếu `BaiTap`/`DapAn` -> chuyển sang `Human-in-the-loop`.
  
  **3. Code lõi (Python Pseudo-code cho Manager):**
  ```python
  import os, glob

  class AgentManager:
      def __init__(self, root_path):
          self.root_path = root_path
          self.curriculum = {}

      def scan_curriculum(self):
          for module in sorted(glob.glob(os.path.join(self.root_path, "Module_*"))):
              self.curriculum[os.path.basename(module)] = []
              for lesson in sorted(glob.glob(os.path.join(module, "Bai_*"))):
                  self.curriculum[os.path.basename(module)].append({
                      "name": os.path.basename(lesson),
                      "lecture": glob.glob(os.path.join(lesson, "BaiGiang_*.ipynb")),
                      "exercise": glob.glob(os.path.join(lesson, "BaiTap_*.ipynb")),
                      "solution": glob.glob(os.path.join(lesson, "DapAn_*.ipynb"))
                  })

      def execute_cycle(self, lesson):
          knowledge = trigger_agent("Learner", "read", lesson['lecture'][0])
          status = "FAIL"
          while status == "FAIL":
              code = trigger_agent("Learner", "solve", lesson['exercise'][0], knowledge)
              if lesson['solution']:
                  res = trigger_agent("Reviewer", "grade", code, lesson['solution'][0])
                  status = res['status']
  ```
  **4. Xử lý file `.ipynb`:** File notebook là JSON. Cần parse key `"source"` trong mảng `"cells"`, lọc `"cell_type": "markdown"` và `"code"`, bỏ metadata để tránh vượt token limit.
  **5. Quản lý Quota & Model:** Kiểm tra qua CLI (`antigravity quota list`) hoặc Dashboard. Chiến lược: Dùng model nhẹ (7B/8B) để nháp/debug, model mạnh (GPT-4/Claude 3.5) cho logic phức tạp. Ưu tiên model có `Context Window` lớn cho code dài và `Vision` cho phân tích sơ đồ/mạch.
- **Ghi chú quan trọng:** 
  - Nếu mục tiêu là *tự học*, không nên tốn thời gian code hệ thống Agent phức tạp. Hãy dùng chính LLM đóng vai Manager/Reviewer trong chat để tối ưu tốc độ.
  - Prompt phải nghiêm ngặt về phân quyền dữ liệu (đặc biệt cách ly `DapAn` khỏi `Learner`) để đảm bảo tính tự học thực sự.
  - Workflow cần cơ chế `Human-in-the-loop` khi thiếu đáp án chuẩn hoặc gặp lỗi logic ngoài dự kiến.
