# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Lập trình Minecraft Education Edition (JavaScript Code Builder & Command Blocks) để tự động hóa xây dựng cấu trúc 3D.
- **Vấn đề / Câu hỏi:** 
  1. Tạo khối Rubik 10x10x10 với các ô màu 2x2.
  2. Thiết kế thuật toán & code để Agent xây hình lục giác (rỗng & đặc).
  3. Khắc phục lỗi Agent không đặt được khối trên mặt đất & lỗi xây ra hình vuông thay vì lục giác.
- **Cách giải quyết / Logic lõi:** 
  **1. Khối Rubik 10x10x10:**
  - *Công thức kích thước:* `(3 ô màu × 2 block) + (4 đường viền × 1 block) = 10 block`.
  - *Command Block:* Tạo khung rỗng `/fill ~ ~ ~ ~9 ~9 ~9 black_concrete hollow`, sau đó dùng 54 lệnh `/fill` riêng lẻ cho các ô 2x2 trên 6 mặt.
  - *JavaScript:* Dùng vòng lặp lồng nhau vẽ lưới 3x3 sticker.
  ```javascript
  // Logic vẽ 1 mặt (ví dụ mặt trước +Z)
  for (let i = 0; i < 3; i++) {
      for (let j = 0; j < 3; j++) {
          let from = positions.create(startX + 1 + i*3, startY + 1 + j*3, startZ + 9);
          let to = from.add(positions.create(1, 1, 0)); // Tạo khối 2x2
          blocks.fill(Block.BlueConcrete, from, to);
      }
  }
  ```
  **2. Thuật toán Lục giác (Agent):**
  - *Ràng buộc:* Agent chỉ di chuyển 4 hướng & quay 90° → Lục giác là hình xấp xỉ (approximation).
  - *Thuật toán 1 (Rỗng - Đỉnh phẳng):* Chia 6 cạnh (2 thẳng, 4 chéo). Lặp 6 lần, mỗi cạnh kết hợp `move(FORWARD/BACK)` + `move(LEFT/RIGHT)` + `place(DOWN)`.
  - *Thuật toán 2 (Đặc):* Chia tầng thành 3 phần: Tam giác trên (mở rộng `width += 2`), Hình chữ nhật giữa (giữ nguyên), Tam giác dưới (thu hẹp `width -= 2`). Xây từng hàng ngang, reset `homePos` sau mỗi hàng.
  **3. Khắc phục lỗi phổ biến:**
  - *Lỗi không đặt được block trên đất:* `agent.place(DOWN)` thất bại nếu không có khoảng trống không khí.
  ```javascript
  agent.teleportToPlayer();
  agent.move(UP, 1); // Bắt buộc: Nhấc Agent lên 1 block trước khi xây
  ```
  - *Lỗi xây ra hình vuông:* Sai điều kiện vòng lặp cạnh. Phải là `side < 6`, không phải `side < 4`.
- **Ghi chú quan trọng:** 
  - Luôn dùng `agent.teleport(homePos, WEST)` hoặc tương đương để reset tọa độ Agent sau mỗi hàng/cạnh, tránh lệch hướng tích lũy.
  - Khi xây nhiều tầng, cập nhật vị trí Y đúng cách: `agent.move(UP, h + 1)` hoặc `h + 2` tùy vào việc đã nhấc Agent ban đầu hay chưa.
  - Ưu tiên Thuật toán 1 cho lục giác rỗng do logic đơn giản, ít lỗi tính toán tọa độ hơn Thuật toán 2.
