# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Lộ trình tự học Data Engineering & Data Analytics (1h/ngày), Chiến lược tối ưu khóa học Google, Kiến trúc Data Pipeline & Yêu cầu hệ thống Google Antigravity.
- **Vấn đề / Câu hỏi:** Làm thế nào để xây dựng lộ trình học Data Engineering/Analytics hiệu quả với quỹ thời gian 1h/ngày, đảm bảo sức khỏe, đồng thời nắm rõ yêu cầu kỹ thuật của công cụ AI Antigravity và xu hướng thị trường 2025-2026?
- **Cách giải quyết / Logic lõi:** 
  1. **Chiến lược học 1h/ngày (30% Lý thuyết - 70% Thực hành):**
     - `0-15p`: Xem video, ghi chú 3 khái niệm cốt lõi.
     - `15-45p`: Thực hành code trực tiếp (BigQuery/Colab). Debug lỗi bằng AI là bước học quan trọng nhất.
     - `45-50p`: Nghỉ ngơi (Quy tắc 20-20-20 cho mắt).
     - `50-60p`: Làm Quiz hoặc commit code lên GitHub.
  2. **Lộ trình Data Engineering (8 tháng):**
     - `Tháng 1-2`: SQL nâng cao (Window Functions, CTEs, Data Modeling) + Python (Pandas, API, Cron jobs).
     - `Tháng 3-5`: Pipeline Orchestration với `Apache Airflow` (viết DAGs) & `dbt` (Transform, version control, testing).
     - `Tháng 6-7`: Cloud (GCP/AWS) & Data Warehouse (`BigQuery`/`Snowflake` - lưu trữ columnar).
     - `Tháng 8`: Streaming (`Kafka`/`PubSub`) & Capstone Project.
  3. **Kiến trúc Pipeline mẫu (Capstone):**
     ```python
     # Logic luồng dữ liệu tự động (Airflow DAG)
     extract_task >> transform_task(dbt) >> load_task(BigQuery) >> dashboard_task
     # Nguồn: IoT/Cảm biến -> Cloud Storage -> Airflow lên lịch -> dbt làm sạch -> BigQuery -> BI Dashboard
     ```
  4. **Google Antigravity:** Bắt buộc có Internet cho AI Agents (Gemini), xác thực tài khoản, Browser Agent & cập nhật. Chế độ Offline chỉ hỗ trợ chỉnh sửa code cơ bản (nhân VS Code).
- **Ghi chú quan trọng:** 
  - **Sức khỏe & Nhất quán:** Học khung giờ `21:30-22:30`, ngủ trước `23:00`. Ưu tiên duy trì thói quen hàng ngày hơn học bù cuối tuần.
  - **Portfolio chuẩn:** Cấu trúc `Problem → Approach → Insight → Impact`. Dùng dữ liệu thực tế (log máy móc, cảm biến) thay vì dataset mẫu.
  - **Xu hướng 2025-2026:** AI-native platforms, Multi-cloud, MLOps & Data Governance là nhóm kỹ năng tăng thu nhập & thăng tiến nhanh nhất.
  - **Tối ưu khóa Google DA:** Lướt nhanh phần 1-3, chuyên sâu SQL/Tableau (4-6), tự chuyển đổi bài tập R sang Python (Pandas) để phù hợp với Data Engineering/Automation.
