# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Hiệu chuẩn máy in 3D (Bambu Studio), Phần mềm mô hình 3D offline nhẹ, Thao tác nâng vật thể trong Slicer, Xử lý HTTP Request trên MicroPython, Đánh giá tài liệu nền tảng Robot.
- **Vấn đề / Câu hỏi:** 
  1. Cách tinh chỉnh thông số in 3D tối ưu cho máy ngoài/filament mới.
  2. Tìm công cụ thiết kế 3D thay thế Tinkercad (chạy offline, nhẹ, cấu hình thấp).
  3. Khắc phục hiện tượng vật thể tự "hít" xuống bàn in khi kéo trục Z trong Bambu Studio.
  4. Sửa lỗi cú pháp callback bất đồng bộ khi gọi API HTTP trên Yolo UNO/MicroPython.
  5. Xác định đối tượng phù hợp cho giáo trình Toán ứng dụng trong Robot.
- **Cách giải quyết / Logic lõi:** 
  **1. Hiệu chuẩn Bambu Studio (Calibration):**
  - `Flow Rate`: Thực hiện 2 pass (Thô & Tinh). Công thức cập nhật: `Flow_Mới = Flow_Cũ × (100 + Modifier) / 100`.
  - `Pressure Advance`: Dùng tháp/đường/mẫu để tìm giá trị K giúp góc in vuông vức, không phình/tóp.
  - `Max Flowrate`: `Max = Tốc_độ_đầu + (Chiều_cao_bắt_đầu_lỗi × Bước_nhảy)`. Khuyến nghị giảm 5-10% kết quả để in ổn định.
  - `Retraction`: PLA/ABS thường tối ưu ở 0.2-0.4mm. Nếu vẫn bị kéo tơ → sấy nhựa hoặc kiểm tra nozzle.
  
  **2. Phần mềm 3D Offline thay thế Tinkercad:**
  - `Microsoft 3D Builder`: Nhẹ, tư duy cộng/trừ khối, có sẵn trên Windows.
  - `SketchUp Make 2017`: Bản offline miễn phí cuối cùng, công cụ Push/Pull trực quan, chạy tốt trên máy yếu.
  - `Blockbench`: Portable (~100MB), low-poly, xuất `.obj`, giao diện kéo thả khối phù hợp giáo dục STEM.
  - `FreeCAD`: Tham số hóa, chính xác mm, độ khó cao, phù hợp thiết kế kỹ thuật.

  **3. Nâng vật thể trong Bambu Studio (Tránh Auto Drop):**
  - Cơ chế mặc định giữ vật chạm bàn. Để nâng tự do: Tạo khối neo nhỏ (ví dụ Cube 1x1mm) đặt trên bàn → Chọn vật cần nâng + khối neo → Chuột phải `Assemble` → Trong panel `Objects`, chỉ chọn phần tử con (sub-part) để kéo trục Z. Bật `Support` nếu in lơ lửng.

  **4. Xử lý HTTP Request trên MicroPython (Yolo UNO/OhStem):**
  - `urequests` hoạt động đồng bộ (blocking), không hỗ trợ callback `.on_successed()`. Phải dùng `try...except` kết hợp kiểm tra `status_code`.
  ```python
  import urequests

  def on_success(data): print("✅ OK:", data)
  def on_fail(reason): print("❌ Lỗi:", reason)

  try:
      response = urequests.get('https://api.example.com/data')
      if response.status_code == 200:
          on_success(response.text)
      else:
          on_fail(f"HTTP {response.status_code}")
      response.close() # Bắt buộc giải phóng RAM trên ESP32
  except Exception as e:
      on_fail(str(e))
  ```
- **Ghi chú quan trọng:** 
  - Luôn tạo `New Project` sau khi hoàn tất bất kỳ bài hiệu chuẩn nào trong Bambu Studio.
  - Sấy khô filament trước khi hiệu chuẩn để loại bỏ sai số do ẩm.
  - Giáo trình *"Cơ sở Toán ứng dụng trong Kỹ thuật Robot"* (NXB ĐHQG Hà Nội) nặng lý thuyết (ma trận, động học thuận/nghịch, động lực học). Phù hợp sinh viên/kỹ sư chuyên sâu, không dành cho người mới chơi DIY/Arduino thực hành nhanh.
  - Trên vi điều khiển RAM thấp (ESP32), luôn đóng kết nối HTTP (`response.close()`) và ưu tiên xử lý lỗi đồng bộ để tránh treo chương trình.
