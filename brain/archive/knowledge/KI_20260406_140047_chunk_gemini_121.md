# TÓM TẮT TRI THỨC KỸ THUẬT
- **Chủ đề chính:** Kiến trúc MCP Server, Cấu hình Cursor IDE, Quản lý Driver Windows & Giới hạn AI Video Generation
- **Vấn đề / Câu hỏi:** 
  1. Tại sao cần MCP Server khi AI Agent đã có tool tích hợp? Cách thiết lập MCP Hub (>40 tools) trong Cursor IDE?
  2. Làm thế nào để dọn dẹp an toàn thư mục `DriverStore\FileRepository` trên Windows?
  3. Google Veo 2 có hỗ trợ tạo âm thanh native không?
- **Cách giải quyết / Logic lõi:** 
  **1. MCP Server & Cursor IDE:**
  - *Lợi ích cốt lõi:* MCP đóng vai trò lớp trừu tượng, giúp mở rộng tool (API bên thứ 3, DB, CLI), quản lý tập trung, kiểm soát bảo mật/quyền hạn, phân phối tải xử lý, và cập nhật độc lập với IDE.
  - *Cấu hình Cursor (Settings/JSON):*
    ```json
    {
      "mcpServers": {
        "tên-server": {
          "command": "npx",
          "args": ["-y", "tên-gói-mcp"]
        }
      }
    }
    ```
  - *Phương thức kết nối:* Hỗ trợ `command` (chạy local qua npx/node) hoặc `url` (HTTP/SSE). Cursor tự động quét và hiển thị tools từ các server đã kết nối.
  **2. Windows DriverStore Cleanup:**
  - Tuyệt đối không xóa thủ công `C:\Windows\System32\DriverStore\FileRepository\`.
  - *Quy trình an toàn:* Mở `Disk Cleanup` → Nhấn `Clean up system files` → Tick chọn `Device driver packages` → `OK` → `Delete Files`.
  **3. AI Video (Google Veo):**
  - Veo 2: Chỉ xuất video, không hỗ trợ âm thanh native. Cần dùng tool chỉnh sửa ngoài để ghép audio.
  - Veo 3: Hỗ trợ tạo âm thanh native trực tiếp (SFX, tiếng môi trường, thoại).
- **Ghi chú quan trọng:** 
  - Cursor có giới hạn hiển thị ~40 tools/session. Nên vô hiệu hóa các MCP server không dùng để tránh lỗi quá tải.
  - MCP Server giúp cách ly AI Agent khỏi tài nguyên nhạy cảm (API Keys, DB credentials), cho phép logging & lọc request không hợp lệ.
  - `DriverStore` lưu nhiều phiên bản driver (vd: `intcoed.inf`, `nvaci.inf`, `iigd_dch.inf`) để hỗ trợ rollback & tương thích phần cứng. Chỉ dùng Disk Cleanup để xóa version cũ an toàn.
  - Các nội dung sáng tạo/kịch bản quảng cáo đã được lọc bỏ theo tiêu chí kỹ thuật.
