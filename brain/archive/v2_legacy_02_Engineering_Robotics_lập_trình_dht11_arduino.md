# Lập trình & Khắc phục lỗi DHT11 (3 chân) trên Arduino

## Kết nối phần cứng
- **VCC**: 3.3V hoặc 5V (tùy board vi điều khiển)
- **DATA (OUT)**: Chân Digital bất kỳ (ví dụ: D2)
- **GND**: GND của board
- **Lưu ý**: Module DHT11 3 chân thường đã tích hợp sẵn điện trở pull-up 10kΩ. Nếu dùng cảm biến dạng chân trần, bắt buộc mắc thêm điện trở 10kΩ giữa chân DATA và VCC.

## Cài đặt thư viện
1. Arduino IDE → `Sketch` → `Include Library` → `Manage Libraries`
2. Tìm và cài đặt: `DHT sensor library by Adafruit`
3. Cài đặt kèm: `Adafruit Unified Sensor`

## Code mẫu cơ bản
```cpp
#include "DHT.h"
#define DHTPIN 2
#define DHTTYPE DHT11
DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(9600);
  dht.begin();
}

void loop() {
  float h = dht.readHumidity();
  float t = dht.readTemperature();
  
  if (isnan(h) || isnan(t)) {
    Serial.println("Lỗi đọc dữ liệu!");
    return;
  }
  
  Serial.print("Độ ẩm: "); Serial.print(h); Serial.println(" %");
  Serial.print("Nhiệt độ: "); Serial.print(t); Serial.println(" °C");
  delay(2000); // DHT11 yêu cầu tối thiểu 1-2s giữa các lần đọc
}
```

## Xử lý lỗi `avrdude: stk500_cmd(): programmer is out of sync`
Lỗi này cho thấy Arduino IDE mất đồng bộ giao tiếp với bootloader trong quá trình nạp code.
**Nguyên nhân & Giải pháp:**
- **Cáp USB chỉ sạc**: Không hỗ trợ truyền dữ liệu. → Thay cáp USB data chuẩn.
- **Sai cấu hình Board/Port**: Kiểm tra `Tools` → `Board` (Arduino Uno) và `Port` (COM đúng).
- **Xung đột chân Serial (D0/D1)**: Rút mọi thiết bị nối vào chân 0 và 1 trước khi nạp code.
- **Driver cổng COM (board clone)**: Cài driver CH340 hoặc CP2102 phù hợp với Windows.
- **Bootloader lỗi**: Thử nạp lại bootloader qua ISP hoặc kiểm tra chip ATmega328P.

## Lưu ý vận hành
- **Độ chính xác**: Sai số ±2°C (nhiệt độ) và ±5% (độ ẩm). Không dùng cho ứng dụng y tế/công nghiệp chính xác cao.
- **Tần suất đọc**: Tối đa 1Hz. Khuyến nghị `delay(2000)` để cảm biến ổn định và tránh treo.
- **Tương thích ESP8266/ESP32**: Code tương tự, chỉ cần đổi `DHTPIN` sang GPIO phù hợp và đảm bảo nguồn 3.3V ổn định.
