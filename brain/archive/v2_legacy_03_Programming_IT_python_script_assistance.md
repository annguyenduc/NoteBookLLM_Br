# Thực hành Python, Xử lý Ảnh & Quản lý Hệ thống

## 📦 Xử lý File & Thư viện Ảnh
- **Giải nén dữ liệu**: Luôn dùng lệnh `unzip <file.zip>` để trích xuất trước khi sử dụng. Không mở trực tiếp file nén bằng trình chỉnh sửa ảnh hoặc tự ý đổi đuôi file.
- **Thư viện chuẩn**: `Pillow` (Python Imaging Library) là công cụ phổ biến nhất để thao tác ảnh và thêm text overlay trong Python.
- **Truy xuất metadata ảnh**: Sau khi mở ảnh bằng Pillow, dùng `img.format` để lấy định dạng (JPEG, PNG...) và `img.size` để lấy kích thước `(width, height)`.

## 🧪 Kiểm thử & Xác thực Script
- **Kiểm thử từng bước**: Chạy và xác minh logic từng dòng lệnh trong Python interpreter trước khi thực thi script hoàn chỉnh, giúp phát hiện lỗi sớm và tránh gián đoạn.
- **Xác nhận kết quả đầu ra**: Dùng lệnh `ls <đường_dẫn>` (ví dụ: `ls /opt/icons`) để kiểm tra nhanh danh sách file đã được xử lý/lưu đúng vị trí.

## 🌐 Kiến trúc Hệ thống & Tích hợp API
- **Tích hợp thiết bị IoT/Smart**: Ưu tiên sử dụng thư viện/module có sẵn hỗ trợ API của thiết bị thay vì tự viết giao thức giao tiếp từ đầu.
- **Hệ thống phân tán (Distributed Systems)**:
  - **Mở rộng linh hoạt (Scalability)**: Tự động tăng/giảm tài nguyên server theo biến động lưu lượng thực tế.
  - **Dự phòng & Độ tin cậy (Redundancy)**: Sao chép dữ liệu trên nhiều node đảm bảo tính sẵn sàng cao, hệ thống vẫn hoạt động bình thường khi một server gặp sự cố.

## 🐳 Quản lý Môi trường Docker
- **Dọn dẹp an toàn**: Sử dụng `docker image prune` để xóa các Docker image không được gắn với container nào, giải phóng dung lượng ổ đĩa mà không làm ảnh hưởng đến môi trường đang vận hành.
