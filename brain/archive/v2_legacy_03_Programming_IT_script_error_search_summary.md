Tìm kiếm lỗi trong tệp tin log sử dụng Python
Tạo một tập lệnh Python để tìm kiếm lỗi trong tệp tin log. Sử dụng module `re` để tìm kiếm mẫu lỗi được định nghĩa bởi người dùng. 

Các bước thực hiện:
- Tạo một tệp tin Python mới và thêm dòng shebang.
- Nhập các module cần thiết như `sys`, `os`, và `re`.
- Định nghĩa một hàm `error_search` để tìm kiếm lỗi trong tệp tin log.
- Sử dụng hàm `input` để nhận loại lỗi mà người dùng muốn tìm kiếm.
- Mở tệp tin log và đọc từng dòng một.
- Sử dụng phương thức `search` của module `re` để tìm kiếm mẫu lỗi trong từng dòng.
- Nếu tìm thấy mẫu lỗi, thêm dòng đó vào danh sách lỗi.

Mã code:
```python
import sys
import os
import re

def error_search(log_file):
    error = input("What is the error? ")
    returned_errors = []
    with open(log_file, mode='r', encoding='UTF-8') as file:
        for log in file.readlines():
            error_patterns = ["error"]
            for i in range(len(error.split(' '))):
                error_patterns.append(r"{}".format(error.split(' ')[i].lower()))
            for error_pattern in error_patterns:
                if re.search(error_pattern, log):
                    returned_errors.append(log)
    return returned_errors
```
