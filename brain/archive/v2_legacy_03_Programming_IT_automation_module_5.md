# Kết nối VSCode (Windows) với WSL/Ubuntu

Không cần cài đặt lại VSCode bên trong WSL. Chỉ cần sử dụng extension chính thức để VSCode trên Windows điều khiển trực tiếp môi trường Ubuntu.

## Thiết lập nhanh
1. Mở VSCode trên Windows → cài extension **`Remote - WSL`** (Microsoft).
2. Mở dự án trong WSL bằng 1 trong 2 cách:
   - **Từ terminal WSL:** `cd /đường/dẫn/thư/mục` → chạy lệnh `code .`
   - **Từ VSCode:** `Ctrl + Shift + P` → gõ `Remote-WSL: New Window` → chọn thư mục cần làm việc.

## Cơ chế hoạt động & Lưu ý kỹ thuật
- **Phân tách giao diện & thực thi:** Editor render trên Windows, nhưng mọi lệnh biên dịch, thực thi script và quản lý package (`pip`, `apt`, `npm`...) đều chạy trong môi trường Ubuntu WSL.
- **Terminal tích hợp:** Tự động chuyển thành WSL shell, không cần chuyển đổi thủ công giữa Windows Terminal và WSL.
- **Khuyến nghị:** Atom đã chính thức ngừng phát triển (cuối 2022). VSCode kết hợp Remote WSL là giải pháp thay thế tối ưu, ổn định và tương thích hoàn toàn với các khóa học yêu cầu môi trường Linux trên Windows.
