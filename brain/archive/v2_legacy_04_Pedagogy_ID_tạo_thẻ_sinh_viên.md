# Kỹ thuật Prompt tạo Thẻ sinh viên bằng AI Image Generation

- **Mục đích:** Sinh hình ảnh thẻ sinh viên rõ nét, chuẩn bố cục thực tế thông qua mô hình tạo ảnh AI (DALL-E).
- **Cấu trúc Prompt tối ưu:**
  - `Thông tin định danh`: Tên trường, Họ tên, Mã số sinh viên, Ngày sinh, Ngày cấp, Ngày hết hạn.
  - `Mô tả hình ảnh`: Ảnh chân dung chuyên nghiệp, đặc điểm ngoại hình cụ thể (tóc, biểu cảm, góc máy).
  - `Thành phần thiết kế`: Chữ ký, nhãn phân loại (màu tương phản), logo chính thức, font chữ hiện đại, nền sáng, bố cục tối giản.
- **Lưu ý kỹ thuật:**
  - Viết prompt bằng tiếng Anh để AI xử lý từ khóa thiết kế và ngữ cảnh chính xác nhất.
  - Chỉ định rõ kích thước đầu ra (ví dụ: `1536x1024` hoặc `xlimage`) để đảm bảo độ phân giải và khả năng in ấn.
  - Sử dụng lệnh hệ thống để ngắt luồng xử lý ngay sau khi trả về ảnh, tránh hallucination hoặc phản hồi thừa.
  - Metadata trả về chứa `asset_pointer` và `gen_id` dùng để versioning, truy xuất file gốc hoặc tích hợp vào pipeline quản lý tài sản số.
