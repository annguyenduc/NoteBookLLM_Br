## 🛑 Настоящая проблема

У вас есть отличная идея. Вы описываете её AI-инструменту. Но сгенерированный интерфейс выглядит... **ужасно**.
Он выглядит шаблонно, «сделано нейросетью», без единой системы дизайна. Формы слишком длинные, кнопки слишком маленькие, а пользовательский опыт — сплошное разочарование.

Это реальность, с которой сталкиваются 90% основателей, разработчиков и продакт-менеджеров. Отличный код с уродливым интерфейсом = мёртвый продукт.

## ✨ Магическое решение

**MasterDesign Agent** превращает любой AI-инструмент для кодинга в **профессиональную дизайн-студию**.
Он работает незаметно в фоновом режиме вашей любимой IDE или CLI. Вы просто говорите свои идеи, а AI автоматически применяет мировые стандарты дизайна.

| ❌ Без MasterDesign Agent | ✦ С MasterDesign Agent |
|---------------------|--------------|
| AI генерирует типовой «нейро-стиль» | Интерфейс студийного качества |
| Непоследовательные цвета и отступы | Целостная профессиональная дизайн-система |
| Крошечные кнопки, запутанные меню | Оптимизация по 48 научным UX-законам |
| Нет проверки качества | 37 автоматизированных дизайн-тестов |
| Нанять дизайнера: $3K–$15K, ждать недели | **$0, результат за 60 секунд** |

## ⚡ Начните за 60 секунд

### Шаг 1: Проверьте Python
Убедитесь, что установлен Python 3:
```bash
python3 --version
```

### Шаг 2: Клонирование и установка
```bash
git clone https://github.com/relukdev/ux-master.git
cd ux-master
python3 scripts/install.py
```

Установщик автоматически обнаружит ваши AI-инструменты и настроит их.
*Хотите указать конкретную платформу?*
```bash
python3 scripts/install.py --platform cursor
python3 scripts/install.py --platform amp --global
# Или установить для всех:
python3 scripts/install.py --platform all --global
```

### Шаг 3: Увидьте магию
Откройте AI-инструмент (например, Cursor) и напишите:
> *"Создай лендинг для премиального спа-салона, современный минимализм."*

Если AI упомянет **дизайн-систему, UX-законы или дизайн-тесты** — MasterDesign Agent активирован! 🎉
