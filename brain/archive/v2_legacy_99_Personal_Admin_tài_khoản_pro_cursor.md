Lưu ý: Tính năng Tab (autocomplete) và mode `Auto` hoạt động không giới hạn, không tiêu tốn credit $20.

# Cơ chế tính phí Grok 4 trên Cursor

Chi phí được trừ trực tiếp vào credit hoặc thanh toán theo cơ chế pay-as-you-go. Công thức ước tính: `(Input × $0.003) + (Cached × $0.00075) + (Output × $0.015)`. Ví dụ thực tế: 8k input + 2k output ≈ `$0.054/request`.

# Chiến lược chọn Agent/Mode tối ưu hiệu suất lập trình trong Cursor
Việc chọn đúng mode trong Cursor giúp cân bằng giữa tốc độ, độ chính xác và chi phí credit:
- **Agent** (`Ctrl/Cmd + I`): Tự động đọc/sửa đa file, thực thi lệnh terminal, debug. Tối ưu cho task phức tạp, cần context rộng.
- **Ask/Chat**: Chỉ trả lời, giải thích, phân tích code. Không tự động sửa file. Phù hợp tra cứu nhanh, học hỏi.
- **Manual/Custom**: Kiểm soát từng bước thực thi, phù hợp workflow nghiêm ngặt, review code hoặc tích hợp pipeline đặc thù.
- **Tab/Auto**: Gợi ý code inline thời gian thực. Không giới hạn, không tính phí, tối ưu cho tốc độ gõ & hoàn thiện cú pháp.
Nguyên tắc vận hành: Dùng Agent cho tác vụ đa file/phức tạp, Ask cho tra cứu, Manual cho kiểm soát rủi ro, Tab cho viết code nhanh.
