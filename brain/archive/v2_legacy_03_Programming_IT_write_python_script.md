# So sánh lệnh tương tác với Script Python trong Terminal

Khi thao tác với file Python (`hello.py`) qua dòng lệnh, 3 lệnh phổ biến có cơ chế và mục đích hoàn toàn khác nhau:

- `cat hello.py` → **Đọc nội dung file**. Lệnh Unix/Linux chỉ in mã nguồn ra màn hình, không thực thi code.
- `python hello.py` → **Thực thi tường minh**. Gọi trực tiếp trình thông dịch Python để chạy script. Không yêu cầu quyền thực thi hay cấu hình đặc biệt, hoạt động ổn định trên mọi hệ điều hành.
- `./hello.py` → **Thực thi trực tiếp**. Hệ điều hành tự nhận diện và chạy file như một chương trình độc lập.
  - ⚠️ **Điều kiện bắt buộc:**
    1. Dòng đầu file phải chứa `shebang`: `#!/usr/bin/env python3`
    2. File phải được cấp quyền thực thi: `chmod +x hello.py`
  - 🌍 **Môi trường:** Chỉ hỗ trợ Unix-like (Linux, macOS). Windows (CMD/PowerShell) không nhận diện cú pháp `./` mặc định.

💡 **Bản chất khác biệt:** `cat` dùng để **xem mã nguồn** | `python` dùng để **chạy qua trình thông dịch** | `./` dùng để **chạy trực tiếp** (yêu cầu cấu hình quyền & shebang).
