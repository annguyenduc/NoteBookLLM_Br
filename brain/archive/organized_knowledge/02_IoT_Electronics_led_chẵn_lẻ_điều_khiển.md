# Điều khiển LED theo chẵn/lẻ & Cơ chế thời gian (Arduino)

## Nguyên lý cốt lõi
- **Phân loại chẵn/lẻ:** Sử dụng toán tử chia lấy dư (`% 2`) trên chỉ số vòng lặp (`i`). `i % 2 == 0` → chẵn, ngược lại → lẻ.
- **Trạng thái đối nghịch:** Dùng `if-else` hoặc toán tử ba ngôi để gán `HIGH` cho nhóm này và `LOW` cho nhóm kia, sau đó đảo ngược ở chu kỳ tiếp theo.

## Kiểm soát thứ tự thực thi
Vị trí đặt `delay()` quyết định hành vi vật lý của cụm LED:
- **Sáng đồng loạt:** Đặt `delay()` **ngoài** vòng `for`. MCU cập nhật trạng thái toàn bộ chân trong mảng ngay lập tức, sau đó mới chờ.
- **Sáng tuần tự:** Đặt `delay()` **bên trong** vòng `for`. Mỗi chân được xử lý xong sẽ chờ trước khi chuyển sang chân tiếp theo.

## Mẫu triển khai tối ưu
```cpp
const int ledPins[] = {2, 3, 4, 5, 6, 7, 8, 9};

void loop() {
  // Chu kỳ 1: Chẵn sáng, Lẻ tắt
  for (int i = 0; i < 8; i++) {
    digitalWrite(ledPins[i], (i % 2 == 0) ? HIGH : LOW);
    // delay(500); // Bỏ comment nếu muốn tuần tự, giữ ngoài vòng for nếu muốn đồng loạt
  }
  delay(500); // Chờ giữa 2 chu kỳ

  // Chu kỳ 2: Chẵn tắt, Lẻ sáng
  for (int i = 0; i < 8; i++) {
    digitalWrite(ledPins[i], (i % 2 == 0) ? LOW : HIGH);
    // delay(500); // Tương tự
  }
  delay(500);
}
```

## Lưu ý kỹ thuật
- Toán tử `%` kiểm tra **chỉ số mảng** (`0, 1, 2...`), không phải số chân vật lý. Nếu yêu cầu chẵn/lẻ theo số chân, cần kiểm tra trực tiếp giá trị phần tử mảng: `if (ledPins[i] % 2 == 0)`.
- `delay()` là hàm blocking. Với dự án phức tạp hoặc cần đọc cảm biến song song, nên chuyển sang kiến trúc non-blocking dùng `millis()`.
