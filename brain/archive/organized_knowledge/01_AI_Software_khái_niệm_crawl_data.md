# Crawl Data (Web Scraping)
**Định nghĩa:** Kỹ thuật tự động thu thập dữ liệu từ website thông qua các chương trình bot (crawler/spider).

**Quy trình chuẩn:**
1. **Xác định mục tiêu:** Chọn nguồn website và loại dữ liệu cần trích xuất (văn bản, giá, metadata...).
2. **Chọn công cụ:** Python là ngôn ngữ tối ưu. Dùng `requests` + `BeautifulSoup` cho trang tĩnh, `Scrapy` cho hệ thống lớn, `Selenium`/`Playwright` cho trang động (cần render JavaScript).
3. **Viết script & Phân tích:** Gửi HTTP request, phân tích cú pháp HTML/DOM, trích xuất phần tử mục tiêu.
4. **Kiểm thử & Lưu trữ:** Xử lý ngoại lệ, đảm bảo tính ổn định khi cấu trúc trang thay đổi, xuất dữ liệu sang định dạng có cấu trúc (CSV, JSON, Database).

**Nguyên tắc bắt buộc:**
- Kiểm tra `robots.txt` tại thư mục gốc website để tuân thủ quy tắc cho phép/cấm thu thập.
- Giới hạn tần suất request (rate limiting) để tránh gây quá tải server.
- Tuân thủ điều khoản sử dụng và quy định bảo mật dữ liệu.

**Ứng dụng chính:** Lập chỉ mục công cụ tìm kiếm, phân tích thị trường, theo dõi biến động giá và tin tức.
