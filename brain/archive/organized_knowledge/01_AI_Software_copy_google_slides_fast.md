# Sao chép Google Slides: Thủ công & Tự động hóa

## Nguyên lý kỹ thuật
- **Thủ công:** Dùng tính năng `File > Tạo bản sao` của giao diện. Bản sao mặc định thuộc quyền sở hữu tài khoản thực hiện, không kế thừa danh sách chia sẻ từ file gốc.
- **Tự động hàng loạt:** Google Slides không hỗ trợ native. Giải pháp chuẩn là dùng **Google Apps Script** kết hợp `DriveApp` để nhân bản trực tiếp từ URL/ID, đảm bảo tốc độ, kiểm soát thư mục đích và giữ nguyên chế độ `Riêng tư`.

## Hướng dẫn thực thi
### 1. Sao chép đơn lẻ (Đảm bảo không chia sẻ quyền)
1. Mở file gốc → `File` > `Tạo bản sao...`
2. Đổi tên file mới (tùy chọn).
3. Xác minh quyền: `File` > `Cài đặt chia sẻ` → Đảm bảo trạng thái `Riêng tư` (Chỉ bạn).

### 2. Sao chép hàng loạt qua URL (Google Apps Script)
**Bước 1:** Tạo script mới trong Google Drive (`Mới` > `Google Apps Script`).
**Bước 2:** Sử dụng mã tối ưu (thay thế phương pháp `SlidesApp` gốc để tránh lỗi định dạng):
```javascript
function duplicateSlidesBulk() {
  const slideUrls = ['URL_SLIDE_1', 'URL_SLIDE_2']; // Danh sách URL cần sao chép
  const destFolderId = 'ID_THU_MUC_LUU'; // ID thư mục đích (lấy từ URL Drive)
  const destFolder = DriveApp.getFolderById(destFolderId);
  
  slideUrls.forEach(url => {
    try {
      const fileId = url.match(/[-\w]{25,}/)[0]; // Trích xuất File ID từ URL
      const originalFile = DriveApp.getFileById(fileId);
      originalFile.makeCopy(`Copy_${originalFile.getName()}`, destFolder);
    } catch (e) {
      Logger.log(`Lỗi xử lý ${url}: ${e.message}`);
    }
  });
  Logger.log('Hoàn tất sao chép hàng loạt.');
}
```
**Bước 3:** Chạy hàm `duplicateSlidesBulk` → Cấp quyền truy cập `Drive` khi trình duyệt yêu cầu.

## Lưu ý vận hành
- **Bảo mật:** Bản sao tạo qua script mặc định là `Riêng tư`. Không cần cấu hình thêm để tránh rò rỉ quyền truy cập.
- **Giới hạn Quota:** Tài khoản Google cá nhân giới hạn ~20.000 lệnh gọi dịch vụ/ngày. Với danh sách >500 file, thêm `Utilities.sleep(1000)` trong vòng lặp để tránh trigger giới hạn.
- **Xử lý lỗi:** Mã đã tích hợp `try...catch` để bỏ qua URL hỏng hoặc file không tồn tại, đảm bảo script không dừng đột ngột.
