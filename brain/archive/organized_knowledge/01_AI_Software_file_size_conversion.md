# Kỹ thuật: Giải nén Tuple và Định dạng Dữ liệu trong Python

**Tóm tắt kỹ thuật:** Sử dụng cú pháp giải nén tuple (tuple unpacking) để trích xuất trực tiếp giá trị mục tiêu từ cấu trúc dữ liệu cố định, kết hợp phép toán chuyển đổi đơn vị và định dạng chuỗi để trả về kết quả số thực chuẩn xác.

**Mã nguồn minh họa:**
```python
def file_size(file_info):
    _, _, size = file_info
    return "{:.2f}".format(size / 1024)
```

**Nguyên lý cốt lõi:**
- **Giải nén có chọn lọc (`_, _, size = file_info`)**: Dấu `_` là quy ước chuẩn trong Python để bỏ qua giá trị không cần dùng. Kỹ thuật này thay thế truy cập bằng chỉ mục (`file_info[2]`), giúp code tự tài liệu hóa và tránh lỗi `IndexError`.
- **Chuyển đổi & Định dạng**: Phép chia `size / 1024` quy đổi bytes sang kilobytes. Phương thức `"{:.2f}".format()` đảm bảo đầu ra luôn là chuỗi số thực với đúng 2 chữ số thập phân, phù hợp cho hiển thị hoặc ghi log.
- **Ưu điểm của Tuple**: Tính bất biến (immutable) bảo vệ dữ liệu gốc khỏi thay đổi ngoài ý muốn. Tuple truy xuất nhanh hơn list, tiêu tốn ít bộ nhớ, và là cấu trúc lý tưởng để nhóm các thuộc tính cố định của một thực thể khi truyền vào hàm.

**Ứng dụng:** Xử lý dữ liệu đầu vào dạng nhóm cố định, chuyển đổi đơn vị đo lường, và chuẩn hóa định dạng đầu ra trong các script xử lý file hoặc pipeline dữ liệu cơ bản.
