## 📝 Content
## Cài đặt Python và cấu hình PATH
- Tải trình cài đặt chính thức từ `python.org/downloads`.
- Trong bước cài đặt, bắt buộc tích chọn `Add Python to PATH` để hệ điều hành nhận diện lệnh `python` từ terminal/command prompt mà không cần khai báo đường dẫn tuyệt đối.

## Viết chương trình Python cơ bản
- Tạo tệp mới với phần mở rộng `.py` (ví dụ: `hello.py`).
- Sử dụng hàm `print()` để xuất chuỗi hoặc biến ra màn hình console.
- Luôn lưu tệp (`Ctrl+S` / `Cmd+S`) trước khi thực thi để đảm bảo mã nguồn được cập nhật.

## Chạy script Python qua dòng lệnh
- Mở Terminal (macOS/Linux) hoặc Command Prompt/PowerShell (Windows).
- Dùng lệnh `cd đường_dẫn` để di chuyển đến thư mục chứa tệp `.py`.
- Thực thi chương trình bằng lệnh: `python ten_tep.py`.

## Chạy script Python trong VS Code
- Mở tệp `.py` trực tiếp trong VS Code.
- Nhấn `F5` hoặc chuột phải vào vùng soạn thảo, chọn `Run Python File`.
- Kết quả thực thi được hiển thị trực tiếp trong panel Terminal tích hợp của IDE.

## Mở và thao tác với IDLE
- **Windows:** Gõ `IDLE` vào thanh tìm kiếm Start Menu, chọn phiên bản tương ứng.
- **macOS:** Mở Spotlight (`Cmd + Space`), gõ `IDLE` và nhấn Enter.
- **Linux:** Gõ `idle` trong Terminal. Nếu báo lỗi, cài đặt bằng `sudo apt-get install idle3`.
- **Giao diện:** `Python Shell` cho phép gõ và chạy lệnh tức thì. `File > New File` mở trình soạn thảo script riêng, chạy toàn bộ file bằng `F5`.

## Nhập liệu và xử lý số trong Python
- Hàm `input()` luôn trả về dữ liệu kiểu chuỗi (`str`).
- Dùng `int()` hoặc `float()` để ép kiểu sang số trước khi thực hiện phép toán.
- Quy trình chuẩn: nhận input → ép kiểu → gán biến → tính toán → `print()` kết quả.
