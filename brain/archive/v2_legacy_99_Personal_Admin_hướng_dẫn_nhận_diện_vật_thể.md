## 📝 Content
# Hướng dẫn nhanh: Xây dựng hệ thống nhận diện vật thể cơ bản

## 🎯 Tiêu chí lựa chọn công cụ
- **Phân loại ảnh (Classification):** Chỉ cần xác định ảnh thuộc nhóm nào → Dùng **Teachable Machine**.
- **Phát hiện vị trí (Detection):** Cần tọa độ & khung bao quanh vật thể → Dùng **OpenCV + mô hình sẵn**.

## 🛠 Lộ trình 1: Teachable Machine (Image Classification)
- **Phạm vi:** 2–5 lớp vật thể, chạy web/mobile, không yêu cầu bounding box.
- **Quy trình triển khai:**
  1. Chuẩn bị dữ liệu: ≥100–300 ảnh/lớp, đa dạng góc chụp/ánh sáng/nền.
  2. Huấn luyện: Upload hoặc chụp trực tiếp qua webcam → Train Model (giữ tham số mặc định).
  3. Xuất model: Chọn `TensorFlow.js` (web) hoặc `TensorFlow Lite` (Android/iOS/RPi).
  4. Tích hợp: Nhúng model vào HTML/JS qua thư viện `@teachablemachine/image`, gọi `model.predict()` trong vòng lặp `requestAnimationFrame` để nhận diện real-time.
- **Ưu/Nhược điểm:** Triển khai cực nhanh, không cần kiến thức ML. Hạn chế: chỉ phân loại toàn bộ khung hình, không trả về tọa độ vật thể.

## 💻 Lộ trình 2: OpenCV + Mô hình phát hiện sẵn (MobileNet-SSD)
- **Phạm vi:** Phát hiện vị trí real-time, xử lý bằng Python, yêu cầu bounding box.
- **Quy trình triển khai:**
  1. Cài đặt môi trường: `pip install opencv-python numpy`.
  2. Tải model: Cần cặp file `.prototxt` (kiến trúc mạng) và `.caffemodel`/`.onnx` (trọng số), ví dụ `MobileNetSSD_deploy`.
  3. Xử lý pipeline: Load model qua `cv2.dnn.readNet()` → Đọc frame camera → Tiền xử lý ảnh (`cv2.dnn.blobFromImage`) → Chạy `net.forward()` → Lọc kết quả theo ngưỡng confidence → Vẽ khung & nhãn bằng `cv2.rectangle()` & `cv2.putText()`.
- **Ưu/Nhược điểm:** Linh hoạt, chạy offline, hỗ trợ detection chính xác. Yêu cầu hiểu biết cơ bản về Python và pipeline xử lý ảnh.

## 💡 Gợi ý triển khai
- Bắt đầu với **Teachable Machine** để kiểm chứng ý tưởng hoặc xây dựng prototype nhanh.
- Nâng cấp lên **OpenCV + SSD/YOLO** khi cần độ chính xác vị trí, xử lý video liên tục hoặc tích hợp vào hệ thống Python phức tạp.
