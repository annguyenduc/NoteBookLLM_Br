# Viết hoa chữ cái đầu từ trong Google Sheets (Xử lý tại chỗ)

## Bối cảnh & Hạn chế
- Google Sheets không hỗ trợ hàm mặc định cho "Sentence case" (chỉ viết hoa chữ đầu câu).
- Hàm `=PROPER()` viết hoa chữ cái đầu của **mọi từ**, không phân biệt cấu trúc câu hay từ nối.

## Phương pháp 1: Thủ công (Không tạo ô phụ)
1. Nhập công thức tạm `=PROPER(Ô_gốc)` vào một ô bất kỳ.
2. Copy kết quả → Chuột phải ô gốc → **Dán đặc biệt** → **Chỉ dán giá trị** (`Ctrl+Shift+V` / `Cmd+Shift+V`).
3. Xóa ô công thức tạm.

## Phương pháp 2: Tự động hóa bằng Apps Script
Script cho phép ghi đè trực tiếp lên vùng chọn, loại bỏ bước trung gian và phù hợp xử lý hàng loạt:

```javascript
function capitalizeInPlace() {
  const range = SpreadsheetApp.getActiveRange();
  const values = range.getValues();
  const updated = values.map(row => 
    row.map(cell => typeof cell === 'string' ? cell.replace(/\b[a-z]/g, c => c.toUpperCase()) : cell)
  );
  range.setValues(updated);
}
```

**Triển khai:**
1. `Tiện ích mở rộng > Apps Script` → Dán code → Lưu dự án.
2. Quay lại Sheet, chọn vùng dữ liệu cần xử lý.
3. `Tiện ích mở rộng > Apps Script` → Chọn hàm `capitalizeInPlace` → **Run** (cấp quyền nếu được yêu cầu).

**Lưu ý kỹ thuật:**
- Regex `/\b[a-z]/g` chỉ khớp chữ cái thường đứng ngay sau ranh giới từ (`\b`), đảm bảo không làm biến đổi số, ký tự đặc biệt hoặc viết hoa sai vị trí (khắc phục lỗi xử lý thừa ký tự cuối nếu dùng regex không chuẩn).
- Hàm hoạt động trực tiếp trên `getActiveRange()`, không tạo ô/cột phụ, giữ nguyên cấu trúc bảng tính.
