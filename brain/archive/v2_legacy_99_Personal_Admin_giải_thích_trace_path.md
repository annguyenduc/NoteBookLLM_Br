# MakeCode Builder API: Trace Path & Giới hạn đặt Block State

## `builder.tracePath(block)`
- **Mục đích:** Tự động đặt khối dọc theo lộ trình di chuyển của Builder, bắt đầu từ lần gọi `builder.mark()` gần nhất.
- **Cơ chế:** Truy vết mọi lệnh thay đổi vị trí (`teleportTo`, `move`, `shift`) và ghi đè khối tại các tọa độ đó bằng `block` chỉ định.
- **Quy trình chuẩn:** 
  1. `builder.mark()` → đánh dấu điểm xuất phát
  2. Thực thi chuỗi di chuyển
  3. `builder.tracePath(LOẠI_KHỐI)` → vẽ đường
  4. Gọi lại `mark()` nếu cần truy vết đoạn tiếp theo.
- **Lưu ý:** Khối được đặt sẽ ghi đè lên vật thể có sẵn trên đường đi.

## Giới hạn API khi đặt Hopper & Chest
- **Vấn đề:** Thư viện `builder`/`blocks` không hỗ trợ thiết lập `block state` (ví dụ: `facing` của hopper) hay mô phỏng thao tác `sneak`. Do đó, không thể đảm bảo hopper gắn ngang vào cạnh chest bằng code thuần.
- **Giải pháp khả thi (Kết nối dọc):** Đặt chest ở tọa độ thấp hơn, hopper ngay phía trên. Hopper mặc định hướng xuống sẽ tự động kết nối với chest.
  ```javascript
  const base = player.position()
  blocks.place(TRAPPED_CHEST, positions.add(base, pos(0, -1, 0)))
  blocks.place(HOPPER, base)
  ```
- **Kết nối ngang:** Chỉ khả thi khi dùng lệnh gốc `/setblock` kèm tham số `hopper[facing=...]` hoặc điều chỉnh thiết kế hệ thống sang cơ chế dẫn hướng từ trên xuống.
