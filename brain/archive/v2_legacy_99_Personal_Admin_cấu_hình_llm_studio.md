# LLM Studio: Cấu hình & Hướng dẫn sử dụng

**Định nghĩa:** Công cụ hỗ trợ xây dựng, huấn luyện và tinh chỉnh (fine-tune) mô hình ngôn ngữ lớn (LLM) cho các tác vụ NLP (QA, phân loại, tóm tắt, sinh nội dung).

**Yêu cầu phần cứng:**
- **CPU:** Đa lõi (Intel i9/AMD Ryzen 9)
- **GPU:** Tối thiểu RTX 3060 (12GB VRAM) | Khuyến nghị RTX 3090/A100 (24GB+ VRAM)
- **RAM:** Tối thiểu 16GB | Khuyến nghị 32–64GB
- **Lưu trữ:** SSD ≥500GB (ưu tiên NVMe)
- **HĐH:** Ubuntu 20.04/22.04 hoặc Windows 11 (WSL2)

**Môi trường & Cài đặt:**
- Python ≥3.8
- Thư viện cốt lõi: `torch`, `transformers`, `datasets` (PyTorch/TensorFlow tùy framework)
- GPU: Cài driver NVIDIA + CUDA/cuDNN tương thích
- Lệnh triển khai:
  ```bash
  pip install torch torchvision transformers datasets
  git clone <repo-url> && cd llm-studio && pip install -r requirements.txt
  python app.py
  ```

**Tính năng chính:**
- Giao diện trực quan tinh chỉnh siêu tham số (learning rate, batch size, epochs)
- Hỗ trợ đa ngôn ngữ & tích hợp mô hình Hugging Face (GPT, BERT, BLOOM)
- Theo dõi huấn luyện thời gian thực (log, biểu đồ)

**Lưu ý vận hành:**
- Ưu tiên GPU mạnh để giảm thời gian huấn luyện
- Làm sạch dữ liệu đầu vào & khởi tạo từ mô hình pre-trained
- Giám sát hiệu năng bằng TensorBoard hoặc MLflow
- *Khả năng tương thích laptop (6 nhân/12 luồng):* Phù hợp inference hoặc fine-tune nhẹ. Cần xác định RAM/VRAM cụ thể để đánh giá khả năng chạy tác vụ nặng.
