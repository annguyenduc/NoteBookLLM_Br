# [QUESTION] Câu 22 (Nhận biết)

## ❓ Câu hỏi
Để thêm phiên bản rút gọn cho bài viết, trang ta thực hiện như thế nào?

## 📝 Đáp án lựa chọn
- A/ Edit bài viết, thêm block “Heading 1” và gõ nội dung rút gọn
- **B/ (Đáp án) Edit bài viết, trong phần setting chọn Excerpt rồi gõ nội dung rút gọn (Đáp án)**
- C/ Edit bài viết, thêm block “Quote” và gõ nội dung rút gọn
- D/ Edit bài viết, thêm block “Post Excerpt” và gõ nội dung rút gọn
