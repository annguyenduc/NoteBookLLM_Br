# 📊 Ma trận Năng lực Chuyên môn Giáo viên (Teacher Subject Matter Competency Matrix)

> **Mục tiêu**: Thiết lập khung tham chiếu kỹ thuật để đánh giá năng lực giáo viên trong từng Module riêng biệt.
> **Tiêu chuẩn**: UNESCO ICT CFT v3 & DNA Sư phạm Antigravity.
> **Nền tảng chủ đạo**: Lập trình khối lệnh (Block-based) như **mBlock, Ohstem** và lập trình văn bản (**Python/C++**).

---

## 1. Module AI (Trí tuệ Nhân tạo)

Các môn học: AI Tiểu học, AI THCS, Python (mBlock).

| Cấp độ | Tiêu chí Chuyên môn (Subject Matter Mastery) |
| :--- | :--- |
| **L1: Literacy** | Hiểu khái niệm cơ bản về AI. Nhận diện các khối lệnh nhận diện khuôn mặt, giọng nói trong mBlock/Ohstem. |
| **L2: Deepening** | Làm chủ quy trình huấn luyện mô hình (Teachable Machine/ML4Kids) và nhúng vào Blockly để xử lý dữ liệu. |
| **L3: Creation** | **Chuyên gia**: Tối ưu logic rẽ nhánh phức tạp. Đối với **Python**: Khả năng nhúng thư viện AI, xử lý dữ liệu thô và tối ưu hóa mô hình thực tế. |

## 2. Module Coding (Lập trình Khối lệnh & Logic)

Các môn học: Scratch Jr, Scratch 1+2, Tynker, Python (mBlock).

| Cấp độ | Tiêu chí Chuyên môn (Subject Matter Mastery) |
| :--- | :--- |
| **L1: Literacy** | Sử dụng thành thạo các khối lệnh cơ bản (Xanh, Tím, Vàng). Hiểu logic tuần tự và vòng lặp đơn giản. |
| **L2: Deepening** | Làm chủ Biến (Variables), Danh sách (Lists) và Hàm tùy chỉnh (My Blocks) trong môi trường Blockly. |
| **L3: Creation** | **Chuyên gia**: Xây dựng thuật toán phức tạp. Đối với **Python**: Làm chủ cấu trúc dữ liệu nâng cao (Dict, Set), Class/Object và quản lý Library. |

## 3. Module Robotics (Robot & Điều khiển)

Các môn học: Codey Rocky, mBot, Rover, GBot.

| Cấp độ | Tiêu chí Chuyên môn (Subject Matter Mastery) |
| :--- | :--- |
| **L1: Literacy** | Kết nối thiết bị qua Bluetooth/USB. Sử dụng các khối lệnh "Di chuyển", "Hiển thị" cơ bản trên mBlock/Ohstem. |
| **L2: Deepening** | Xử lý logic cảm biến (Dò đường, Siêu âm) bằng các khối lệnh điều kiện phức tạp. Hiểu logic ngắt (Events). |
| **L3: Creation** | **Chuyên gia**: Tư duy kiến trúc hệ thống Robot tự hành. **An toàn thiết bị**: Kiểm soát pin, dòng điện qua thông số hiển thị trên phần mềm. |

## 4. Module IoT & Automation (Tự động hóa)

Các môn học: Arduino (Ohstem/mBlock), Halocode, YoloBit.

| Cấp độ | Tiêu chí Chuyên môn (Subject Matter Mastery) |
| :--- | :--- |
| **L1: Literacy** | Hiểu sơ đồ cắm dây Grove. Sử dụng được các khối lệnh I/O cơ bản (Bật/tắt đèn, đọc nút nhấn). |
| **L2: Deepening** | Sử dụng tốt các khối lệnh giao tiếp I2C cho màn hình LCD, Servo. Xử lý logic Dashboard IoT trên App/Web Ohstem. |
| **L3: Creation** | **Chuyên gia**: **Tùy biến & Sáng tạo** - Thiết kế logic phức tạp kết nối nhiều thiết bị (Cloud IoT). Đảm bảo 100% **An toàn thiết bị** (không cắm ngược cực). |

---

## 🛠️ Yêu cầu chung cho Cấp độ Chuyên gia (Level 3)

1. **Thành thạo Platform**: Giáo viên phải làm chủ các ứng dụng mBlock5, Ohstem App hoặc Scratch, bao gồm cả việc quản lý Extension.
2. **An toàn thiết bị**: GV phải nắm vững các chân VCC, GND của chuẩn Grove/Jumper, tuyệt đối không cắm ngược cực.
3. **Kết nối**: Ưu tiên cắm dây Jumper wires hoặc Grove. **Không sử dụng hàn mạch**.
4. **Troubleshooting**: Phân biệt được lỗi do phần cứng (lỏng dây) hay do chọn sai Cổng (Port) hoặc Device Type trên phần mềm.
5. **Logic Master**: Có khả năng giải thích cấu trúc mã nguồn (C++/Python) đằng sau các khối lệnh (Code mapping) cho giáo viên khác.
