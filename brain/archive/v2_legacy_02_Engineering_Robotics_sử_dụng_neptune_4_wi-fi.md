## 📝 Content
Sử dụng Neptune 4 với Wi-Fi
Neptune 4 bản gốc không có module Wi-Fi tích hợp sẵn. Tuy nhiên, bạn vẫn có thể dùng Wi-Fi để in theo một số cách:

1. Sử dụng thẻ nhớ / USB gắn Wi-Fi
- Một số người dùng mua USB Wi-Fi SD Card (ví dụ: FlashAir, Eye-Fi) → chép file `.gcode` từ máy tính/điện thoại qua Wi-Fi, sau đó máy in đọc trực tiếp từ thẻ.
- Ưu điểm: không cần cắm tháo thẻ nhiều lần.
- Nhược điểm: cấu hình phức tạp, tốc độ chép chậm, ít hãng còn sản xuất.

2. Dùng OctoPrint (hoặc Klipper + Fluidd/Mainsail)
- Bạn cần 1 Raspberry Pi (hoặc máy mini cũ), cài OctoPrint/Fluidd.
- Kết nối Pi với Neptune 4 bằng cáp USB-C.
- Sau đó in qua Wi-Fi bằng giao diện web (trên PC hoặc điện thoại).
- Ưu điểm: có thể điều khiển, xem nhiệt độ, quản lý nhiều file từ xa.
- Nhược điểm: tốn thêm chi phí phần cứng.

3. Dùng Neptune 4 Pro/Plus/Max + module Wi-Fi
- Một số bản Neptune 4 cao hơn có hỗ trợ cắm thẻ nhớ kèm Wi-Fi (hoặc mainboard hỗ trợ mở rộng).
- Kiểm tra menu “Network” trong máy, nếu không có → bản bạn dùng là bản thường, không hỗ trợ trực tiếp.

4. Chia sẻ qua LAN bằng PC
- Bạn cắm Neptune 4 vào PC qua USB-C.
- Sau đó dùng phần mềm như Pronterface hoặc Cura → bật chế độ in qua mạng LAN (chia sẻ bằng remote desktop hoặc phần mềm trung gian).
- Thực chất máy vẫn in qua USB, nhưng bạn điều khiển PC từ xa.

Nếu bạn muốn tiện nhất thì nên chọn OctoPrint + Raspberry Pi (hoặc Klipper + Fluidd) để in qua Wi-Fi. Đây là cách phổ biến nhất cho Neptune 4.

Hướng dẫn cài OctoPrint cho Neptune 4
Chuẩn bị phần cứng
1. Raspberry Pi (khuyên dùng Pi 3B/3B+/4B; Pi Zero 2 W chỉ phù hợp in chậm/nhẹ).
2. Nguồn Pi chính hãng (5V–3A cho Pi 4).
3. Thẻ microSD 16–32GB (U1/U3).
4. Cáp USB tốt (USB-C ↔ USB-A) để nối Pi ↔ Neptune 4.
5. (Tuỳ chọn) Webcam USB hoặc camera Pi nếu muốn livestream Time-lapse.

Flash OctoPi vào thẻ nhớ
1. Tải Raspberry Pi Imager (Windows/macOS/Linux).
2. Mở Imager → Choose OS → Other specific purpose OS → OctoPi (chọn bản mới nhất).
3. Choose Storage → chọn thẻ microSD.
4. Bấm bánh răng ⚙️ (Advanced):
   - Set hostname: `octopi` (mặc định).
   - Enable SSH + đặt user/pass (VD: user `pi`, pass mạnh).
   - Configure Wi-Fi: nhập SSID & password (chú ý đúng hoa/thường), Wi-Fi country → chọn `VN`.
   - Set Locale/Timezone: `Asia/Ho_Chi_Minh`.
5. Write → chờ hoàn tất → rút thẻ, cắm vào Pi.

Kết nối phần cứng & boot lần đầu
1. Cắm nguồn cho Pi → đợi ~2–3 phút cho Pi khởi động.
2. Dùng điện thoại/PC cùng mạng, truy cập:
   - http://octopi.local (máy hỗ trợ Bonjour/mDNS), hoặc
   - Tìm IP trong modem/router rồi mở http://<IP-của-Pi>.
3. Kết nối cáp USB-C từ Pi sang cổng USB của Neptune 4 (bật máy in).
