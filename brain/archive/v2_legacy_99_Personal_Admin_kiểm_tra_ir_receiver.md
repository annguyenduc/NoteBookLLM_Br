# Kiểm tra & Khắc phục lỗi IR Receiver trên Yolo:Bit

## 🔌 Kết nối phần cứng
- **Sơ đồ chân:** `OUT` → `P0/P1/P2` (Yolo:Bit), `VCC` → `3.3V`, `GND` → `GND`.
- ⚠️ **Lưu ý:** Tuyệt đối không cấp `5V` (Yolo:Bit chỉ tương thích logic 3.3V).
- ✅ **Kiểm tra nhanh:** Đèn LED trên module phải nhấp nháy khi remote hướng vào. Không nhấp nháy → sai dây hoặc hỏng phần cứng.

## 💻 Kiểm tra bằng Code
### Khối lệnh (app.ohstem.vn)
- Thêm extension `IR Remote`.
- Logic: `mãi mãi` → `nếu IR nhận tín hiệu` → hiển thị ký tự, `ngược lại` → hiển thị `-`.
- Kết quả: Ký tự thay đổi theo nút bấm = OK; Luôn hiển thị `-` = Lỗi IR/Remote/Code.

### MicroPython
```python
from yolobit import *
import ir_rx
from ir_rx import NEC_8

def callback(data, addr, ctrl):
    print('Nhận:', data, 'từ remote')

ir = NEC_8(P0, callback)  # Thay P0 nếu cần
while True:
    pass
```
- Mở Serial Monitor, bấm remote. Nếu console in ra dữ liệu → IR hoạt động bình thường.

## 📡 Xử lý xung đột khi bật IoT (Wi-Fi/MQTT)
Khi kích hoạt IoT, IR thường mất tín hiệu do xung đột Timer/Interrupt hoặc nhiễu nguồn từ module Wi-Fi.
- **Đổi chân GPIO:** Tránh các chân `TX/RX` (UART), `GPIO0/2/15` (chế độ boot), hoặc chân đang chạy PWM/I²C. Ưu tiên chuyển sang `P1`, `P2` hoặc các chân digital rảnh.
- **Lọc nhiễu nguồn:** Hàn thêm tụ `10–47µF` song song `VCC–GND` ngay trên module IR để ổn định điện áp khi Wi-Fi phát sóng.
- **Đồng bộ giao thức:** Đảm bảo remote dùng chuẩn `NEC` (mặc định của `ir_rx`). Nếu dùng remote khác, cần đổi thư viện giải mã (`RC5`, `SIRC`, v.v.).

## 📋 Bảng chẩn đoán nhanh
| Hiện tượng | Nguyên nhân | Khắc phục |
|---|---|---|
| LED IR không sáng/nhấp nháy | Sai nguồn, đứt dây, hỏng module | Đo lại 3.3V, kiểm tra dây, thay module |
| LED nhấp nháy nhưng không nhận data | Sai chân OUT, code chưa đúng | Đổi chân P0→P1/P2, kiểm tra extension/thư viện |
| Remote không phản hồi | Hết pin, sai chuẩn mã | Thay pin, dùng remote TV chuẩn NEC |
| Chỉ nhận 1 lần rồi đứng | Thiếu vòng lặp, xung đột interrupt | Đảm bảo `while True: pass`, ưu tiên chân không dùng cho Wi-Fi |
