# Kiểm tra dung lượng file & Xuất danh sách ra Excel bằng Python

Sử dụng Python kết hợp `os` và `pandas` để quét đệ quy dung lượng toàn bộ file trong thư mục, xử lý ngoại lệ truy cập, sắp xếp giảm dần và xuất kết quả định dạng Excel.

## Mã nguồn thực thi
```python
import os
import pandas as pd

def get_files_sizes(folder_path):
    file_data = []
    for dirpath, _, filenames in os.walk(folder_path):
        for file in filenames:
            file_path = os.path.join(dirpath, file)
            try:
                size = os.path.getsize(file_path)
                file_data.append({"File Path": file_path, "Size (MB)": size / (1024 ** 2)})
            except OSError:
                pass  # Bỏ qua file bị khóa hoặc không có quyền truy cập
    return file_data

def export_to_excel(file_data, output_path):
    df = pd.DataFrame(file_data)
    df.sort_values(by="Size (MB)", ascending=False, inplace=True)
    df.to_excel(output_path, index=False)
    print(f"Đã xuất dữ liệu ra {output_path}")

# Cấu hình đường dẫn
folder_path = r"C:\Users\anngu\AppData\Local"
output_excel = r"C:\Users\anngu\Desktop\folder_sizes.xlsx"

data = get_files_sizes(folder_path)
export_to_excel(data, output_excel)
```

## Lưu ý kỹ thuật
- `os.walk()` duyệt toàn bộ cây thư mục con.
- `try...except OSError` đảm bảo script không dừng khi gặp file hệ thống bị khóa hoặc thiếu quyền.
- Kết quả tự động sắp xếp từ lớn → nhỏ, giúp nhanh chóng xác định file chiếm dung lượng.
- Đường dẫn nên dùng raw string (`r"..."`) để tránh lỗi escape ký tự `\` trên Windows.
