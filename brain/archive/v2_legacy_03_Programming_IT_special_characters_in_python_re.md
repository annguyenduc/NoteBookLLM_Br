# Ký tự đặc biệt & Định lượng trong Regex Python (`re`)

Biểu thức chính quy (Regex) trong Python dùng các ký tự đặc biệt để định nghĩa mẫu khớp chuỗi. Khi kết hợp với toán tử định lượng, chúng cho phép kiểm soát chính xác cấu trúc và số lần lặp của dữ liệu.

## Bảng ký tự đặc biệt cốt lõi
| Ký tự | Chức năng | Ví dụ |
|---|---|---|
| `.` | Khớp bất kỳ ký tự nào (trừ `\n`) | `a.c` → `abc`, `a c` |
| `^` / `$` | Khớp vị trí đầu / cuối chuỗi | `^A` / `z$` |
| `*` / `+` / `?` | Định lượng: 0-nhiều, 1-nhiều, 0-1 lần | `ab*` → `a`, `ab`, `abb` |
| `[]` / `[^]` | Tập ký tự được phép / bị loại trừ | `[abc]` / `[^0-9]` |
| `\|` | Toán tử HOẶC | `cat\|dog` |
| `()` | Nhóm biểu thức & lưu kết quả khớp | `(ab)+` |
| `\` | Ký tự thoát hoặc mở đầu lớp đặc biệt | `\.` , `\d` |

## Phân tích mẫu: `\d{3}-\d{2}-\d{4}`
Mẫu này khớp định dạng số an sinh xã hội Mỹ (SSN). Cấu trúc được bóc tách tuần tự:
1. `\d` : Lớp ký tự đại diện cho chữ số `[0-9]`.
2. `{3}` : Định lượng chính xác, yêu cầu `\d` lặp đúng 3 lần → khớp 3 chữ số liên tiếp.
3. `-` : Ký tự thường, khớp chính xác dấu gạch ngang.
4. `\d{2}` : Kết hợp lớp chữ số và định lượng chính xác 2 lần → khớp 2 chữ số.
5. `-` : Dấu gạch ngang thứ hai.
6. `\d{4}` : Khớp chính xác 4 chữ số liên tiếp.

**Kết quả khớp:** Chuỗi dạng `XXX-XX-XXXX` (ví dụ: `123-45-6789`).

## Ghi chú kỹ thuật
- Luôn dùng raw string (`r"pattern"`) trong Python để tránh xung đột ký tự thoát (`\`).
- `re.search()` quét toàn bộ chuỗi và trả về đối tượng `Match` đầu tiên tìm thấy. Dùng `match.group()` để trích xuất chuỗi khớp.
- Các lớp ký tự viết tắt: `\d` (số), `\w` (chữ/số/gạch dưới), `\s` (khoảng trắng). Dạng in hoa (`\D`, `\W`, `\S`) đảo ngược nghĩa.
