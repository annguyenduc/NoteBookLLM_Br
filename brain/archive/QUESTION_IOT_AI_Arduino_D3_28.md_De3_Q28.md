# ❓ Câu hỏi
Hãy sắp xếp thứ tự các bước để sử dụng Teachable Machine dạy máy tính nhận biết vật thể:
- A. Chọn "Use the model"
- B. Chọn "Build a new model" và nhập số lượng vật thể.
- C. Cung cấp hình ảnh từ nhiều góc độ khác nhau.
- D. Thêm tiện ích mở rộng "Teachable Machine".
- E. Chọn "Training model".
- F. Đặt tên và nhấn "Learn" để bắt đầu dạy.

## 📝 Đáp án lựa chọn
- **A/ D - B - C - E - F - A (Đáp án)**
- B/ F - E - D - B - C
- C/ D - A - C - B
- D/ D - A - B - C

## 💡 Giải thích
Quy trình chuẩn bao gồm: Kích hoạt extension -> Thiết lập model -> Thu thập dữ liệu -> Huấn luyện -> Gán nhãn/Học -> Sử dụng kết quả.
