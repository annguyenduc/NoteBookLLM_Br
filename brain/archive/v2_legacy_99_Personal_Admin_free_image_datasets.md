**Tags:** #MachineLearning #Dataset #Image #Free

**Nguồn dữ liệu hình ảnh miễn phí cho Machine Learning:**

* **ImageNet:**
    * Mô tả: Cơ sở dữ liệu hình ảnh lớn, chia thành nhiều danh mục.
    * Link: http://www.image-net.org/
* **COCO (Common Objects in Context):**
    * Mô tả: Dữ liệu đa dạng với nhiều đối tượng và ngữ cảnh.
    * Link: http://cocodataset.org/
* **Open Images Dataset:**
    * Mô tả: Hình ảnh từ nhiều nguồn khác nhau.
    * Link: https://storage.googleapis.com/openimages/web/index.html
* **Flickr30k:**
    * Mô tả: Khoảng 30,000 hình ảnh từ Flickr.
    * Link: http://shannon.cs.illinois.edu/DenotationGraph/
* **Stanford Dogs Dataset:**
    * Mô tả: Hình ảnh của nhiều loại chó.
    * Link: http://vision.stanford.edu/aditya86/ImageNetDogs/
* **MNIST:**
    * Mô tả: Dữ liệu chữ số viết tay, phổ biến trong Machine Learning.
    * Link: http://yann.lecun.com/exdb/mnist/
* **Fashion MNIST:**
    * Mô tả: Tương tự MNIST, nhưng với ảnh thời trang.
    * Link: https://github.com/zalandoresearch/fashion-mnist
* **Kaggle Datasets:**
    * Mô tả: Cộng đồng trực tuyến với nhiều bộ dữ liệu khác nhau.
    * Link: https://www.kaggle.com/datasets

**Lưu ý:** Kiểm tra điều khoản sử dụng của từng bộ dữ liệu.

**Nguồn dữ liệu hình ảnh miễn phí cho Teachable Machine:**

* **Unsplash:**
    * Mô tả: Bộ sưu tập hình ảnh chất lượng cao và miễn phí.
    * Link: https://unsplash.com/
* **Pexels:**
    * Mô tả: Nguồn ảnh miễn phí với nhiều chủ đề.
    * Link: https://www.pexels.com/
* **Pixabay:**
    * Mô tả: Hình ảnh, video và vector miễn phí.
    * Link: https://pixabay.com/
* **Freepik:**
    * Mô tả: Đồ họa vector và ảnh miễn phí (có thể yêu cầu ghi chú tác giả).
    * Link: https://www.freepik.com/
* **Burst by Shopify:**
    * Mô tả: Hình ảnh chất lượng cao cho doanh nghiệp và tiếp thị.
    * Link: https://burst.shopify.com/
* **Wikimedia Commons:**
    * Mô tả: Kho tàng phương tiện miễn phí do cộng đồng đóng góp.
