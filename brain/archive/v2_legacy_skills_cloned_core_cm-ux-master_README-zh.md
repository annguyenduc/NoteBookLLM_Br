## 🛑 真正的问题

你有一个绝妙的想法。你把它描述给 AI 工具。但 AI 生成的界面……**丑得不忍直视**。
看起来千篇一律，"AI味"十足，没有统一的设计系统。表单太长、按钮太小、用户体验一塌糊涂。

这是 90% 创始人、开发者和产品经理面临的残酷现实。再好的代码，界面丑陋等于白做。

**MasterDesign Agent** 把你的任何 AI 编码工具变成一个**专业设计工作室**。

## 🚀 v2.0 新功能

我们对工具包进行了全面的重磅升级：

- 🧬 **Harvester v3（全面提取）：** 从任意网站提取50-80+设计令牌。捕获色彩直方图、语义色彩、灰度梯度、字体比例、间距和布局参数。5分钟内克隆顶级网站的设计DNA。
- 🗂️ **项目注册表（Project Registry）：** 轻松管理多个设计系统项目。将多个页面的采集数据合并为一个统一项目。
- 📖 **设计文档生成器：** 自动生成精美的HTML设计文档（色板、字体样本、组件示例）。
- 🔗 **Token Mapper & Semi MCP Bridge：** 将设计令牌直接映射到 Semi Design UI 变量。自动生成CSS覆盖、Figma JSON 和 React 组件模板。
- 🧪 **7套完整测试：** 全面的设计TDD（测试驱动设计）覆盖。

## ⚡ 60秒快速上手

### 第一步：检查 Python
确保已安装 Python 3：
```bash
python3 --version
```

### 第二步：克隆 & 安装
```bash
git clone https://github.com/relukdev/ux-master.git
cd ux-master
python3 scripts/install.py
```

安装程序会自动检测你的AI工具并立即配置。
*想指定特定平台？*
```bash
python3 scripts/install.py --platform cursor
python3 scripts/install.py --platform amp --global
# 或者全部安装：
python3 scripts/install.py --platform all --global
```

### 第三步：见证魔法
打开你的AI工具（如Cursor），输入提示：
> *"创建一个高端水疗中心的着陆页，现代简约风格。"*

如果 AI 提到了 **设计系统、UX法则或设计测试** — 恭喜，MasterDesign Agent 已经激活了！🎉
