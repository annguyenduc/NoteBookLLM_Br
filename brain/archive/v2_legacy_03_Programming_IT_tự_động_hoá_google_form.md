# Tự động điền Google Form bằng Apps Script

## Nguyên tắc & Ràng buộc
- **Điều kiện tiên quyết:** Chỉ khả thi và an toàn khi bạn là **chủ sở hữu (owner) hoặc biên tập viên (editor)** của form.
- **Giới hạn API chính thức:** Google Forms API chỉ hỗ trợ *đọc* (get/list) phản hồi, không hỗ trợ *gửi* (submit) dữ liệu.
- **Rủi ro phương pháp không chính thức:** Dùng headless browser/bot (Selenium, Puppeteer...) cho form của người khác dễ bị chặn CAPTCHA, vi phạm chính sách chống spam và thiếu ổn định. Không khuyến khích.
- **Hạn mức hệ thống:** Apps Script chịu giới hạn quota (thời gian thực thi, số lệnh gọi/ngày). Cần tối ưu batch nếu xử lý dữ liệu lớn.

## Phương pháp triển khai (Apps Script + Google Sheet)
Sử dụng `FormApp` và `FormResponse.submit()` để đọc dữ liệu từ Sheet và gửi phản hồi tự động. Đây là phương pháp được Google hỗ trợ chính thức.

### Quy trình
1. Chuẩn bị Google Sheet: Hàng 1 chứa tiêu đề cột **trùng khớp chính xác** với tiêu đề câu hỏi trong Form. Các hàng tiếp theo chứa dữ liệu cần điền.
2. Mở Form → `⋮` → **Script editor** (hoặc tạo script độc lập gắn với Sheet).
3. Dán mã nguồn, thay `PASTE_YOUR_FORM_ID` bằng ID form.
4. Chạy hàm `submitFromSheet()` và cấp quyền truy cập khi trình duyệt yêu cầu.

### Mã nguồn lõi
```javascript
function submitFromSheet() {
  const FORM_ID = 'PASTE_YOUR_FORM_ID';
  const form = FormApp.openById(FORM_ID);
  const sheet = SpreadsheetApp.getActive().getSheetByName('Data');
  const values = sheet.getDataRange().getDisplayValues();
  const headers = values.shift();

  const itemByTitle = {};
  form.getItems().forEach(it => itemByTitle[it.getTitle()] = it);

  values.forEach(row => {
    const resp = form.createResponse();
    headers.forEach((h, i) => {
      const item = itemByTitle[h];
      if (!item) return;
      const val = row[i];

      switch (item.getType()) {
        case FormApp.ItemType.TEXT:
        case FormApp.ItemType.PARAGRAPH_TEXT:
          resp.withItemResponse(item.asTextItem().createResponse(val)); break;
        case FormApp.ItemType.MULTIPLE_CHOICE:
          resp.withItemResponse(item.asMultipleChoiceItem().createResponse(val)); break;
        case FormApp.ItemType.CHECKBOX:
          resp.withItemResponse(item.asCheckboxItem().createResponse(String(val).split(/\s*,\s*/).filter(Boolean))); break;
        case FormApp.ItemType.LIST:
          resp.withItemResponse(item.asListItem().createResponse(val)); break;
        case FormApp.ItemType.SCALE:
          resp.withItemResponse(item.asScaleItem().createResponse(Number(val))); break;
        case FormApp.ItemType.DATE:
          resp.withItemResponse(item.asDateItem().createResponse(new Date(val))); break;
      }
    });
    resp.submit();
  });
}
```

## Lưu ý vận hành
- Script ánh xạ 1-1 giữa tiêu đề cột Sheet và tiêu đề câu hỏi Form. Sai chính tả/ký tự đặc biệt sẽ bỏ qua trường đó.
- Xử lý đa dạng loại input: Văn bản, trắc nghiệm đơn, hộp kiểm (tự động tách chuỗi bằng dấu phẩy), danh sách thả xuống, thang đo, ngày tháng.
- Với form bên ngoài không có quyền chỉnh sửa: Liên hệ chủ form để được cấp quyền editor hoặc yêu cầu họ tạo **Pre-filled link** chính thức. Không nên dùng automation bên thứ ba.
