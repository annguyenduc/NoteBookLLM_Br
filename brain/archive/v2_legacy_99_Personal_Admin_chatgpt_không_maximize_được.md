## 🐛 Mô tả vấn đề
- Khi nhấn `Alt + Space`, ứng dụng ChatGPT không hiển thị menu cửa sổ để chọn **Maximize**, mà lại mở ra **Command Window (CMD)**.
- Khi mở bằng cách click đúp vào icon trên Desktop/Taskbar, ứng dụng hoạt động bình thường và có thể phóng to toàn màn hình.

## 🔍 Nguyên nhân
> [!warning] Lưu ý quan trọng
> `Alt + Space` là phím tắt hệ thống gọi **System Menu** của cửa sổ đang được focus. Nếu nó mở CMD, nghĩa là:
> - Shortcut đang trỏ sai (gọi `cmd.exe` hoặc script thay vì file thực thi chính).
> - Cửa sổ đang focus không phải ChatGPT.
> - Cấu hình shortcut bị lỗi hoặc xung đột với phần mềm quản lý cửa sổ.

## 🛠️ Giải pháp khắc phục
### ✅ 1. Mở đúng cách & Ghim vào Taskbar (Khuyến nghị)
1. Nhấn `Win`, gõ `ChatGPT` → `Enter`.
2. Chuột phải icon trên Taskbar → **Pin to taskbar**.
3. Lần sau chỉ cần click trực tiếp vào icon này để mở nhanh, không cần dùng `Alt + Space`.

### ✅ 2. Kiểm tra & Sửa thuộc tính Shortcut
1. Chuột phải shortcut ChatGPT → **Properties**.
2. Tab **Shortcut** → Mục **Run**: chọn `Maximized`.
3. Kiểm tra mục **Target**: Đảm bảo đường dẫn trỏ đúng file `.exe` của ChatGPT (không phải `cmd.exe` hoặc `.bat`).
4. **Apply** → **OK**.

### ✅ 3. Tạo Shortcut mới (nếu file cũ bị lỗi)
1. `Win + R` → nhập `shell:appsfolder` → `Enter`.
2. Tìm **ChatGPT** trong danh sách → Chuột phải → **Create shortcut** (hệ thống sẽ hỏi tạo trên Desktop).
3. Sử dụng shortcut mới này để mở ứng dụng.

### ✅ 4. Kiểm tra xung đột phần mềm
- Tạm tắt các công cụ quản lý cửa sổ: **PowerToys (FancyZones)**, **AutoHotkey**, **DisplayFusion**, v.v.
- Thử lại `Alt + Space` sau khi tắt để kiểm tra.
