# Tích hợp Teachable Machine với Yolo UNO

## 🤖 AI Summary
> Hướng dẫn cách sử dụng model Teachable Machine (TM) chạy trên trình duyệt máy tính để điều khiển Yolo UNO qua cổng USB-Serial, sau đó đẩy dữ liệu lên IoT Dashboard của OhStem.

## 📝 Content
### 1. Ý tưởng chung (Workflow)
- **Máy tính (JS + WebCam)**: Chạy model Teachable Machine để dự đoán nhãn (label) và độ tin cậy (score).
- **Trình duyệt (Web Serial API)**: Gửi nhãn nhận được qua cổng USB sang Yolo UNO dưới dạng chuỗi: `TM:<label>:<score>\n`.
- **Yolo UNO (Blockly/MicroPython)**: Đón nhận chuỗi này, tách nhãn và gởi trực tiếp lên Dashboard IoT.

### 2. Mã nguồn JavaScript (Gởi từ Trình duyệt)
Công thức gởi Serial trong JS (trích lược):
```javascript
async function sendSerial(line) {
  if (!writer) return;
  const data = new TextEncoder().encode(line);
  await writer.write(data);
}
// Log output: TM:Label:Score\n
```

### 3. Giải pháp trên Yolo UNO (Blockly)
Sử dụng các khối chức năng sau:
- **Khởi tạo UART**: Tốc độ Baud 115200.
- **Vòng lặp**: Nếu có dữ liệu Serial đọc một dòng -> Kiểm tra xem có bắt đầu bằng `"TM:"` không.
- **Tách chuỗi (Split)**: Sử dụng dấu `:` để tách lấy `label` và `score`.
- **Gởi IoT**: Đẩy lên các key `tm_label` và `tm_score` của dashboard ohstem.vn.
