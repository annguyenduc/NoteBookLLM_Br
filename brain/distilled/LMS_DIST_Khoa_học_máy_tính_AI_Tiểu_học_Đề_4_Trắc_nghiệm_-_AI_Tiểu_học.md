## PHẦN 1: FACT BANK (Kiến thức chuẩn)

*   **[LMS] [Fact] :: Định nghĩa AI** :: AI là viết tắt của cụm từ tiếng Anh "Artificial Intelligence", nghĩa là Trí tuệ nhân tạo. [[Câu 1]]
*   **[LMS] [Fact] :: Công cụ lập trình AI** :: Trong học phần AI Tiểu học, phần mềm mBlock 5 (chế độ Sprites và Background) được sử dụng để lập trình các dự án trí tuệ nhân tạo. [[Câu 2]]
*   **[LMS] [Fact] :: Giao diện mBlock 5** :: Stage (Sân khấu) là khu vực hiển thị và diễn ra tất cả các hoạt động của đối tượng (nhân vật) trong chương trình. [[Câu 3]]
*   **[LMS] [Fact] :: Khối lệnh Đóng (Cap blocks)** :: Là những khối lệnh có cạnh dưới phẳng, không cho phép ghép thêm bất kỳ khối lệnh nào phía sau (ví dụ: khối lệnh dừng chương trình hoặc xóa bản sao). [[Câu 4]]
*   **[LMS] [Fact] :: Khối lệnh giá trị (Reporters & Booleans)** :: Các khối lệnh hình oval chứa giá trị là chuỗi ký tự hoặc số. Các khối lệnh hình lục giác chứa giá trị logic Đúng (True) hoặc Sai (False). Cả hai loại này không thể nối trực tiếp vào trên hoặc dưới các khối lệnh thực thi khác. [[Câu 5]]
*   **[LMS] [Fact] :: Tương tác Ask-Answer** :: Khối lệnh `Ask` (Hỏi) và `Answer` (Trả lời) thuộc nhóm lệnh Sensing (Cảm biến). Khi thực hiện lệnh `Ask`, nhân vật sẽ hiển thị câu hỏi và một khung nhập liệu; dữ liệu người dùng nhập từ bàn phím sẽ được lưu vào biến hệ thống `Answer`. [[Câu 8]]
*   **[LMS] [Fact] :: Xử lý chuỗi với lệnh Join** :: Khối lệnh `join` dùng để nối các chuỗi ký tự hoặc giá trị biến lại với nhau thành một chuỗi duy nhất. Ví dụ: `join [apple] [123]` sẽ cho kết quả là "apple123". [[Câu 11, 14]]
*   **[LMS] [Fact] :: Kiểm tra chuỗi với lệnh Contains** :: Khối lệnh `[A] contains [B]?` dùng để kiểm tra xem chuỗi A có chứa chuỗi B hay không. Kết quả trả về là giá trị Boolean (True/False). [[Câu 15, 17]]
*   **[LMS] [Fact] :: Dịch vụ nhận diện (Cognitive Services)** :: mBlock cung cấp các khối lệnh AI để nhận diện khuôn mặt (nụ cười, giới tính, cử chỉ đầu), nhận diện vật thể, và nhận diện chữ viết (in hoặc tay). [[Câu 27, 28]]
*   **[LMS] [Fact] :: Logic nhận diện vật thể** :: Khi sử dụng AI nhận diện đồ vật, máy tính thường trả về kết quả dưới dạng một danh sách hoặc chuỗi chứa các đối tượng tìm thấy. Do đó, sử dụng lệnh `contains` để kiểm tra từ khóa sẽ chính xác hơn lệnh so sánh bằng `=` tuyệt đối. [[Câu 29]]
*   **[LMS] [Fact] :: Chuyển đổi văn bản và giọng nói** :: Khối lệnh `Translate` dùng để phiên dịch ngôn ngữ; khối lệnh `Text to Speech` dùng để phát âm thanh từ văn bản với các tùy chọn về giọng đọc và ngôn ngữ. [[Câu 23, 24, 25]]
