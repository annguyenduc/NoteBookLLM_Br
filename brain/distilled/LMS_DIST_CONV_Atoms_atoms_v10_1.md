Dưới đây là các sự kiện kỹ thuật được trích xuất từ nguồn dữ liệu Volume 10 (v10) liên quan đến lập trình Python, tự động hóa hệ thống và công cụ phát triển:

- **Fact:** Atom là một trình soạn thảo mã nguồn (code editor) đã bị GitHub chính thức ngừng hỗ trợ và phát triển vào cuối năm 2022.
- **Source:** (v10 - Conversation: NIC X Google Automation with Python - ASSISTANT response)
- **Tag:** [vv10]

- **Fact:** Extension "WSL" (tên cũ là Remote - WSL) do Microsoft phát triển cho phép VSCode kết nối trực tiếp với môi trường Ubuntu trên WSL, giúp chạy code và terminal Linux ngay trong giao diện Windows.
- **Source:** (v10 - Conversation: NIC X Google Automation with Python - Các bước để VSCode làm việc với WSL)
- **Tag:** [vv10]

- **Fact:** Module `os` trong Python được sử dụng để tương tác với hệ điều hành, bao gồm các chức năng như liệt kê thư mục (`os.listdir`), kiểm tra sự tồn tại của tệp (`os.path.exists`), tạo thư mục (`os.mkdir`) và xóa tệp (`os.remove`).
- **Source:** (v10 - Conversation: NIC X Google Automation with Python - Tóm tắt Module 5)
- **Tag:** [vv10]

- **Fact:** Module `subprocess` cho phép thực thi các lệnh hệ thống (bash/shell) từ bên trong script Python thông qua hàm `subprocess.run()`.
- **Source:** (v10 - Conversation: NIC X Google Automation with Python - Tóm tắt Module 5)
- **Tag:** [vv10]

- **Fact:** Virtual Environment (`venv`) là công cụ tạo môi trường Python cô lập cho từng dự án, giúp quản lý thư viện riêng biệt và tránh xung đột phiên bản giữa các dự án khác nhau.
- **Source:** (v10 - Conversation: NIC X Google Automation with Python - Bước 5: Tạo Virtual Environment)
- **Tag:** [vv10]

- **Fact:** Module `shutil` cung cấp các thao tác tệp tin cấp cao như sao chép (`shutil.copy`) và di chuyển (`shutil.move`) tệp hoặc thư mục.
- **Source:** (v10 - Conversation: NIC X Google Automation with Python - Tóm tắt Module 5)
- **Tag:** [vv10]

- **Fact:** Hàm `os.path.join()` được khuyến nghị sử dụng để nối các thành phần đường dẫn vì nó tự động xử lý các ký tự phân tách (`/` hoặc `\`) tương thích với từng hệ điều hành.
- **Source:** (v10 - Conversation: NIC X Google Automation with Python - Tóm tắt Module 5)
- **Tag:** [vv10]

- **Fact:** Lệnh `os.walk()` thực hiện duyệt đệ quy qua tất cả các thư mục và tệp tin, trả về một tuple gồm (đường dẫn hiện tại, danh sách thư mục con, danh sách tệp tin).
- **Source:** (v10 - Conversation: NIC X Google Automation with Python - Tóm tắt Module 5)
- **Tag:** [vv10]

- **Fact:** Để cài đặt môi trường ảo trên Ubuntu/WSL, cần cài đặt gói `python3-venv` thông qua lệnh `sudo apt install python3-venv`.
- **Source:** (v10 - Conversation: NIC X Google Automation with Python - 1. Cài sẵn module venv)
- **Tag:** [vv10]

- **Fact:** "Simple Test" trong lập trình Python là phương pháp kiểm tra tính đúng đắn của hàm bằng cách sử dụng các câu lệnh điều kiện cơ bản (`if`, `print`) mà không cần đến các thư viện kiểm thử chuyên dụng như unittest hay pytest.
- **Source:** (v10 - Conversation: NIC X Google Automation with Python - Simple Test trong Module 5 - Tóm tắt)
- **Tag:** [vv10]
