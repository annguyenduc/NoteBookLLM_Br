# PHẦN 1: FACT BANK (KIẾN THỨC CHUẨN)

*   **[LMS] [Fact] :: Đặc trưng của công nghệ AI** :: Công nghệ AI có năng lực "tự học" của máy tính, cho phép nó tự phán đoán, phân tích trước các dữ liệu mới mà không cần sự hỗ trợ của con người. AI có khả năng xử lý dữ liệu với số lượng rất lớn và tốc độ cao. :: [[Câu 1]]
*   **[LMS] [Fact] :: Đặc quyền tài khoản mBlock 5** :: Khi đăng nhập tài khoản mBlock 5, người dùng có thể: Lưu và chia sẻ dự án online; Sử dụng các công cụ/nhóm lệnh mở rộng (như Cognitive Services); Truy cập thư viện nhân vật (sprite) và phông nền (background) online phong phú hơn. :: [[Câu 2, Câu 28]]
*   **[LMS] [Fact] :: Khối lệnh Boolean (Hình lục giác)** :: Các khối lệnh có hình lục giác trong mBlock/Scratch là các khối lệnh điều kiện (Boolean), luôn trả về một trong hai giá trị: Đúng (True) hoặc Sai (False). Chúng thường được chèn vào các khối điều khiển như `if-then` hoặc `wait until`. :: [[Câu 4, Câu 5]]
*   **[LMS] [Fact] :: Cơ chế hoạt động của Ask và Answer** :: Khối lệnh `Ask` dùng để nhận dữ liệu nhập từ người dùng. Biến `Answer` (Trả lời) sẽ lưu trữ giá trị của câu trả lời cuối cùng. Biến này có thể chứa cả kiểu dữ liệu chuỗi (văn bản) và số, đồng thời có thể kết hợp với các khối lệnh tính toán hoặc ghép nối (`join`). :: [[Câu 8]]
*   **[LMS] [Fact] :: Khối lệnh Join (Ghép nối)** :: Khối lệnh `join` dùng để kết hợp các chuỗi ký tự, giá trị biến hoặc kết quả tính toán lại với nhau để tạo thành một chuỗi văn bản hoàn chỉnh. :: [[Câu 11, Câu 12]]
*   **[LMS] [Fact] :: Nhóm lệnh Translate và Text to Speech** :: Nhóm lệnh "Translate" cho phép dịch văn bản sang các ngôn ngữ khác nhau. Nhóm lệnh "Text to Speech" giúp máy tính phát âm văn bản, trong đó khối lệnh `set voice to` dùng để thay đổi giọng đọc (ví dụ: Giant cho giọng nam trầm, Alto cho giọng nữ). :: [[Câu 23, Câu 24, Câu 26]]
*   **[LMS] [Fact] :: Dịch vụ nhận thức (Cognitive Services)** :: Đây là nhóm lệnh AI mở rộng yêu cầu đăng nhập tài khoản. Các khối lệnh thực thi (Stack blocks) trong nhóm này sẽ kích hoạt Camera hoặc Micro để thu thập dữ liệu hình ảnh/âm thanh nhằm mục đích nhận diện. :: [[Câu 27, Câu 28]]

A. lưu trữ/ phân tích/ truyền tải/ tốc độ.
