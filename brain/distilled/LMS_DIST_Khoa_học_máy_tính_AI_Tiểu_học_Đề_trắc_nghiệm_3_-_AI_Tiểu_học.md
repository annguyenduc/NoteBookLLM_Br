[LMS] [Fact] :: Khối lệnh báo cáo (Reporter blocks) có đặc điểm nhận dạng là hình bầu dục. Chúng trả về dữ liệu dưới dạng chữ hoặc số và không thể kết nối trực tiếp vào phía trên hoặc phía dưới các khối lệnh khác; thay vào đó, chúng được đặt vào các ô trống của các khối lệnh khác. :: [[LMS_RAW_Khoa_học_máy_tính_AI_Tiểu_học_Đề_trắc_nghiệm_3_-_AI_Tiểu_học.md]]

[LMS] [Fact] :: Khối lệnh điều kiện (Boolean blocks) có hình lục giác, luôn trả về một trong hai giá trị logic là True (Đúng) hoặc False (Sai). :: [[LMS_RAW_Khoa_học_máy_tính_AI_Tiểu_học_Đề_trắc_nghiệm_3_-_AI_Tiểu_học.md]]

[LMS] [Fact] :: Khối lệnh `contains` dùng để kiểm tra xem một chuỗi ký tự có chứa một đoạn ký tự cụ thể nào đó hay không. Ví dụ: "apple" chứa "a" là True, nhưng "apple" chứa "b" là False. :: [[LMS_RAW_Khoa_học_máy_tính_AI_Tiểu_học_Đề_trắc_nghiệm_3_-_AI_Tiểu_học.md]]

[LMS] [Fact] :: Trong mBlock/Scratch, hướng di chuyển được xác định bằng độ: hướng 0 là lên trên (12 giờ), hướng 90 là bên phải (3 giờ), hướng 180 là xuống dưới (6 giờ) và hướng -90 là bên trái (9 giờ). :: [[LMS_RAW_Khoa_học_máy_tính_AI_Tiểu_học_Đề_trắc_nghiệm_3_-_AI_Tiểu_học.md]]

[LMS] [Fact] :: Khối lệnh `set voice to [language]` trong nhóm mở rộng Text to Speech dùng để thiết lập ngôn ngữ hoặc giọng điệu phát âm cho nhân vật khi thực hiện chuyển đổi văn bản thành giọng nói. :: [[LMS_RAW_Khoa_học_máy_tính_AI_Tiểu_học_Đề_trắc_nghiệm_3_-_AI_Tiểu_học.md]]

[LMS] [Fact] :: Khối lệnh `join` được sử dụng để kết hợp (nối) các chuỗi ký tự hoặc giá trị biến lại với nhau để tạo thành một câu hoàn chỉnh. :: [[LMS_RAW_Khoa_học_máy_tính_AI_Tiểu_học_Đề_trắc_nghiệm_3_-_AI_Tiểu_học.md]]
