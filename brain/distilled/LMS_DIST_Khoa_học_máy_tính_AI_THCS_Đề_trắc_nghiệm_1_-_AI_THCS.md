# PHẦN 1: FACT BANK (KIẾN THỨC CHUẨN)

*   **[LMS] [Fact] :: Trình duyệt dancingwithai.media.mit.edu** :: Là môi trường lập trình chuyên dụng cho AI THCS vì tích hợp sẵn các nhóm lệnh mở rộng (extension) về trí tuệ nhân tạo, giao diện kéo thả quen thuộc và thư viện nhân vật/phông nền phong phú. :: [[Câu 1]]
*   **[LMS] [Fact] :: Khối lệnh báo cáo (Reporter blocks)** :: Là các khối lệnh có hình oval, trả về giá trị là một chuỗi ký tự (string) hoặc một giá trị số. Chúng không thể đứng độc lập mà phải được đặt bên trong các khối lệnh khác. :: [[Câu 2]]
*   **[LMS] [Fact] :: Khối lệnh Ask-Answer** :: Câu lệnh `ask` hiển thị khung nhập liệu; giá trị người dùng nhập vào sẽ được lưu trữ duy nhất trong biến hệ thống `answer`. Lần nhập sau sẽ ghi đè lên giá trị của lần nhập trước. :: [[Câu 3, Câu 4]]
*   **[LMS] [Fact] :: Khối lệnh Boolean (Điều kiện)** :: Là các khối lệnh hình lục giác, luôn trả về một trong hai giá trị: Đúng (True) hoặc Sai (False). :: [[Câu 5]]
*   **[LMS] [Fact] :: Khối lệnh "contains"** :: Dùng để kiểm tra xem một chuỗi ký tự (hoặc ký tự) có xuất hiện bên trong một chuỗi ký tự khác hay không. Kết quả trả về là kiểu Boolean. :: [[Câu 5, Câu 7]]
*   **[LMS] [Fact] :: Khối lệnh "join"** :: Dùng để kết hợp (nối) hai hoặc nhiều chuỗi ký tự lại với nhau theo thứ tự từ trái sang phải. :: [[Câu 6]]
*   **[LMS] [Fact] :: Biến số (Variable)** :: Là một vùng lưu trữ dữ liệu có thể thay đổi trong quá trình chạy chương trình. Khối lệnh biến số có thể chứa giá trị số hoặc chuỗi ký tự. :: [[Câu 12]]
*   **[LMS] [Fact] :: Danh sách (List)** :: Cho phép lưu trữ nhiều giá trị cùng lúc. Các thao tác cơ bản bao gồm: thêm (add), xóa (delete), chèn (insert), và truy xuất theo chỉ số (index). :: [[Câu 14, Câu 15, Câu 16]]
*   **[LMS] [Fact] :: Extension Translate & Text to Speech** :: Cho phép chương trình dịch thuật giữa các ngôn ngữ và chuyển đổi văn bản thành giọng nói với nhiều tùy chọn giọng đọc (nam, nữ, trầm, bổng). :: [[Câu 17, Câu 18]]
*   **[LMS] [Fact] :: Machine Learning (Học máy)** :: Là một lĩnh vực của AI cho phép hệ thống tự học hỏi từ dữ liệu để thực hiện nhiệm vụ mà không cần lập trình các quy tắc cụ thể cho mọi trường hợp. :: [[Câu 21]]
*   **[LMS] [Fact] :: Computer Vision (Thị giác máy tính)** :: Ứng dụng trong nhận diện khuôn mặt, lọc email (dạng hình ảnh), phân tích cảm xúc và các bộ lọc (filter) camera. :: [[Câu 20, Câu 22]]
*   **[LMS] [Fact] :: Teachable Machine Workflow** :: Quy trình chuẩn gồm: Truy cập trang web -> Cung cấp dữ liệu (Webcam/Upload) -> Chia nhóm (Class) -> Huấn luyện (Train) -> Kiểm tra (Preview) -> Xuất mô hình (Export). :: [[Câu 27]]

### Question 1: Vì sao cần phải sử dụng trình duyệt dancingwithai.media.mit.edu để dạy học các dự án trong học phần AI THCS?
A. Vì trình duyệt có đa dạng các nhân vật, phông nền phù hợp cho việc sáng tạo các dự án lập trình.
B. Vì trình duyệt có giao diện lập trình kéo thả quen thuộc và dễ sử dụng với mọi người.
