## Phần 1: FACT BANK (Kiến thức chuẩn)

*   **[LMS] [Fact] :: Mục tiêu của AI** :: Mục tiêu chính của ngành Trí tuệ nhân tạo (AI) là giúp máy tính có khả năng suy nghĩ, học tập và thực hiện các tác vụ thông minh như con người. :: [[Câu 1]]
*   **[LMS] [Fact] :: Đăng ký tài khoản mBlock** :: Người dùng có thể tạo tài khoản mBlock thông qua trang web chính thức hoặc ứng dụng, yêu cầu cung cấp email để nhận mã xác nhận kích hoạt. :: [[Câu 2]]
*   **[LMS] [Fact] :: Giao diện mBlock** :: Khu vực chứa thông tin chi tiết của một nhân vật (tên, tọa độ x-y, kích thước, hướng) thường nằm ở bảng quản lý nhân vật phía dưới sân khấu. :: [[Câu 3]]
*   **[LMS] [Fact] :: Hình dáng khối lệnh** :: Khối lệnh báo cáo (Report blocks) có hình oval (bo tròn) dùng để chứa giá trị; Khối lệnh điều kiện (Boolean blocks) có hình lục giác dùng để kiểm tra tính đúng/sai. :: [[Câu 4, 5]]
*   **[LMS] [Fact] :: Tương tác Ask-Answer** :: Câu lệnh `ask` dùng để hiển thị câu hỏi và khung nhập liệu; nội dung người dùng nhập vào sẽ được lưu trữ duy nhất trong biến hệ thống `answer`. :: [[Câu 8]]
*   **[LMS] [Fact] :: Khối lệnh Join** :: Khối lệnh `join` dùng để ghép nối các chuỗi ký tự hoặc giá trị biến lại với nhau. Để ghép $n$ thành phần, cần sử dụng ít nhất $n-1$ khối lệnh `join` lồng nhau. :: [[Câu 11, 13, 14]]
*   **[LMS] [Fact] :: Khối lệnh Contain** :: Khối lệnh `[A] contains [B]?` là khối lệnh Boolean, trả về `True` nếu chuỗi B xuất hiện trong chuỗi A và ngược lại. :: [[Câu 15]]
*   **[LMS] [Fact] :: Text to Speech & Translate** :: Để sử dụng các tính năng chuyển đổi văn bản thành giọng nói và dịch thuật, máy tính cần có kết nối Internet và người dùng phải đăng nhập tài khoản mBlock. :: [[Câu 24]]
*   **[LMS] [Fact] :: Cognitive Service (AI)** :: Đây là nhóm lệnh dịch vụ nhận diện thông minh (khuôn mặt, cảm xúc, vật thể). Giới hạn miễn phí thường là 200 lần sử dụng mỗi ngày; lỗi thường gặp là "Request timeout" do đường truyền. :: [[Câu 28]]
*   **[LMS] [Fact] :: Tính toán với Biến (Variable)** :: Biến số được dùng để lưu trữ các giá trị thay đổi trong quá trình chạy chương trình như: tổng dãy số, chu vi, diện tích hoặc thể tích. :: [[Câu 20, 21, 22]]

## Phần 2: TEST BANK (Ngân hàng đề)

### Question 1: Mục tiêu chính của ngành trí tuệ nhân tạo là gì?
A. Giúp máy tính suy nghĩ như con người
B. Giúp con người thông minh hơn
C. Giúp cải thiện tình trạng an ninh, trật tự
D. Tất cả đều đúng
**Correct: A**
**Explanation:** AI tập trung vào việc mô phỏng các quá trình suy nghĩ và hành động của con người trên máy tính.

### Question 2: Để tạo tài khoản mBlock mới, bạn cần thực hiện các bước nào sau đây? (Nhiều đáp án)
A. Điền email và nhận mã xác nhận qua email đăng ký
B. Tải về phần mềm mBlock, mở ứng dụng và chọn "Đăng ký tài khoản".
C. Truy cập trang web mBlock, nhấp vào "Tạo tài khoản" và điền thông tin cá nhân.
D. Gửi yêu cầu tạo tài khoản qua email cho nhóm phát triển mBlock.
**Correct: A, B, C**
**Explanation:** mBlock hỗ trợ đăng ký qua cả nền tảng web và ứng dụng bằng email cá nhân.
