# PHẦN 1: FACT BANK (KIẾN THỨC CHUẨN)

*   **[LMS] [Fact] :: Nhóm lệnh cơ bản trong Scratch** :: Các nhóm lệnh được phân loại theo màu sắc và chức năng: Motion (Chuyển động - Xanh dương), Looks (Hiển thị - Tím), Sound (Âm thanh - Hồng), Events (Sự kiện - Vàng), Control (Điều khiển - Cam), Sensing (Cảm biến - Xanh dương nhạt), Operators (Các phép toán - Xanh lá), Variables (Biến số - Cam đậm). :: [[Câu 1]]
*   **[LMS] [Fact] :: Chức năng Upload trong Scratch** :: Biểu tượng hình mũi tên hướng lên trong khu vực quản lý phông nền (Backdrop) hoặc nhân vật (Sprite) dùng để tải tệp tin từ máy tính lên dự án. :: [[Câu 2]]
*   **[LMS] [Fact] :: Tương tác người dùng với Ask-Answer** :: Khối lệnh `ask [văn bản] and wait` dùng để hiển thị câu hỏi và khung nhập liệu; dữ liệu người dùng nhập vào sẽ được lưu trữ tạm thời trong khối `answer`. :: [[Câu 3]]
*   **[LMS] [Fact] :: Khối lệnh Boolean "contains"** :: Khối lệnh `[chuỗi A] contains [chuỗi B]?` trả về giá trị True nếu chuỗi B nằm trong chuỗi A, ngược lại trả về False. Đây là khối lệnh điều kiện (hình lục giác). :: [[Câu 4]]
*   **[LMS] [Fact] :: Cơ chế Broadcast-Receive** :: Lệnh `broadcast [tin nhắn]` dùng để gửi một tín hiệu từ một nhân vật đến tất cả các nhân vật khác trong chương trình để kích hoạt các hành động đồng bộ. :: [[Câu 5]]
*   **[LMS] [Fact] :: Đặc điểm của Biến số (Variable)** :: Một biến số trong Scratch có thể lưu trữ nhiều loại dữ liệu khác nhau bao gồm: số, chuỗi ký tự (văn bản) hoặc giá trị logic (True/False). :: [[Câu 6]]
*   **[LMS] [Fact] :: Phân biệt Say và Speak** :: Lệnh `say` (nhóm Looks) chỉ hiển thị bong bóng văn bản trên màn hình; lệnh `speak` (nhóm Text to Speech) sẽ phát ra âm thanh đọc văn bản đó. :: [[Câu 7]]
*   **[LMS] [Fact] :: Công nghệ nhận diện khuôn mặt** :: Ứng dụng mở khóa điện thoại sử dụng kết hợp Face Detection (Phát hiện khuôn mặt) để tìm vị trí khuôn mặt và Face Recognition (Nhận diện khuôn mặt) để xác định danh tính chủ nhân. :: [[Câu 8]]
*   **[LMS] [Fact] :: Định nghĩa Deep Learning (Học sâu)** :: Deep Learning là một tập con của Machine Learning, sử dụng các mạng thần kinh nhân tạo để học từ lượng dữ liệu khổng lồ, giúp tối ưu hóa việc dự đoán và ra quyết định. :: [[Câu 9]]
*   **[LMS] [Fact] :: Extension Hand Sensing** :: Nhóm lệnh này cho phép máy tính nhận diện các điểm chạm trên bàn tay (ngón trỏ, ngón cái...) thông qua camera để điều khiển ứng dụng. :: [[Câu 10, Câu 26]]
*   **[LMS] [Fact] :: Quy trình Teachable Machine** :: Quy trình chuẩn gồm: 1. Thu thập dữ liệu (chụp ảnh/ghi âm) -> 2. Huấn luyện mô hình (Train) -> 3. Kiểm tra và Xuất mô hình (Preview/Export). :: [[Câu 11]]
*   **[LMS] [Fact] :: Thao tác với Danh sách (List)** :: Lệnh `delete [last] of [list]` được dùng để xóa phần tử cuối cùng trong danh sách. Chỉ số (index) trong danh sách Scratch bắt đầu từ 1. :: [[Câu 19, Câu 20]]
*   **[LMS] [Fact] :: Extension Translate** :: Cho phép dịch văn bản từ ngôn ngữ này sang ngôn ngữ khác một cách tự động trong chương trình. :: [[Câu 24]]
