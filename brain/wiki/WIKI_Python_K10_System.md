# 🐍 Python K10 — Hệ thống lập trình

## 📌 Định nghĩa cốt lõi
Ngôn ngữ lập trình Python trong chương trình Tin học Khối 10 là công cụ chính để học sinh làm quen với tư duy lập trình cấu trúc, xử lý dữ liệu và giải quyết bài toán thông qua mã lệnh văn bản (text-based programming).

## 🔍 Chi tiết kỹ thuật (Master Nodes)
Hệ thống kiến thức Python K10 bao gồm các thành phần chính:
- **Cú pháp cơ bản**: Quy tắc đặt tên biến, thụt lề (indentation), và chú thích (comments).
- **Kiểu dữ liệu**: Số nguyên (int), số thực (float), chuỗi (str), danh sách (list).
- **Cấu trúc điều khiển**: Câu lệnh rẽ nhánh (`if-elif-else`) và vòng lặp (`for`, `while`).
- **Hàm và Thư viện**: Cách định nghĩa hàm (`def`) và sử dụng các thư viện chuẩn như `math`, `random`.
- **mBlock 5 Python API**:
    - **Thư viện**: `import mblock`, `from mblock import event`.
    - **Đối tượng**: `sprite` đại diện cho nhân vật. Ví dụ: `sprite.forward(10)`.
    - **Cơ chế**: IntelliSense (gợi ý mã), thụt lề chuẩn 4 spaces.
    - **Lưu trữ**: File mBlock Python có đuôi `.mblock`.

## 💡 Ví dụ thực tế
Lập trình bài toán tính chỉ số BMI (Body Mass Index) từ dữ liệu nhập vào (cân nặng, chiều cao), sau đó dùng cấu trúc `if-else` để đưa ra lời khuyên sức khỏe.
