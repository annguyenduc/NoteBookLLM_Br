A PivotTable is an interactive way to quickly summarize large amounts of data. You can use a PivotTable to analyze numerical data in detail, and answer unanticipated questions about your data.

### Steps to Create a PivotTable
1. **Select Data**: Click any cell in the range or table.
2. **Insert**: Go to Insert > PivotTable.
3. **Choose Location**: Select New Worksheet or Existing Worksheet.
4. **Field List**: Drag fields into the four areas:
   - **Filters**: For top-level filtering.
   - **Columns**: For horizontal headings.
   - **Rows**: For vertical headings.
   - **Values**: For the data to be calculated (Sum, Average, etc.).

### Advanced Features
- **Slicers**: Interactive filtering buttons.
- **PivotCharts**: Visual representation of PivotTable data.
- **Refresh**: Right-click > Refresh to update data when the source changes.
