Power Query (Get & Transform) is a data connection and transformation engine. It allows users to import, clean, and shape data from various sources without writing code.

### Core Capabilities
- **Connect**: Access data from files (Excel, CSV), databases (SQL Server), web pages, and Azure.
- **Transform**: Use the Power Query Editor to remove columns, filter rows, change data types, and merge tables.
- **Automate**: Each step is recorded as a repeatable process (Query). Refreshing the data re-applies all steps.
- **M Language**: The underlying formula language that Power Query uses for transformations.

### Key Benefits
- **Repeatability**: No need to repeat the same cleaning steps every month.
- **Efficiency**: Handles millions of rows (beyond Excel's 1M limit when loading to Data Model).
- **Consistency**: Reduces manual errors during data preparation.
