# ❓ Câu hỏi
Hãy sắp xếp các hành động sau theo thứ tự phù hợp để sử dụng Teachable Machine dạy máy tính về (hình ảnh) một sự vật bất kỳ.

## 📝 Đáp án lựa chọn
- **A/ D (Extension) -> E (Training model) -> F (Name & Learn) -> A (Use the model). (Đáp án)**
- B/ F -> E -> D -> B -> C.
- C/ D -> A -> C -> B.
- D/ D -> A -> B -> C.

## 💡 Giải thích
Quy trình chuẩn:
1. Thêm tiện ích mở rộng (Extension).
2. Mở cửa sổ huấn luyện (Training model).
3. Đặt tên lớp và học mẫu (Name & Learn).
4. Sử dụng mô hình đã huấn luyện (Use model).
