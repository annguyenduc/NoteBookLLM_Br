# ❓ Câu hỏi
Ta có thể thực hiện cách nào sau đây để kết nối Arduino với máy tính?

## 📝 Đáp án lựa chọn
- **A/ Kết nối bằng dây cáp kết nối. Cổng dùng để kết nối với máy tính là cổng USB - A type. Cổng dùng để kết nối với Arduino là cổng USB - B type. (Đáp án)**
- B/ Kết nối bằng dây cáp kết nối. Cổng dùng để kết nối với máy tính là cổng USB - B type. Cổng dùng để kết nối với Arduino là cổng USB - C type.
- C/ Kết nối bằng Bluetooth.
- D/ Kết nối bằng dây điện jumper male-male.

## 💡 Giải thích
Arduino Uno thường sử dụng cáp USB Type A-to-B (giống cáp máy in) để nạp code và giao tiếp dữ liệu với máy tính.
