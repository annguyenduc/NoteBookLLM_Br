# [QUESTION] Câu 6 (Nhận biết)

## ❓ Câu hỏi
Wordpress là gì?

## 📝 Đáp án lựa chọn
- **A/ (Đáp án) WordPress là phần mềm mã nguồn mở mà bạn có thể sử dụng để tạo một trang web, blog hoặc ứng dụng. (Đáp án)**
- B/ Wordpress là một mạng xã hội giống như Facebook và Twitter.
- C/ Wordpress là một trang HTML builder bạn có thể sử dụng để tạo ra các trang HTML tĩnh của bạn.
- D/ Wordpress là một phần mềm trả phí mà bạn có thể sử dụng để tạo một trang web, blog hoặc ứng dụng đẹp.
