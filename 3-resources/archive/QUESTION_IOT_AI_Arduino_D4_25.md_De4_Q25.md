# ❓ Câu hỏi
Khối lệnh hình lục giác (như khối so sánh, khối chứa từ khóa) sau đây sẽ mang giá trị gì?

![Image](../assets/IMG_AI_Arduino_Đề_4_trắc_nghiệm_-_AI_Arduino_rId68.png)

## 📝 Đáp án lựa chọn
- **A/ True - False (Đáp án)**
- B/ Số
- C/ Chuỗi
- D/ Tất cả đều đúng

## 💡 Giải thích
Trong lập trình Scratch/mBlock, các khối lệnh hình lục giác đại diện cho các biểu thức logic (Boolean), chỉ trả về một trong hai giá trị: Đúng (True) hoặc Sai (False).
