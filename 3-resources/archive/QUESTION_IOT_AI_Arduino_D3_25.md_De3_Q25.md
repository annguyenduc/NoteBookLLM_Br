# ❓ Câu hỏi

| Cột A | Cột B |
| :--- | :--- |
| 1. Text-to-Speech | b. Chuyển đổi văn bản thành giọng nói tự nhiên. |
| 2. Emotion Recognition | e. Phân tích cảm xúc từ hình ảnh (cười, giận...). |
| 3. Language Understanding | d. Hiểu và phân tích ý định, truy vấn người dùng. |
| 4. Computer Vision | c. Nhận dạng khuôn mặt, phân loại hình ảnh, vật thể. |
| 5. Speech Recognition | a. Nhận dạng và chuyển đổi giọng nói thành văn bản. |
