# ❓ Câu hỏi
Khi thực hiện đoạn chương trình sau, robot hoạt động như thế nào?
![Image](../assets/ASSET_ROBOT_xBot_De1_rId56.png)

# 📝 Đáp án lựa chọn
A. Robot bắt đầu di chuyển tới. Nếu nghe thấy tiếng vỗ tay thì robot sẽ dừng lại.
B. Robot bắt đầu di chuyển tới. Nếu không nghe thấy tiếng vỗ tay thì robot sẽ dừng lại.
C. Robot đứng yên. Nếu nghe thấy tiếng vỗ tay thì robot sẽ bắt đầu di chuyển tới.
D. Robot đứng yên. Nếu nghe thấy tiếng vỗ tay thì robot sẽ di chuyển lùi.

# 🔗 Liên kết tư duy
- Phân tích luồng lệnh: Lệnh "Tới" nằm ngay sau "Bắt đầu". Lệnh "Dừng lại" nằm trong cấu trúc "Nếu [Âm thanh > 50]".
