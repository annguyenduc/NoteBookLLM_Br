# ❓ Câu hỏi
Khi lập trình xBot di chuyển, cần lưu ý điều gì để đảm bảo an toàn? (Nhiều đáp án)

# 📝 Đáp án lựa chọn
A. Cầm robot trên tay khi nạp code
B. Lật ngửa robot khi lập trình
C. Đặt robot trên sàn nhà để robot di chuyển
D. Tháo pin và tắt công tắc của robot mỗi khi nạp code

# 🔗 Liên kết tư duy
- Khi nạp code có lệnh di chuyển, robot có thể chạy bất ngờ và rơi khỏi bàn. Việc cầm trên tay hoặc lật ngửa giúp bánh xe quay tự do mà không làm robot di chuyển.
