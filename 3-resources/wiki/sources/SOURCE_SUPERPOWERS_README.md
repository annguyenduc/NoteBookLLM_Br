# SOURCE: Superpowers README (Jesse Vincent)

Tài liệu gốc về bộ phương pháp luận **Superpowers** - hệ thống skill tích hợp cho AI coding agents.

## Tóm tắt nội dung
- **Triết lý**: TDD (Test-Driven Development), YAGNI, DRY, bằng chứng hơn lời nói (evidence over claims).
- **Quy trình cơ bản**: Brainstorming -> Git Worktrees -> Writing Plans -> Subagent Development -> TDD -> Code Review -> Finishing.
- **Thư viện Skill**:
    - **Testing**: RED-GREEN-REFACTOR.
    - **Debugging**: 4-phase root cause process, defense-in-depth.
    - **Collaboration**: Socratic design, parallel subagents.
    - **Meta**: writing-skills (TDD applied to process docs).
