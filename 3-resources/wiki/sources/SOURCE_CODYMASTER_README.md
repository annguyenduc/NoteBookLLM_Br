# SOURCE: CodyMaster README (Tody Le)

Tài liệu gốc về **CodyMaster** - một hệ điều hành (OS) cho AI agents với hơn 68+ skills tích hợp và hệ thống quản lý bộ nhớ 5 tầng.

## Tóm tắt nội dung
- **Triết lý**: "Code Đi" - Hệ thống Senior Team trong một bộ công cụ.
- **Kiến trúc bộ nhớ (Unified Brain)**:
    - 5 tầng: Sensory, Working (`cm-continuity`), Long-term, Semantic (`qmd`), Structural (`codeintell`).
    - **Smart Spine**: SQLite + FTS5 để tìm kiếm từ khóa nhanh.
- **Hệ thống bảo vệ (Safety Gates)**: TDD -> Security -> Isolation -> Multi-gate deploy.
- **Skill Evolution**: Cơ chế tự chữa lành (Self-healing) và học hỏi liên tục (`cm-learning-promoter`).
- **Tương thích**: Hỗ trợ Claude Code, Cursor, Gemini CLI, Goose (MCP).
