# TOOL Excel Data Analysis

## 🔍 Phân tích Ingest
- **Thực thể & Khái niệm then chốt:** PivotTables, Power Query, Solver, Goal Seek, Sensitivity Analysis, Statistical functions, Financial modeling.
- **Kết nối Wiki:** Là nền tảng thực thi cho nhóm công cụ văn phòng. Kết nối với [[ENTITY_EXCEL]]. Cung cấp các kỹ thuật phân tích nhanh trước khi chuyển lên [[ENTITY_SQL|SQL]]/Python.
- **Điểm khác biệt/Mâu thuẫn:** Nhấn mạnh vào việc sử dụng Excel như một công cụ ra quyết định (Decision Making) thay vì chỉ lưu trữ dữ liệu. Giới thiệu các kỹ thuật mô phỏng Monte Carlo bằng Excel.
- **Đề xuất cấu trúc:** Cần tạo các trang về CONCEPT_TOOL_Excel_Power_Query để hướng dẫn DA cách tự động hóa việc làm sạch dữ liệu thủ công.
