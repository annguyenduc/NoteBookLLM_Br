# AutoGen

**AutoGen** là framework phát triển bởi Microsoft, tập trung vào việc xây dựng các ứng dụng Multi-Agent thông qua các cuộc hội thoại (conversations) giữa các agents.

## Đặc điểm cốt lõi

- **Conversational Patterns**: Agent tương tác với nhau bằng ngôn ngữ tự nhiên.
- **Customizable Agents**: Dễ dàng tạo các agents có tính cách và kỹ năng khác nhau.
- **Human-in-the-loop**: Hỗ trợ tích hợp sự can thiệp của con người vào cuộc hội thoại.

## So sánh nhanh

- So với **CrewAI**: AutoGen tập trung vào tính hội thoại (conversational), trong khi CrewAI tập trung vào quy trình (process-driven).
- So với **LangGraph**: AutoGen linh hoạt hơn cho các task không định hình trước, nhưng LangGraph kiểm soát tốt hơn cho production workflows.
