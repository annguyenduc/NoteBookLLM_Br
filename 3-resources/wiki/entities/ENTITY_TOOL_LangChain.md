# LangChain

**LangChain** là framework phổ biến nhất để xây dựng các ứng dụng dựa trên LLM thông qua việc kết nối (chaining) các thành phần như Prompts, Models, Memory và Tools.

> [!NOTE]
> Đây là trang Stub. Cần ingest thêm tài liệu chuyên sâu về LangChain để hoàn thiện.

## Đặc điểm cốt lõi

- **Chains**: Kết nối nhiều bước xử lý LLM.
- **Components**: Modules cho Prompts, Models, Vector Stores.
- **LCEL**: LangChain Expression Language để xây dựng pipeline linh hoạt.
