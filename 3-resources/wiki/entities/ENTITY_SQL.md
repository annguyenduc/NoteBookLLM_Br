# Ngôn ngữ Truy vấn có cấu trúc (SQL)

## 1. Định nghĩa
SQL (Structured Query Language) là ngôn ngữ tiêu chuẩn để quản lý và thao tác dữ liệu trong các Hệ quản trị cơ sở dữ liệu quan hệ (RDBMS).

## 2. Các ứng dụng chính
- **Data Retrieval**: Trích xuất dữ liệu từ các bảng.
- **Data Transformation**: Làm sạch và biến đổi dữ liệu trực tiếp trong database.
- **Data Aggregation**: Tính toán các chỉ số tổng hợp (Sum, Avg, Count).
