# pandas (Python Library)

`pandas` là thư viện mã nguồn mở phổ biến nhất của Python dành cho việc thao tác và phân tích dữ liệu. Nó cung cấp các cấu trúc dữ liệu hiệu suất cao và dễ sử dụng.

## Vai trò trong Data Science
- **Data Wrangling:** Làm sạch, biến đổi và chuẩn bị dữ liệu thô thành dạng có thể phân tích được.
- **SQL-like Operations:** Hỗ trợ các thao tác giống SQL như Join, Merge, GroupBy ngay trong Python.
- **Time Series:** Hỗ trợ mạnh mẽ cho dữ liệu chuỗi thời gian (tài chính, IoT).
