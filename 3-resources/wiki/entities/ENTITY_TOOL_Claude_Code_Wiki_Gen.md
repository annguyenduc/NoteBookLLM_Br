# Antigravity Wiki Generation & Intelligence Skill

## 1. Tổng quan
**Antigravity Wiki Gen** là bộ kỹ năng tối cao được chuyển hóa từ phương pháp luận Wiki-Gen Protocol, tối ưu hóa cho hệ sinh thái **Antigravity**. Hệ thống này biến Wiki từ một kho lưu trữ tĩnh thành một "Bản đồ tâm trí" tự tiến hóa.

## 2. Các lệnh thực thi (Commands)
Dưới đây là bộ lệnh đầy đủ được hiện thực hóa trong Antigravity:

| Lệnh | Chức năng | Trạng thái |
||---|
| `/wiki ingest` | Chuyển đổi dữ liệu thô (JSON, CSV, iMessage, Notes) thành các entry MD có timestamp và metadata. | **Sắp triển khai (Custom Script)** |
| `/wiki absorb [all/range]` | **Writer's Synthesis**: Hấp thụ tri thức theo dòng thời gian, thực hiện quy trình Rewrite bài viết để đảm bảo tính mạch lạc. | **Sắp triển khai** |
| `/wiki query <question>` | Truy vấn sâu dựa trên Navigating Wiki: Đọc Index, Check Backlinks và tổng hợp đa nguồn. | **Đang nâng cấp** |
| `/wiki cleanup` | **Audit & Enrich**: Tự động rà soát cấu trúc (Diary vs Narrative), Tone (Wikipedia style) và sửa link hỏng. | **Sắp triển khai** |
| `/wiki breakdown` | **Mining**: Phát hiện các thực thể cụ thể (Concrete Nouns) chưa có trang định nghĩa để đề xuất tạo Atom mới. | **Sắp triển khai** |
| `/wiki status` | Hiển thị thống kê: Tỷ lệ hấp thụ, mật độ liên kết, bài viết mồ côi (orphans) và các entry đang chờ. | **Đang nâng cấp** |
| `/wiki rebuild-index` | Tái cấu trúc `_index.md` (hỗ trợ Aliases) và `_backlinks.json` từ trạng thái hiện tại. | **Sắp triển khai** |

## 3. Đặc tính kỹ thuật (Antigravity Optimization)
- **Engine Core**: Sử dụng Python 3.x để quản lý các file Index JSON cục bộ.
- **Surgical Writing**: Thay vì append dữ liệu, Antigravity thực hiện đọc-hiểu toàn bộ file và viết lại (Rewrite) để giữ Narrative Coherence.
- **Relationship-Aware**: Tự động gợi ý Metadata `relationships` dựa trên nội dung bài viết.
