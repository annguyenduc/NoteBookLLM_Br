# Làm chủ SQL cho Phân tích Dữ liệu

## 1. Tổng quan
SQL là kỹ năng "phải có" đối với bất kỳ ai làm việc với dữ liệu. Nó cho phép truy cập trực tiếp vào các kho lưu trữ dữ liệu lớn của doanh nghiệp.

## 2. Các cấp độ kỹ năng
1.  **Cơ bản**: SELECT, WHERE, JOIN, GROUP BY.
2.  **Trung cấp**: Subqueries, CASE statements, Data Aggregation.
3.  **Cao cấp**: [[CONCEPT_SQL_Window_Functions|Window Functions]], [[CONCEPT_SQL_CTEs|CTEs]], Tối ưu hóa truy vấn.

## 3. Các thực thể liên quan
- [[ENTITY_SQL|ENTITY SQL]]
- [[SOURCE_TOOL_SQL_for_Data_Analysis|SOURCE SQL for Data Analysis]]

## 4. Kết luận
Kỹ năng SQL tốt giúp giảm tải công việc cho Python bằng cách thực hiện các bước làm sạch và tổng hợp dữ liệu ngay tại nguồn (Database layer), giúp quy trình phân tích trở nên tinh gọn và hiệu quả hơn.
