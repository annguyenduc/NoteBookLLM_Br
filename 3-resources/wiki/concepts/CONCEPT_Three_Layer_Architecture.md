# CONCEPT: Three-Layer Architecture (DRAFT)

## Core Principle (Raw Snippets)
- Layer 1: **Raw Sources**. Immutable. Articles, papers, images. Source of truth.
- Layer 2: **The Wiki**. LLM-owned. Markdown files (summaries, entities, concepts). Sits between user and raw.
- Layer 3: **The Schema**. (CLAUDE.md / AGENTS.md). Tells LLM how to behave. "Persistent memory" across sessions.

## Role Definition
- Obsidian is the IDE.
- The LLM is the programmer.
- The Wiki is the codebase.
