# CONCEPT: Tooling Stack (DRAFT)

## Core Principle (Raw Snippets)
- **Obsidian**: IDE/viewer for markdown and graph view.
- **qmd**: Local search engine (BM25 + Vector + LLM re-ranking).
- **Web Clipper**: Converts web articles to markdown with YAML.
- **Marp**: Generates slide decks from markdown.
- **Dataview**: SQL-like queries over YAML metadata.
- **Git**: Version control, diffs, and rollback.

## Technical Configuration
- Set "Attachment folder path" to raw/assets/.
- Download images locally to let LLM see them.
