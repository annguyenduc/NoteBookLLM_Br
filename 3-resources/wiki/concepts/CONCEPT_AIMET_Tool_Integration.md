# Tích hợp Công cụ cho Agent (Tool Integration)

## 1. Định nghĩa
Tích hợp công cụ là quá trình kết nối LLM với các hàm (Functions), API hoặc tập lệnh bên ngoài để mở rộng khả năng thực thi của Agent ngoài việc chỉ xử lý văn bản.

## 2. Quy trình thực hiện
1.  Định nghĩa mô tả công cụ (Tool Definition) rõ ràng cho LLM.
2.  Thiết lập cơ chế gọi hàm (Function Calling).
3.  Xử lý kết quả trả về từ công cụ và đưa lại vào ngữ cảnh hội thoại.

***
*Đây là kỹ năng cốt lõi của các Agent trong Wiki 2.0.*
