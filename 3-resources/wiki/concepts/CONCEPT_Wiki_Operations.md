# CONCEPT: Wiki Operations (DRAFT)

## Core Principle (Raw Snippets)
- **Ingest**: Read source -> discuss takeaways -> create/update summary -> update index -> ripple updates to concept/entity pages.
- **Query**: Search relevant pages -> synthesize answer with citations -> file valuable answers back into the wiki.
- **Lint**: Health-check. Find contradictions, orphans, missing pages, and stale claims.

## Key Insight
- A single ingest might touch 10-15 wiki pages to keep everything interlinked.
- Linting keeps the wiki "healthy" as it grows.
