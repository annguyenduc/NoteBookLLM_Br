# AIMET Learning Order Checklist (2026)

This concept captures the recommended sequence to learn Agentic AI engineering from the roadmap,
so training efforts do not jump to frameworks before core engineering foundations are stable.

## Core sequence
1. Python fundamentals: types, API design, async patterns, tests.
2. LLM fundamentals: tokens, context budgeting, prompting, tool calling.
3. Framework choice: start simple, then move to graph/workflow orchestration.
4. Advanced composition: retries, fallback, verification, runnable pipelines.
5. Memory system: summaries, vector retrieval, checkpointing.
6. Tooling discipline: schemas, safety gates, observability.
7. RAG quality: chunking, hybrid retrieval, reranking, evaluation.
8. Agent orchestration: ReAct, supervisor patterns, agent protocols.
9. Production delivery: API + UI + container + cloud + CI/CD + eval loop.

## Why this matters
- Reduces fragile "framework-first" implementations.
- Improves interview and project outcomes by aligning practice with system design maturity.
- Creates a reusable rubric for course/module sequencing in the vault.
