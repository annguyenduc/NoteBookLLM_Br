# Product Growth Metrics Cheat Sheet

![Metrics Visualization](images/DSML_Product_Metrics_CheatSheet/chart_0.png)


## Metrics Cheat Sheet

## Introduction

This cheat sheet is designed to serve as a quick reference for key metrics, helping you track product growth, make data-driven decisions, and optimize your strategies efficiently.

| Acquisition                       | Acquisition                                                                                                                                                    |
|-----------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------|
| METRIC                            | DESCRIPTION                                                                                                                                                    |
| CustomerAcquisitionCost(CAC)      | The total cost of acquiring a newcustomer, including marketing and sales expenses                                                                              |
| Conversion Rate                   | The percentage of users whotake a desired action after visiting a website or app                                                                               |
| Leads Generated                   | The number of potential customers generated from marketing efforts                                                                                             |
| NewUserSign-Ups                   | The total number of newusers acquired over a specific period                                                                                                   |
| Traffic Source                    | The breakdown of where newusers are coming from                                                                                                                |
| Cost Per Lead (CPL)               | Measures the cost of generating a potential customer lead                                                                                                      |
| CustomerAcquisition PaybackPeriod | The time it takes to recover CACthrough revenue from a customer                                                                                                |
| Activation Rate                   | The percentage of users whocomplete key onboarding actions that indicate they've experienced the product's value and are 'activated'                           |
| Time to 'Aha!'Moment              | The time it takes for a user to reach the momentwhentheyfirst experience the core value of your product, often crucial for turning sign-ups into engaged users |

| Activation                 | Activation                                                                                                           |
|----------------------------|----------------------------------------------------------------------------------------------------------------------|
| METRIC                     | DESCRIPTION                                                                                                          |
| Onboarding Completion Rate | The percentage of newusers whocomplete all key onboarding steps to start using the product.                          |
| Time to First KeyAction    | The average time it takes for a newuser to complete a crucial in-product action after signing up.                    |
| Product Usagein FirstWeek  | Measures howfrequently a newuser interacts with core features during their first week of use.                        |
| Feature Adoption Rate      | The percentage of users whoengage with specific features after completing onboarding.                                |
| Activation Rate            | The percentage of users whoachieve the initial milestone that signifies product activation (e.g., setup completion). |

| Engagement                  |                                                                                           |
|-----------------------------|-------------------------------------------------------------------------------------------|
| METRIC                      | DESCRIPTION                                                                               |
| Daily Active Users (DAU)    | The number of unique users interacting with a product daily                               |
| Monthly Active Users(MAU)   | The number of unique users interacting with a product monthly                             |
| Session Duration            | The average amount of time users spend in the app or on the platform during a session     |
| Pages/Screens Per Session   | The average number of pages or screens viewed in each user session                        |
| FeatureEngagementRate       | The percentage of users whointeract with a specific feature                               |
| Stickiness (DAU/MAURatio)   | This ratio shows howengaged users are by comparing daily to monthly usage                 |
| CustomerSatisfaction (CSAT) | Measures customer satisfaction with a product or service, often collected via surveys     |
| Net Promoter Score (NPS)    | Gauges customer loyalty by asking howlikely users are to recommend your product to others |
| Drop-off Rate               | Measures howmanyusers abandon your product at various stages of interaction               |

| Referral                           |                                                                                                   |
|------------------------------------|---------------------------------------------------------------------------------------------------|
| METRIC                             | DESCRIPTION                                                                                       |
| Referral Rate                      | The percentage of users whosuccessfully refer newcustomers to the product.                        |
| Referral Conversion Rate           | The percentage of referred users whosign up or take a desired action after being referred.        |
| Viral Coefficient                  | Measures howmanynewuserseachexisting user brings in through referrals.                            |
| Net Promoter Score (NPS)           | Asurvey metric that measures user satisfaction and likelihood to recommend the product to others. |
| Social Shares                      | The number of times users share your product or content on social media platforms.                |
| Referral ProgramParticipation Rate | The percentage of users whoengage with a referral program whenit's available.                     |
| Average Referrals Per User         | The average number of newusers that each existing user refers over a specific period.             |

| Retention                        | Retention                                                                                                                           |
|----------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| METRIC                           | DESCRIPTION                                                                                                                         |
| Cohort Retention Rate            | Tracks retention of a user cohort over time, showing howtrends vary based on signup date.                                           |
| Churn Rate                       | The percentage of users whostop using a product or cancel their service over a given period.                                        |
| CustomerRetention Rate (CRR)     | The percentage of customers whocontinue to use a product after a specific time period.                                              |
| Net Dollar Retention Rate (NDRR) | Tracks recurring revenue from existing customers, factoring in upgrades, downgrades, and churn to show net growth or decline.       |
| Repeat Purchase Rate             | The percentage of customers whomakemorethanonepurchase over a specified timeframe.                                                  |
| Time to Value (TTV)              | The time it takes for a user to achieve the first meaningful outcome from the product.                                              |
| CustomerHealth Score (CHS)       | Apredictive measure combining multiple factors like engagement and support tickets to estimate how"healthy" or loyal a customer is. |

| Revenue                         | Revenue                                                                                             |
|---------------------------------|-----------------------------------------------------------------------------------------------------|
| METRIC                          | DESCRIPTION                                                                                         |
| Average Revenue Per User (ARPU) | The average amount of revenue generated per user over a given period.                               |
| CustomerLifetime Value (LTV)    | The predicted total revenue from a customer over the lifetime of their engagement with the product. |
| Revenue Growth Rate             | The rate at which revenue increases over time, usually measured quarterly or annually.              |
| ChurnedRevenue                  | The amount of revenue lost due to customers discontinuing the service.                              |
| Average Order Value (AOV)       | The average amount spent by customers per transaction.                                              |
| Gross Margin                    | The difference between revenue and the cost of goods sold (COGS), showing profitability.            |
| Revenue Per Paying User (RPPU)  | Similar to ARPUbutfocuses only on users whomadepurchases, useful for freemium models.               |
| GrossMerchandiseVolume(GMV)     | Total value of transactions over a period (for e-commerce or marketplace products).                 |
