## Candidates bị bỏ sót

- [Concept] Exponential Process: <A process that generates growth at an increasing rate.>
- [Concept] Feedback Loop: <A process where the output of a system influences its input.>
- [Concept] Dynamic Systems: <Systems that change over time, exhibiting complex behaviors.>
- [Concept] Nonlinearity: <A relationship where changes in input don’t proportionally affect output.>
- [Concept] Delays: <Time lags in system responses affecting behavior.>
- [Concept] Self-Organization: <The spontaneous emergence of order within a system.>
- [Concept] Congruence: <The degree to which a model accurately reflects reality.>
- [Concept] Bounded Rationality: <The cognitive limitations that prevent humans from making perfectly rational decisions.>
- [Concept] State: <A condition or configuration of a system at a particular point in time.>
- [Entity] Kenneth Boulding: <An economist who provided insights into system thinking through his work on feedback loops and events.>
- [Relationship] Input-Output Pairs: <A method for understanding system behavior by analyzing the relationship between inputs and outputs.>
- [Relationship] Event-Level Understanding: <A superficial understanding based solely on individual events, lacking context and historical perspective.>
- [Relationship] Behavior-Level Understanding: <A deeper understanding of system dynamics based on long-term patterns and historical data.>
